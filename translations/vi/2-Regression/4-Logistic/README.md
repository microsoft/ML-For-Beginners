- N·∫øu m√¥ h√¨nh c·ªßa b·∫°n d·ª± ƒëo√°n m·ªôt qu·∫£ b√≠ ng√¥ l√† tr·∫Øng v√† th·ª±c t·∫ø n√≥ thu·ªôc danh m·ª•c 'tr·∫Øng', ch√∫ng ta g·ªçi ƒë√≥ l√† d∆∞∆°ng t√≠nh th·ª±c s·ª±, ƒë∆∞·ª£c bi·ªÉu th·ªã b·∫±ng s·ªë ·ªü g√≥c d∆∞·ªõi b√™n ph·∫£i.

‚úÖ H·ªèi: Theo ma tr·∫≠n nh·∫ßm l·∫´n, m√¥ h√¨nh ho·∫°t ƒë·ªông nh∆∞ th·∫ø n√†o? Tr·∫£ l·ªùi: Kh√¥ng t·ªá; c√≥ m·ªôt s·ªë l∆∞·ª£ng l·ªõn c√°c gi√° tr·ªã √¢m ƒë√∫ng nh∆∞ng c≈©ng c√≥ m·ªôt v√†i gi√° tr·ªã √¢m sai.

üéì ƒê·ªô ch√≠nh x√°c: (TP + TN)/(TP + TN + FP + FN) Ph·∫ßn trƒÉm c√°c nh√£n ƒë∆∞·ª£c d·ª± ƒëo√°n ch√≠nh x√°c cho m·ªôt m·∫´u.
