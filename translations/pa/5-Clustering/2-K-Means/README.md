1. ‡®π‡©Å‡®£ ‡®§‡©Å‡®π‡®æ‡®®‡©Ç‡©∞ ‡®á‡®π ‡®ö‡©Å‡®£‡®®‡®æ ‡®™‡®µ‡©á‡®ó‡®æ ‡®ï‡®ø ‡®ï‡®ø‡©∞‡®®‡©á ‡®ï‡®≤‡©±‡®∏‡®ü‡®∞ ‡®ü‡®æ‡®∞‡®ó‡®ü ‡®ï‡®∞‡®®‡©á ‡®π‡®®‡•§ ‡®§‡©Å‡®π‡®æ‡®®‡©Ç‡©∞ ‡®™‡®§‡®æ ‡®π‡©à ‡®ï‡®ø ‡®°‡®æ‡®ü‡®æ‡®∏‡©à‡®ü ‡®µ‡®ø‡©±‡®ö‡©ã‡®Ç 3 ‡®ó‡®æ‡®£‡©á ‡®¶‡©á ‡®ú‡®æ‡®®‡®∞ ‡®®‡®ø‡®ï‡®≤‡©á ‡®π‡®®, ‡®§‡®æ‡®Ç ‡®Ü‡®ì 3 ‡®¶‡©Ä ‡®ï‡©ã‡®∏‡®º‡®ø‡®∏‡®º ‡®ï‡®∞‡©Ä‡®è:
