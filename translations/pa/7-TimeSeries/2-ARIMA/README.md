Autoregressive ‡®Æ‡®æ‡®°‡®≤, ‡®ú‡®ø‡®µ‡©á‡®Ç ‡®ï‡®ø ‡®®‡®æ‡®Æ ‡®¶‡®∞‡®∏‡®æ‡®â‡®Ç‡®¶‡®æ ‡®π‡©à, ‡®™‡®ø‡®õ‡®≤‡©á ‡®∏‡®Æ‡©á‡®Ç ‡®µ‡®ø‡©±‡®ö ‡®°‡®æ‡®ü‡®æ ‡®¶‡©á ‡®™‡®ø‡®õ‡®≤‡©á ‡®Æ‡©Å‡©±‡®≤‡®æ‡®Ç ‡®¶‡®æ ‡®µ‡®ø‡®∂‡®≤‡©á‡®∏‡®º‡®£ ‡®ï‡®∞‡®¶‡©á ‡®π‡®® ‡®Ö‡®§‡©á ‡®â‡®®‡©ç‡®π‡®æ‡®Ç ‡®¨‡®æ‡®∞‡©á ‡®ß‡®æ‡®∞‡®®‡®æ ‡®¨‡®£‡®æ‡®â‡®Ç‡®¶‡©á ‡®π‡®®‡•§ ‡®á‡®π ‡®™‡®ø‡®õ‡®≤‡©á ‡®Æ‡©Å‡©±‡®≤‡®æ‡®Ç 'lags' ‡®ï‡®π‡®ø‡©∞‡®¶‡©á ‡®π‡®®‡•§ ‡®â‡®¶‡®æ‡®π‡®∞‡®£ ‡®≤‡®à, ‡®™‡©à‡®Ç‡®∏‡®≤‡®æ‡®Ç ‡®¶‡©Ä ‡®Æ‡®æ‡®∏‡®ø‡®ï ‡®µ‡®ø‡®ï‡®∞‡©Ä ‡®¶‡®ø‡®ñ‡®æ‡®â‡®£ ‡®µ‡®æ‡®≤‡®æ ‡®°‡®æ‡®ü‡®æ‡•§ ‡®π‡®∞ ‡®Æ‡®π‡©Ä‡®®‡©á ‡®¶‡©Ä ‡®µ‡®ø‡®ï‡®∞‡©Ä ‡®¶‡©Ä ‡®ï‡©Å‡©±‡®≤ ‡®∞‡®ï‡®Æ ‡®°‡®æ‡®ü‡®æ‡®∏‡©à‡©±‡®ü ‡®µ‡®ø‡©±‡®ö ‡®á‡©±‡®ï 'evolving variable' ‡®Æ‡©∞‡®®‡©Ä ‡®ú‡®æ‡®µ‡©á‡®ó‡©Ä‡•§ ‡®á‡®π ‡®Æ‡®æ‡®°‡®≤ ‡®á‡®∏ ‡®§‡®∞‡©ç‡®π‡®æ‡®Ç ‡®¨‡®£‡®æ‡®á‡®Ü ‡®ú‡®æ‡®Ç‡®¶‡®æ ‡®π‡©à ‡®ï‡®ø "‡®µ‡®ø‡®ï‡®æ‡®∏‡®∏‡®º‡©Ä‡®≤ ‡®ö‡®∞ ‡®®‡©Ç‡©∞ ‡®Ü‡®™‡®£‡©á ‡®™‡®ø‡®õ‡®≤‡©á (‡®Ö‡®∞‡®•‡®æ‡®§, ‡®™‡®ø‡®õ‡®≤‡©á) ‡®Æ‡©Å‡©±‡®≤‡®æ‡®Ç '‡®§‡©á regress ‡®ï‡©Ä‡®§‡®æ ‡®ú‡®æ‡®Ç‡®¶‡®æ ‡®π‡©à‡•§" [wikipedia](https://wikipedia.org/wiki/Autoregressive_integrated_moving_average)
