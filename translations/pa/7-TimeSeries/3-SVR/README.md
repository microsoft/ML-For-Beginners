1. 1 ‡®∏‡®§‡©∞‡®¨‡®∞ ‡®§‡©ã‡®Ç 31 ‡®Ö‡®ï‡®§‡©Ç‡®¨‡®∞ 2014 ‡®§‡©±‡®ï ‡®¶‡©Ä ‡®¶‡©ã ‡®Æ‡®π‡©Ä‡®®‡©á ‡®¶‡©Ä ‡®Æ‡®ø‡®Ü‡®¶ ‡®®‡©Ç‡©∞ ‡®ü‡©ç‡®∞‡©á‡®®‡®ø‡©∞‡®ó ‡®∏‡©à‡®ü ‡®≤‡®à ‡®Ö‡®≤‡®æ‡®ü ‡®ï‡®∞‡©ã‡•§ ‡®ü‡©à‡®∏‡®ü ‡®∏‡©à‡®ü ‡®µ‡®ø‡©±‡®ö 1 ‡®®‡®µ‡©∞‡®¨‡®∞ ‡®§‡©ã‡®Ç 31 ‡®¶‡®∏‡©∞‡®¨‡®∞ 2014 ‡®§‡©±‡®ï ‡®¶‡©Ä ‡®¶‡©ã ‡®Æ‡®π‡©Ä‡®®‡©á ‡®¶‡©Ä ‡®Æ‡®ø‡®Ü‡®¶ ‡®∏‡®º‡®æ‡®Æ‡®≤ ‡®π‡©ã‡®µ‡©á‡®ó‡©Ä: [^2]

   ```python
   train_start_dt = '2014-11-01 00:00:00'
   ```

2. ‡®Ö‡©∞‡®§‡®∞‡®æ‡®Ç ‡®®‡©Ç‡©∞ ‡®µ‡®ø‡®ú‡©Å‡®Ö‡®≤‡®æ‡®à‡®ú‡®º ‡®ï‡®∞‡©ã: [^2]

### ‡®ü‡©ç‡®∞‡©á‡®®‡®ø‡©∞‡®ó ‡®≤‡®à ‡®°‡®æ‡®ü‡®æ ‡®§‡®ø‡®Ü‡®∞ ‡®ï‡®∞‡©ã

‡®π‡©Å‡®£, ‡®§‡©Å‡®π‡®æ‡®®‡©Ç‡©∞ ‡®´‡®ø‡®≤‡®ü‡®∞‡©Ä‡®Ç‡®ó ‡®Ö‡®§‡©á ‡®∏‡®ï‡©á‡®≤‡®ø‡©∞‡®ó ‡®ï‡®∞‡®ï‡©á ‡®Ü‡®™‡®£‡©á ‡®°‡®æ‡®ü‡®æ ‡®®‡©Ç‡©∞ ‡®ü‡©ç‡®∞‡©á‡®®‡®ø‡©∞‡®ó ‡®≤‡®à ‡®§‡®ø‡®Ü‡®∞ ‡®ï‡®∞‡®® ‡®¶‡©Ä ‡®≤‡©ã‡©ú ‡®π‡©à‡•§ ‡®Ü‡®™‡®£‡©á ‡®°‡®æ‡®ü‡®æ‡®∏‡©à‡®ü ‡®®‡©Ç‡©∞ ‡®´‡®ø‡®≤‡®ü‡®∞ ‡®ï‡®∞‡©ã ‡®§‡®æ‡®Ç ‡®ú‡©ã ‡®∏‡®ø‡®∞‡®´‡®º ‡®≤‡©ã‡©ú‡©Ä‡®Ç‡®¶‡©á ‡®∏‡®Æ‡©á‡®Ç ‡®¶‡©Ä ‡®Æ‡®ø‡®Ü‡®¶ ‡®Ö‡®§‡©á ‡®ï‡®æ‡®≤‡®Æ ‡®∏‡®º‡®æ‡®Æ‡®≤ ‡®ï‡©Ä‡®§‡©á ‡®ú‡®æ ‡®∏‡®ï‡®£, ‡®Ö‡®§‡©á ‡®°‡®æ‡®ü‡®æ ‡®®‡©Ç‡©∞ 0,1 ‡®¶‡©á ‡®á‡©∞‡®ü‡®∞‡®µ‡®æ‡®≤ ‡®µ‡®ø‡©±‡®ö ‡®™‡©ç‡®∞‡©ã‡®ú‡©à‡®ï‡®ü ‡®ï‡®∞‡®® ‡®≤‡®à ‡®∏‡®ï‡©á‡®≤‡®ø‡©∞‡®ó ‡®ï‡®∞‡©ã‡•§

1. ‡®Æ‡©Ç‡®≤ ‡®°‡®æ‡®ü‡®æ‡®∏‡©à‡®ü ‡®®‡©Ç‡©∞ ‡®∏‡®ø‡®∞‡®´‡®º ‡®â‡®™‡®∞‡©ã‡®ï‡®§ ‡®∏‡®Æ‡©á‡®Ç ‡®¶‡©Ä ‡®Æ‡®ø‡®Ü‡®¶ ‡®Ö‡®§‡©á ‡®∏‡®ø‡®∞‡®´‡®º ‡®≤‡©ã‡©ú‡©Ä‡®Ç‡®¶‡®æ ‡®ï‡®æ‡®≤‡®Æ 'load' ‡®Ö‡®§‡©á ‡®§‡®æ‡®∞‡©Ä‡®ñ ‡®∏‡®º‡®æ‡®Æ‡®≤ ‡®ï‡®∞‡®® ‡®≤‡®à ‡®´‡®ø‡®≤‡®ü‡®∞ ‡®ï‡®∞‡©ã: [^2]

   ```python
   train = energy.copy()[(energy.index >= train_start_dt) & (energy.index < test_start_dt)][['load']]
   test = energy.copy()[energy.index >= test_start_dt][['load']]

   print('Test data shape: ', test.shape)
   ```

2. ‡®ü‡©ç‡®∞‡©á‡®®‡®ø‡©∞‡®ó ‡®°‡®æ‡®ü‡®æ ‡®®‡©Ç‡©∞ (0, 1) ‡®¶‡©Ä ‡®∞‡©á‡®Ç‡®ú ‡®µ‡®ø‡©±‡®ö ‡®∏‡®ï‡©á‡®≤ ‡®ï‡®∞‡©ã: [^2]


4. ‡®π‡©Å‡®£, ‡®ü‡©à‡®∏‡®ü‡®ø‡©∞‡®ó ‡®°‡®æ‡®ü‡®æ ‡®®‡©Ç‡©∞ ‡®∏‡®ï‡©á‡®≤ ‡®ï‡®∞‡©ã: [^2]


### ‡®ü‡®æ‡®à‡®Æ-‡®∏‡®ü‡©à‡®™‡®∏ ‡®®‡®æ‡®≤ ‡®°‡®æ‡®ü‡®æ ‡®¨‡®£‡®æ‡®â‡®£‡®æ [^1]

SVR ‡®≤‡®à, ‡®§‡©Å‡®∏‡©Ä‡®Ç ‡®á‡®®‡®™‡©Å‡®ü ‡®°‡®æ‡®ü‡®æ ‡®®‡©Ç‡©∞ `[batch, timesteps]` ‡®¶‡©á ‡®∞‡©Ç‡®™ ‡®µ‡®ø‡©±‡®ö ‡®§‡®¨‡®¶‡©Ä‡®≤ ‡®ï‡®∞‡®¶‡©á ‡®π‡©ã‡•§ ‡®á‡®∏ ‡®≤‡®à, ‡®§‡©Å‡®∏‡©Ä‡®Ç ‡®Æ‡©å‡®ú‡©Ç‡®¶‡®æ `train_data` ‡®Ö‡®§‡©á `test_data` ‡®®‡©Ç‡©∞ ‡®∞‡©Ä‡®∏‡®º‡©á‡®™ ‡®ï‡®∞‡®¶‡©á ‡®π‡©ã ‡®§‡®æ‡®Ç ‡®ï‡®ø ‡®á‡©±‡®ï ‡®®‡®µ‡®æ‡®Ç ‡®°‡®æ‡®á‡®Æ‡©à‡®Ç‡®∏‡®º‡®® ‡®π‡©ã‡®µ‡©á ‡®ú‡©ã ‡®ü‡®æ‡®à‡®Æ-‡®∏‡®ü‡©à‡®™‡®∏ ‡®®‡©Ç‡©∞ ‡®¶‡®∞‡®∏‡®æ‡®â‡®Ç‡®¶‡®æ ‡®π‡©à‡•§

```python
# Converting to numpy arrays
train_data = train.values

```

‡®á‡®∏ ‡®â‡®¶‡®æ‡®π‡®∞‡®® ‡®≤‡®à, ‡®Ö‡®∏‡©Ä‡®Ç `timesteps = 5` ‡®≤‡©à‡®Ç‡®¶‡©á ‡®π‡®æ‡®Ç‡•§ ‡®á‡®∏ ‡®≤‡®à, ‡®Æ‡®æ‡®°‡®≤ ‡®≤‡®à ‡®á‡®®‡®™‡©Å‡®ü ‡®™‡®π‡®ø‡®≤‡©á 4 ‡®ü‡®æ‡®à‡®Æ-‡®∏‡®ü‡©à‡®™‡®∏ ‡®≤‡®à ‡®°‡®æ‡®ü‡®æ ‡®π‡©à, ‡®Ö‡®§‡©á ‡®Ü‡®â‡®ü‡®™‡©Å‡®ü 5‡®µ‡©á‡®Ç ‡®ü‡®æ‡®à‡®Æ-‡®∏‡®ü‡©à‡®™ ‡®≤‡®à ‡®°‡®æ‡®ü‡®æ ‡®π‡©ã‡®µ‡©á‡®ó‡®æ‡•§

```python
timesteps=5
```

Nested list comprehension ‡®¶‡©Ä ‡®µ‡®∞‡®§‡©ã‡®Ç ‡®ï‡®∞‡®ï‡©á ‡®ü‡©ç‡®∞‡©á‡®®‡®ø‡©∞‡®ó ‡®°‡®æ‡®ü‡®æ ‡®®‡©Ç‡©∞ 2D ‡®ü‡©à‡®Ç‡®∏‡®∞ ‡®µ‡®ø‡©±‡®ö ‡®§‡®¨‡®¶‡©Ä‡®≤ ‡®ï‡®∞‡®®‡®æ:

```python
train_data_timesteps=np.array([[j for j in train_data[i:i+timesteps]] for i in range(0,len(train_data)-timesteps+1)])[:,:,0]
```


‡®ü‡©à‡®∏‡®ü‡®ø‡©∞‡®ó ‡®°‡®æ‡®ü‡®æ ‡®®‡©Ç‡©∞ 2D ‡®ü‡©à‡®Ç‡®∏‡®∞ ‡®µ‡®ø‡©±‡®ö ‡®§‡®¨‡®¶‡©Ä‡®≤ ‡®ï‡®∞‡®®‡®æ:

```python
test_data_timesteps=np.array([[j for j in test_data[i:i+timesteps]] for i in range(0,len(test_data)-timesteps+1)])[:,:,0]
```

### SVR ‡®≤‡®æ‡®ó‡©Ç ‡®ï‡®∞‡©ã [^1]

‡®π‡©Å‡®£, SVR ‡®≤‡®æ‡®ó‡©Ç ‡®ï‡®∞‡®® ‡®¶‡®æ ‡®∏‡®Æ‡®æ‡®Ç ‡®π‡©à‡•§ ‡®á‡®∏ ‡®≤‡®æ‡®ó‡©Ç ‡®ï‡®∞‡®® ‡®¨‡®æ‡®∞‡©á ‡®π‡©ã‡®∞ ‡®™‡©ú‡©ç‡®π‡®® ‡®≤‡®à, ‡®§‡©Å‡®∏‡©Ä‡®Ç [‡®á‡®π ‡®°‡©å‡®ï‡©Ç‡®Æ‡©à‡®Ç‡®ü‡©á‡®∏‡®º‡®®](https://scikit-learn.org/stable/modules/generated/sklearn.svm.SVR.html) ‡®®‡©Ç‡©∞ ‡®∞‡®ø‡®´‡®∞ ‡®ï‡®∞ ‡®∏‡®ï‡®¶‡©á ‡®π‡©ã‡•§ ‡®∏‡®æ‡®°‡©á ‡®≤‡®æ‡®ó‡©Ç ‡®ï‡®∞‡®® ‡®≤‡®à, ‡®Ö‡®∏‡©Ä‡®Ç ‡®á‡®π ‡®ï‡®¶‡®Æ ‡®Ö‡®®‡©Å‡®∏‡®∞‡®£ ‡®ï‡®∞‡®¶‡©á ‡®π‡®æ‡®Ç:

  1. ‡®Æ‡®æ‡®°‡®≤ ‡®®‡©Ç‡©∞ `SVR()` ‡®ï‡®æ‡®≤ ‡®ï‡®∞‡®ï‡©á ‡®Ö‡®§‡©á ‡®Æ‡®æ‡®°‡®≤ ‡®π‡®æ‡®à‡®™‡®∞‡®™‡©à‡®∞‡®æ‡®Æ‡©Ä‡®ü‡®∞‡®ú‡®º: kernel, gamma, c ‡®Ö‡®§‡©á epsilon ‡®™‡®æ‡®∏ ‡®ï‡®∞‡®ï‡©á ‡®°‡®ø‡®´‡®æ‡®à‡®® ‡®ï‡®∞‡©ã‡•§
  2. `fit()` ‡®´‡©∞‡®ï‡®∏‡®º‡®® ‡®ï‡®æ‡®≤ ‡®ï‡®∞‡®ï‡©á ‡®ü‡©ç‡®∞‡©á‡®®‡®ø‡©∞‡®ó ‡®°‡®æ‡®ü‡®æ ‡®≤‡®à ‡®Æ‡®æ‡®°‡®≤ ‡®§‡®ø‡®Ü‡®∞ ‡®ï‡®∞‡©ã‡•§
  3. `predict()` ‡®´‡©∞‡®ï‡®∏‡®º‡®® ‡®ï‡®æ‡®≤ ‡®ï‡®∞‡®ï‡©á ‡®≠‡®µ‡®ø‡©±‡®ñ‡®µ‡®æ‡®£‡©Ä ‡®ï‡®∞‡©ã‡•§

‡®π‡©Å‡®£ ‡®Ö‡®∏‡©Ä‡®Ç SVR ‡®Æ‡®æ‡®°‡®≤ ‡®¨‡®£‡®æ‡®â‡®Ç‡®¶‡©á ‡®π‡®æ‡®Ç‡•§ ‡®á‡©±‡®•‡©á ‡®Ö‡®∏‡©Ä‡®Ç [RBF kernel](https://scikit-learn.org/stable/modules/svm.html#parameters-of-the-rbf-kernel) ‡®¶‡©Ä ‡®µ‡®∞‡®§‡©ã‡®Ç ‡®ï‡®∞‡®¶‡©á ‡®π‡®æ‡®Ç, ‡®Ö‡®§‡©á gamma, C ‡®Ö‡®§‡©á epsilon ‡®®‡©Ç‡©∞ 0.5, 10 ‡®Ö‡®§‡©á 0.05 ‡®µ‡®ú‡©ã‡®Ç ‡®∏‡©à‡®ü ‡®ï‡®∞‡®¶‡©á ‡®π‡®æ‡®Ç‡•§

```python
model = SVR(kernel='rbf',gamma=0.5, C=10, epsilon = 0.05)
```

#### ‡®ü‡©ç‡®∞‡©á‡®®‡®ø‡©∞‡®ó ‡®°‡®æ‡®ü‡®æ '‡®§‡©á ‡®Æ‡®æ‡®°‡®≤ ‡®´‡®ø‡®ü ‡®ï‡®∞‡©ã [^1]

```python
model.fit(x_train, y_train[:,0])
```

```output
SVR(C=10, cache_size=200, coef0=0.0, degree=3, epsilon=0.05, gamma=0.5,
    kernel='rbf', max_iter=-1, shrinking=True, tol=0.001, verbose=False)
```

#### ‡®Æ‡®æ‡®°‡®≤ ‡®¶‡©Ä ‡®≠‡®µ‡®ø‡©±‡®ñ‡®µ‡®æ‡®£‡©Ä ‡®ï‡®∞‡©ã [^1]


‡®§‡©Å‡®π‡®æ‡®°‡®æ SVR ‡®¨‡®£ ‡®ó‡®ø‡®Ü ‡®π‡©à! ‡®π‡©Å‡®£ ‡®Ö‡®∏‡©Ä‡®Ç ‡®á‡®∏‡®¶‡©Ä ‡®Æ‡©Å‡®≤‡®æ‡®Ç‡®ï‡®£ ‡®ï‡®∞‡®¶‡©á ‡®π‡®æ‡®Ç‡•§

### ‡®Ü‡®™‡®£‡©á ‡®Æ‡®æ‡®°‡®≤ ‡®¶‡®æ ‡®Æ‡©Å‡®≤‡®æ‡®Ç‡®ï‡®£ ‡®ï‡®∞‡©ã [^1]

‡®Æ‡©Å‡®≤‡®æ‡®Ç‡®ï‡®£ ‡®≤‡®à, ‡®™‡®π‡®ø‡®≤‡®æ‡®Ç ‡®Ö‡®∏‡©Ä‡®Ç ‡®°‡®æ‡®ü‡®æ ‡®®‡©Ç‡©∞ ‡®Ü‡®™‡®£‡©á ‡®Æ‡©Ç‡®≤ ‡®∏‡®ï‡©á‡®≤ ‡®µ‡®ø‡©±‡®ö ‡®µ‡®æ‡®™‡®∏ ‡®∏‡®ï‡©á‡®≤ ‡®ï‡®∞‡®¶‡©á ‡®π‡®æ‡®Ç‡•§ ‡®´‡®ø‡®∞, ‡®™‡©ç‡®∞‡®¶‡®∞‡®∏‡®º‡®® ‡®¶‡©Ä ‡®ú‡®æ‡®Ç‡®ö ‡®ï‡®∞‡®® ‡®≤‡®à, ‡®Ö‡®∏‡©Ä‡®Ç ‡®Æ‡©Ç‡®≤ ‡®Ö‡®§‡©á ‡®≠‡®µ‡®ø‡©±‡®ñ‡®µ‡®æ‡®£‡©Ä ‡®∏‡®Æ‡©á‡®Ç ‡®¶‡©Ä ‡®≤‡©ú‡©Ä ‡®¶‡®æ ‡®™‡®≤‡®æ‡®ü ‡®¨‡®£‡®æ‡®â‡®Ç‡®¶‡©á ‡®π‡®æ‡®Ç, ‡®Ö‡®§‡©á MAPE ‡®®‡®§‡©Ä‡®ú‡®æ ‡®™‡©ç‡®∞‡®ø‡©∞‡®ü ‡®ï‡®∞‡®¶‡©á ‡®π‡®æ‡®Ç‡•§

‡®≠‡®µ‡®ø‡©±‡®ñ‡®µ‡®æ‡®£‡©Ä ‡®ï‡©Ä‡®§‡©Ä ‡®Ö‡®§‡©á ‡®Æ‡©Ç‡®≤ ‡®Ü‡®â‡®ü‡®™‡©Å‡®ü ‡®®‡©Ç‡©∞ ‡®∏‡®ï‡©á‡®≤ ‡®ï‡®∞‡©ã:
