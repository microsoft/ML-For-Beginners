### –ü—Ä–µ—Ç–ø—Ä–æ—Ü–µ—Å–∏—Ä–∞—ö–µ –ø–æ–¥–∞—Ç–∞–∫–∞: –∫–æ–¥–∏—Ä–∞—ö–µ –∫–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∞ –∏ –æ–∑–Ω–∞–∫–∞
–ù–∞—à —Å–∫—É–ø –ø–æ–¥–∞—Ç–∞–∫–∞ –æ –±—É–Ω–¥–µ–≤–∞–º–∞ —Å–∞–¥—Ä–∂–∏ —Å—Ç—Ä–∏–Ω–≥ –≤—Ä–µ–¥–Ω–æ—Å—Ç–∏ –∑–∞ —Å–≤–µ —Å–≤–æ—ò–µ –∫–æ–ª–æ–Ω–µ. –†–∞–¥ —Å–∞ –∫–∞—Ç–µ–≥–æ—Ä–∏—ò–∞–ª–Ω–∏–º –ø–æ–¥–∞—Ü–∏–º–∞ —ò–µ –∏–Ω—Ç—É–∏—Ç–∏–≤–∞–Ω –∑–∞ —ô—É–¥–µ, –∞–ª–∏ –Ω–µ –∏ –∑–∞ –º–∞—à–∏–Ω–µ. –ê–ª–≥–æ—Ä–∏—Ç–º–∏ –º–∞—à–∏–Ω—Å–∫–æ–≥ —É—á–µ—ö–∞ –¥–æ–±—Ä–æ —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∏—à—É —Å–∞ –±—Ä–æ—ò–µ–≤–∏–º–∞. –ó–∞—Ç–æ —ò–µ –∫–æ–¥–∏—Ä–∞—ö–µ –≤–µ–æ–º–∞ –≤–∞–∂–∞–Ω –∫–æ—Ä–∞–∫ —É —Ñ–∞–∑–∏ –ø—Ä–µ—Ç–ø—Ä–æ—Ü–µ—Å–∏—Ä–∞—ö–∞ –ø–æ–¥–∞—Ç–∞–∫–∞, —ò–µ—Ä –Ω–∞–º –æ–º–æ–≥—É—õ–∞–≤–∞ –¥–∞ –∫–∞—Ç–µ–≥–æ—Ä–∏—ò–∞–ª–Ω–µ –ø–æ–¥–∞—Ç–∫–µ –ø—Ä–µ—Ç–≤–æ—Ä–∏–º–æ —É –Ω—É–º–µ—Ä–∏—á–∫–µ –ø–æ–¥–∞—Ç–∫–µ, –±–µ–∑ –≥—É–±–∏—Ç–∫–∞ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—ò–∞. –î–æ–±—Ä–æ –∫–æ–¥–∏—Ä–∞—ö–µ –≤–æ–¥–∏ –∫–∞ –∏–∑–≥—Ä–∞–¥—ö–∏ –¥–æ–±—Ä–æ–≥ –º–æ–¥–µ–ª–∞.

–ó–∞ –∫–æ–¥–∏—Ä–∞—ö–µ –∫–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∞ –ø–æ—Å—Ç–æ—ò–µ –¥–≤–µ –≥–ª–∞–≤–Ω–µ –≤—Ä—Å—Ç–µ –∫–æ–¥–µ—Ä–∞:

1. –û—Ä–¥–∏–Ω–∞–ª–Ω–∏ –∫–æ–¥–µ—Ä: –¥–æ–±—Ä–æ –æ–¥–≥–æ–≤–∞—Ä–∞ –æ—Ä–¥–∏–Ω–∞–ª–Ω–∏–º –≤–∞—Ä–∏—ò–∞–±–ª–∞–º–∞, –∫–æ—ò–µ —Å—É –∫–∞—Ç–µ–≥–æ—Ä–∏—ò–∞–ª–Ω–µ –≤–∞—Ä–∏—ò–∞–±–ª–µ –≥–¥–µ —ö–∏—Ö–æ–≤–∏ –ø–æ–¥–∞—Ü–∏ —Å–ª–µ–¥–µ –ª–æ–≥–∏—á–∫–∏ —Ä–µ–¥–æ—Å–ª–µ–¥, –∫–∞–æ —à—Ç–æ —ò–µ –∫–æ–ª–æ–Ω–∞ `–í–µ–ª–∏—á–∏–Ω–∞ –∞—Ä—Ç–∏–∫–ª–∞` —É –Ω–∞—à–µ–º —Å–∫—É–ø—É –ø–æ–¥–∞—Ç–∞–∫–∞. –û–Ω —Å—Ç–≤–∞—Ä–∞ –º–∞–ø–∏—Ä–∞—ö–µ —Ç–∞–∫–æ –¥–∞ —ò–µ —Å–≤–∞–∫–∞ –∫–∞—Ç–µ–≥–æ—Ä–∏—ò–∞ –ø—Ä–µ–¥—Å—Ç–∞–≤—ô–µ–Ω–∞ –±—Ä–æ—ò–µ–º, –∫–æ—ò–∏ —ò–µ —Ä–µ–¥–æ—Å–ª–µ–¥ –∫–∞—Ç–µ–≥–æ—Ä–∏—ò–µ —É –∫–æ–ª–æ–Ω–∏.

    ```python
    from sklearn.preprocessing import OrdinalEncoder

    item_size_categories = [['sml', 'med', 'med-lge', 'lge', 'xlge', 'jbo', 'exjbo']]
    ordinal_features = ['Item Size']
    ordinal_encoder = OrdinalEncoder(categories=item_size_categories)
    ```

2. –ö–∞—Ç–µ–≥–æ—Ä–∏—ò–∞–ª–Ω–∏ –∫–æ–¥–µ—Ä: –¥–æ–±—Ä–æ –æ–¥–≥–æ–≤–∞—Ä–∞ –Ω–æ–º–∏–Ω–∞–ª–Ω–∏–º –≤–∞—Ä–∏—ò–∞–±–ª–∞–º–∞, –∫–æ—ò–µ —Å—É –∫–∞—Ç–µ–≥–æ—Ä–∏—ò–∞–ª–Ω–µ –≤–∞—Ä–∏—ò–∞–±–ª–µ –≥–¥–µ —ö–∏—Ö–æ–≤–∏ –ø–æ–¥–∞—Ü–∏ –Ω–µ —Å–ª–µ–¥–µ –ª–æ–≥–∏—á–∫–∏ —Ä–µ–¥–æ—Å–ª–µ–¥, –∫–∞–æ —à—Ç–æ —Å—É —Å–≤–µ –∫–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–µ —Ä–∞–∑–ª–∏—á–∏—Ç–µ –æ–¥ `–í–µ–ª–∏—á–∏–Ω–∞ –∞—Ä—Ç–∏–∫–ª–∞` —É –Ω–∞—à–µ–º —Å–∫—É–ø—É –ø–æ–¥–∞—Ç–∞–∫–∞. –¢–æ —ò–µ —ò–µ–¥–Ω–æ-—Ö–æ—Ç –∫–æ–¥–∏—Ä–∞—ö–µ, —à—Ç–æ –∑–Ω–∞—á–∏ –¥–∞ —ò–µ —Å–≤–∞–∫–∞ –∫–∞—Ç–µ–≥–æ—Ä–∏—ò–∞ –ø—Ä–µ–¥—Å—Ç–∞–≤—ô–µ–Ω–∞ –±–∏–Ω–∞—Ä–Ω–æ–º –∫–æ–ª–æ–Ω–æ–º: –∫–æ–¥–∏—Ä–∞–Ω–∞ –≤–∞—Ä–∏—ò–∞–±–ª–∞ —ò–µ —ò–µ–¥–Ω–∞–∫–∞ 1 –∞–∫–æ –±—É–Ω–¥–µ–≤–∞ –ø—Ä–∏–ø–∞–¥–∞ —Ç–æ—ò —Å–æ—Ä—Ç–∏, –∞ 0 —É —Å—É–ø—Ä–æ—Ç–Ω–æ–º.

–ó–∞—Ç–∏–º, `ColumnTransformer` —Å–µ –∫–æ—Ä–∏—Å—Ç–∏ –∑–∞ –∫–æ–º–±–∏–Ω–æ–≤–∞—ö–µ –≤–∏—à–µ –∫–æ–¥–µ—Ä–∞ —É —ò–µ–¥–∞–Ω –∫–æ—Ä–∞–∫ –∏ —ö–∏—Ö–æ–≤–æ –ø—Ä–∏–º–µ–Ω—É –Ω–∞ –æ–¥–≥–æ–≤–∞—Ä–∞—ò—É—õ–µ –∫–æ–ª–æ–Ω–µ.
