1. –ù–∞–ø—Ä–∞–≤–∏—Ç–µ —Ñ—É–Ω–∫—Ü–∏—ò—É `create_ingredient()` —É Python-—É –∑–∞ –∫—Ä–µ–∏—Ä–∞—ö–µ dataframe-–∞ —Å–∞—Å—Ç–æ—ò–∞–∫–∞. –û–≤–∞ —Ñ—É–Ω–∫—Ü–∏—ò–∞ —õ–µ –∑–∞–ø–æ—á–µ—Ç–∏ —É–∫–ª–∞—ö–∞—ö–µ–º –Ω–µ–∫–æ—Ä–∏—Å–Ω–µ –∫–æ–ª–æ–Ω–µ –∏ —Å–æ—Ä—Ç–∏—Ä–∞—ö–µ–º —Å–∞—Å—Ç–æ—ò–∞–∫–∞ –ø–æ —ö–∏—Ö–æ–≤–æ–º –±—Ä–æ—ò—É:

    ```python
    def create_ingredient_df(df):
        ingredient_df = df.T.drop(['cuisine','Unnamed: 0']).sum(axis=1).to_frame('value')

        ingredient_df = ingredient_df.sort_values(by='value', ascending=False,
        inplace=False)
        return ingredient_df
    ```

## –ë–∞–ª–∞–Ω—Å–∏—Ä–∞—ö–µ —Å–∫—É–ø–∞ –ø–æ–¥–∞—Ç–∞–∫–∞

–°–∞–¥–∞ –∫–∞–¥–∞ —Å—Ç–µ –æ—á–∏—Å—Ç–∏–ª–∏ –ø–æ–¥–∞—Ç–∫–µ, –∫–æ—Ä–∏—Å—Ç–∏—Ç–µ [SMOTE](https://imbalanced-learn.org/dev/references/generated/imblearn.over_sampling.SMOTE.html) - "–¢–µ—Ö–Ω–∏–∫–∞ —Å–∏–Ω—Ç–µ—Ç–∏—á–∫–æ–≥ –ø—Ä–µ–∫–æ–º–µ—Ä–Ω–æ–≥ —É–∑–æ—Ä–∫–æ–≤–∞—ö–∞ –º–∞—ö–∏–Ω—Å–∫–∏—Ö –∫–ª–∞—Å–∞" - –¥–∞ –∏—Ö –∏–∑–±–∞–ª–∞–Ω—Å–∏—Ä–∞—Ç–µ.
