### –ö–æ–Ω–≤–µ—Ä—Ç—É—ò—Ç–µ –≤–∞—à –º–æ–¥–µ–ª —É Onnx

–£–≤–µ—Ä–∏—Ç–µ —Å–µ –¥–∞ —Å—Ç–µ –∏–∑–≤—Ä—à–∏–ª–∏ –∫–æ–Ω–≤–µ—Ä–∑–∏—ò—É —Å–∞ –æ–¥–≥–æ–≤–∞—Ä–∞—ò—É—õ–∏–º –±—Ä–æ—ò–µ–º —Ç–µ–Ω–∑–æ—Ä–∞. –û–≤–∞—ò —Å–∫—É–ø –ø–æ–¥–∞—Ç–∞–∫–∞ –∏–º–∞ 380 —Å–∞—Å—Ç–æ—ò–∞–∫–∞, –ø–∞ —ò–µ –ø–æ—Ç—Ä–µ–±–Ω–æ –¥–∞ –Ω–∞–≤–µ–¥–µ—Ç–µ —Ç–∞—ò –±—Ä–æ—ò —É `FloatTensorType`:

1. –ö–æ–Ω–≤–µ—Ä—Ç—É—ò—Ç–µ –∫–æ—Ä–∏—Å—Ç–µ—õ–∏ –±—Ä–æ—ò —Ç–µ–Ω–∑–æ—Ä–∞ 380.

    ```python
    from skl2onnx import convert_sklearn
    from skl2onnx.common.data_types import FloatTensorType
    
    initial_type = [('float_input', FloatTensorType([None, 380]))]
    options = {id(model): {'nocl': True, 'zipmap': False}}
    ```

1. –ù–∞–ø—Ä–∞–≤–∏—Ç–µ onx –∏ —Å–∞—á—É–≤–∞—ò—Ç–µ –∫–∞–æ —Ñ–∞—ò–ª **model.onnx**:

    ```python
    onx = convert_sklearn(model, initial_types=initial_type, options=options)
    with open("./model.onnx", "wb") as f:
        f.write(onx.SerializeToString())
    ```

    > –ù–∞–ø–æ–º–µ–Ω–∞, –º–æ–∂–µ—Ç–µ –ø—Ä–æ—Å–ª–µ–¥–∏—Ç–∏ [–æ–ø—Ü–∏—ò–µ](https://onnx.ai/sklearn-onnx/parameterized.html) —É –≤–∞—à–µ–º —Å–∫—Ä–∏–ø—Ç—É –∑–∞ –∫–æ–Ω–≤–µ—Ä–∑–∏—ò—É. –£ –æ–≤–æ–º —Å–ª—É—á–∞—ò—É, –ø–æ—Å—Ç–∞–≤–∏–ª–∏ —Å–º–æ 'nocl' –¥–∞ –±—É–¥–µ True –∏ 'zipmap' –¥–∞ –±—É–¥–µ False. –ü–æ—à—Ç–æ —ò–µ –æ–≤–æ –º–æ–¥–µ–ª –∫–ª–∞—Å–∏—Ñ–∏–∫–∞—Ü–∏—ò–µ, –∏–º–∞—Ç–µ –æ–ø—Ü–∏—ò—É –¥–∞ —É–∫–ª–æ–Ω–∏—Ç–µ ZipMap –∫–æ—ò–∏ –ø—Ä–æ–∏–∑–≤–æ–¥–∏ –ª–∏—Å—Ç—É —Ä–µ—á–Ω–∏–∫–∞ (–Ω–∏—ò–µ –Ω–µ–æ–ø—Ö–æ–¥–Ω–æ). `nocl` —Å–µ –æ–¥–Ω–æ—Å–∏ –Ω–∞ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—ò–µ –æ –∫–ª–∞—Å–∏ –∫–æ—ò–µ —Å—É —É–∫—ô—É—á–µ–Ω–µ —É –º–æ–¥–µ–ª. –°–º–∞—ö–∏—Ç–µ –≤–µ–ª–∏—á–∏–Ω—É –≤–∞—à–µ–≥ –º–æ–¥–µ–ª–∞ –ø–æ—Å—Ç–∞–≤—ô–∞—ö–µ–º `nocl` –Ω–∞ 'True'.

–ò–∑–≤—Ä—à–∞–≤–∞—ö–µ —Ü–µ–ª–æ–≥ –Ω–æ—Ç–µ–±—É–∫–∞ —Å–∞–¥–∞ —õ–µ –Ω–∞–ø—Ä–∞–≤–∏—Ç–∏ Onnx –º–æ–¥–µ–ª –∏ —Å–∞—á—É–≤–∞—Ç–∏ –≥–∞ —É –æ–≤–∞—ò —Ñ–æ–ª–¥–µ—Ä.

## –ü—Ä–µ–≥–ª–µ–¥–∞—ò—Ç–µ –≤–∞—à –º–æ–¥–µ–ª
