–ü—Ä–µ –Ω–µ–≥–æ —à—Ç–æ —Ä–∞–∑—É–º–µ—Ç–µ –∑–Ω–∞—á–∞—ò SVR-–∞ —É –ø—Ä–µ–¥–≤–∏—í–∞—ö—É –≤—Ä–µ–º–µ–Ω—Å–∫–∏—Ö —Å–µ—Ä–∏—ò–∞, –µ–≤–æ –Ω–µ–∫–∏—Ö –≤–∞–∂–Ω–∏—Ö –∫–æ–Ω—Ü–µ–ø–∞—Ç–∞ –∫–æ—ò–µ —Ç—Ä–µ–±–∞ –¥–∞ –∑–Ω–∞—Ç–µ:

- **–†–µ–≥—Ä–µ—Å–∏—ò–∞:** –ù–∞–¥–≥–ª–µ–¥–∞–Ω–∞ —Ç–µ—Ö–Ω–∏–∫–∞ —É—á–µ—ö–∞ –∑–∞ –ø—Ä–µ–¥–≤–∏—í–∞—ö–µ –∫–æ–Ω—Ç–∏–Ω—É–∏—Ä–∞–Ω–∏—Ö –≤—Ä–µ–¥–Ω–æ—Å—Ç–∏ –∏–∑ –¥–∞—Ç–æ–≥ —Å–∫—É–ø–∞ —É–ª–∞–∑–∞. –ò–¥–µ—ò–∞ —ò–µ –¥–∞ —Å–µ —É–∫–ª–æ–ø–∏ –∫—Ä–∏–≤–∞ (–∏–ª–∏ –ø—Ä–∞–≤–∞) —É –ø—Ä–æ—Å—Ç–æ—Ä –∫–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∞ –∫–æ—ò–∞ —Å–∞–¥—Ä–∂–∏ –º–∞–∫—Å–∏–º–∞–ª–∞–Ω –±—Ä–æ—ò –ø–æ–¥–∞—Ç–∞–∫–∞. [–ö–ª–∏–∫–Ω–∏—Ç–µ –æ–≤–¥–µ](https://en.wikipedia.org/wiki/Regression_analysis) –∑–∞ –≤–∏—à–µ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—ò–∞.
- **Support Vector Machine (SVM):** –¢–∏–ø –º–æ–¥–µ–ª–∞ –º–∞—à–∏–Ω—Å–∫–æ–≥ —É—á–µ—ö–∞ —Å–∞ –Ω–∞–¥–∑–æ—Ä–æ–º –∫–æ—ò–∏ —Å–µ –∫–æ—Ä–∏—Å—Ç–∏ –∑–∞ –∫–ª–∞—Å–∏—Ñ–∏–∫–∞—Ü–∏—ò—É, —Ä–µ–≥—Ä–µ—Å–∏—ò—É –∏ –¥–µ—Ç–µ–∫—Ü–∏—ò—É –æ–¥—Å—Ç—É–ø–∞—ö–∞. –ú–æ–¥–µ–ª –ø—Ä–µ–¥—Å—Ç–∞–≤—ô–∞ —Ö–∏–ø–µ—Ä—Ä–∞–≤–∞–Ω —É –ø—Ä–æ—Å—Ç–æ—Ä—É –∫–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∞, –∫–æ—ò–∞ —É —Å–ª—É—á–∞—ò—É –∫–ª–∞—Å–∏—Ñ–∏–∫–∞—Ü–∏—ò–µ –¥–µ–ª—É—ò–µ –∫–∞–æ –≥—Ä–∞–Ω–∏—Ü–∞, –∞ —É —Å–ª—É—á–∞—ò—É —Ä–µ–≥—Ä–µ—Å–∏—ò–µ –∫–∞–æ –Ω–∞—ò–±–æ—ô–µ —É–∫–ª–æ–ø—ô–µ–Ω–∞ –ª–∏–Ω–∏—ò–∞. –£ SVM-—É —Å–µ –æ–±–∏—á–Ω–æ –∫–æ—Ä–∏—Å—Ç–∏ Kernel —Ñ—É–Ω–∫—Ü–∏—ò–∞ –∑–∞ —Ç—Ä–∞–Ω—Å—Ñ–æ—Ä–º–∞—Ü–∏—ò—É —Å–∫—É–ø–∞ –ø–æ–¥–∞—Ç–∞–∫–∞ —É –ø—Ä–æ—Å—Ç–æ—Ä —Å–∞ –≤–µ—õ–∏–º –±—Ä–æ—ò–µ–º –¥–∏–º–µ–Ω–∑–∏—ò–∞, –∫–∞–∫–æ –±–∏ –±–∏–ª–∏ –ª–∞–∫—à–µ —Ä–∞–∑–¥–≤–æ—ò–∏–≤–∏. [–ö–ª–∏–∫–Ω–∏—Ç–µ –æ–≤–¥–µ](https://en.wikipedia.org/wiki/Support-vector_machine) –∑–∞ –≤–∏—à–µ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—ò–∞ –æ SVM-—É.
- **Support Vector Regressor (SVR):** –¢–∏–ø SVM-–∞ –∫–æ—ò–∏ –ø—Ä–æ–Ω–∞–ª–∞–∑–∏ –Ω–∞—ò–±–æ—ô–µ —É–∫–ª–æ–ø—ô–µ–Ω—É –ª–∏–Ω–∏—ò—É (–∫–æ—ò–∞ —ò–µ —É —Å–ª—É—á–∞—ò—É SVM-–∞ —Ö–∏–ø–µ—Ä—Ä–∞–≤–∞–Ω) —Å–∞ –º–∞–∫—Å–∏–º–∞–ª–Ω–∏–º –±—Ä–æ—ò–µ–º –ø–æ–¥–∞—Ç–∞–∫–∞.

1. –î–æ–¥–µ–ª–∏—Ç–µ –¥–≤–æ–º–µ—Å–µ—á–Ω–∏ –ø–µ—Ä–∏–æ–¥ –æ–¥ 1. —Å–µ–ø—Ç–µ–º–±—Ä–∞ –¥–æ 31. –æ–∫—Ç–æ–±—Ä–∞ 2014. —Å–∫—É–ø—É –∑–∞ –æ–±—É–∫—É. –°–∫—É–ø –∑–∞ —Ç–µ—Å—Ç–∏—Ä–∞—ö–µ —õ–µ —É–∫—ô—É—á–∏–≤–∞—Ç–∏ –¥–≤–æ–º–µ—Å–µ—á–Ω–∏ –ø–µ—Ä–∏–æ–¥ –æ–¥ 1. –Ω–æ–≤–µ–º–±—Ä–∞ –¥–æ 31. –¥–µ—Ü–µ–º–±—Ä–∞ 2014: [^2]

   ```python
   train_start_dt = '2014-11-01 00:00:00'
   test_start_dt = '2014-12-30 00:00:00'
   ```

### –ü—Ä–∏–ø—Ä–µ–º–∞ –ø–æ–¥–∞—Ç–∞–∫–∞ –∑–∞ –æ–±—É–∫—É

–°–∞–¥–∞ —Ç—Ä–µ–±–∞ –¥–∞ –ø—Ä–∏–ø—Ä–µ–º–∏—Ç–µ –ø–æ–¥–∞—Ç–∫–µ –∑–∞ –æ–±—É–∫—É —Ç–∞–∫–æ —à—Ç–æ —õ–µ—Ç–µ –∏–∑–≤—Ä—à–∏—Ç–∏ —Ñ–∏–ª—Ç—Ä–∏—Ä–∞—ö–µ –∏ —Å–∫–∞–ª–∏—Ä–∞—ö–µ –ø–æ–¥–∞—Ç–∞–∫–∞. –§–∏–ª—Ç—Ä–∏—Ä–∞—ò—Ç–µ —Å–∫—É–ø –ø–æ–¥–∞—Ç–∞–∫–∞ —Ç–∞–∫–æ –¥–∞ —É–∫—ô—É—á—É—ò–µ —Å–∞–º–æ –ø–æ—Ç—Ä–µ–±–Ω–µ –≤—Ä–µ–º–µ–Ω—Å–∫–µ –ø–µ—Ä–∏–æ–¥–µ –∏ –∫–æ–ª–æ–Ω–µ, –∞ –∑–∞—Ç–∏–º –∏–∑–≤—Ä—à–∏—Ç–µ —Å–∫–∞–ª–∏—Ä–∞—ö–µ –∫–∞–∫–æ –±–∏ –ø–æ–¥–∞—Ü–∏ –±–∏–ª–∏ –ø—Ä–æ—ò–µ–∫—Ç–æ–≤–∞–Ω–∏ —É –∏–Ω—Ç–µ—Ä–≤–∞–ª—É 0,1.

1. –§–∏–ª—Ç—Ä–∏—Ä–∞—ò—Ç–µ –æ—Ä–∏–≥–∏–Ω–∞–ª–Ω–∏ —Å–∫—É–ø –ø–æ–¥–∞—Ç–∞–∫–∞ —Ç–∞–∫–æ –¥–∞ —É–∫—ô—É—á—É—ò–µ —Å–∞–º–æ –ø—Ä–µ—Ç—Ö–æ–¥–Ω–æ –Ω–∞–≤–µ–¥–µ–Ω–µ –≤—Ä–µ–º–µ–Ω—Å–∫–µ –ø–µ—Ä–∏–æ–¥–µ –ø–æ —Å–∫—É–ø—É –∏ —Å–∞–º–æ –ø–æ—Ç—Ä–µ–±–Ω—É –∫–æ–ª–æ–Ω—É 'load' –ø–ª—É—Å –¥–∞—Ç—É–º: [^2]

   ```python
   train = energy.copy()[(energy.index >= train_start_dt) & (energy.index < test_start_dt)][['load']]
   test = energy.copy()[energy.index >= test_start_dt][['load']]
   
   print('Training data shape: ', train.shape)
   print('Test data shape: ', test.shape)
   ```

   ```output
   Training data shape:  (1416, 1)
   Test data shape:  (48, 1)
   ```
   
2. –°–∫–∞–ª–∏—Ä–∞—ò—Ç–µ –ø–æ–¥–∞—Ç–∫–µ –∑–∞ –æ–±—É–∫—É —Ç–∞–∫–æ –¥–∞ –±—É–¥—É —É –æ–ø—Å–µ–≥—É (0, 1): [^2]

   ```python
   scaler = MinMaxScaler()
   train['load'] = scaler.fit_transform(train)
   ```
   
4. –°–∞–¥–∞ —Å–∫–∞–ª–∏—Ä–∞—ò—Ç–µ –ø–æ–¥–∞—Ç–∫–µ –∑–∞ —Ç–µ—Å—Ç–∏—Ä–∞—ö–µ: [^2]

   ```python
   test['load'] = scaler.transform(test)
   ```

–°–∞–¥–∞ —ò–µ –≤—Ä–µ–º–µ –∑–∞ –∏–º–ø–ª–µ–º–µ–Ω—Ç–∞—Ü–∏—ò—É SVR-–∞. –ó–∞ –≤–∏—à–µ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—ò–∞ –æ –æ–≤–æ—ò –∏–º–ø–ª–µ–º–µ–Ω—Ç–∞—Ü–∏—ò–∏, –º–æ–∂–µ—Ç–µ –ø–æ–≥–ª–µ–¥–∞—Ç–∏ [–æ–≤—É –¥–æ–∫—É–º–µ–Ω—Ç–∞—Ü–∏—ò—É](https://scikit-learn.org/stable/modules/generated/sklearn.svm.SVR.html). –ó–∞ –Ω–∞—à—É –∏–º–ø–ª–µ–º–µ–Ω—Ç–∞—Ü–∏—ò—É, —Å–ª–µ–¥–∏–º–æ –æ–≤–µ –∫–æ—Ä–∞–∫–µ:

1. –î–µ—Ñ–∏–Ω–∏—à–∏—Ç–µ –º–æ–¥–µ–ª –ø–æ–∑–∏–≤–∞—ö–µ–º `SVR()` –∏ –ø—Ä–æ—Å–ª–µ—í–∏–≤–∞—ö–µ–º —Ö–∏–ø–µ—Ä–ø–∞—Ä–∞–º–µ—Ç–∞—Ä–∞ –º–æ–¥–µ–ª–∞: kernel, gamma, c –∏ epsilon
2. –ü—Ä–∏–ø—Ä–µ–º–∏—Ç–µ –º–æ–¥–µ–ª –∑–∞ –ø–æ–¥–∞—Ç–∫–µ –∑–∞ –æ–±—É–∫—É –ø–æ–∑–∏–≤–∞—ö–µ–º —Ñ—É–Ω–∫—Ü–∏—ò–µ `fit()`
3. –ù–∞–ø—Ä–∞–≤–∏—Ç–µ –ø—Ä–µ–¥–≤–∏—í–∞—ö–∞ –ø–æ–∑–∏–≤–∞—ö–µ–º —Ñ—É–Ω–∫—Ü–∏—ò–µ `predict()`

–°–∞–¥–∞ –∫—Ä–µ–∏—Ä–∞–º–æ SVR –º–æ–¥–µ–ª. –û–≤–¥–µ –∫–æ—Ä–∏—Å—Ç–∏–º–æ [RBF kernel](https://scikit-learn.org/stable/modules/svm.html#parameters-of-the-rbf-kernel), –∏ –ø–æ—Å—Ç–∞–≤—ô–∞–º–æ —Ö–∏–ø–µ—Ä–ø–∞—Ä–∞–º–µ—Ç—Ä–µ gamma, C –∏ epsilon –Ω–∞ 0.5, 10 –∏ 0.05 —Ä–µ—Å–ø–µ–∫—Ç–∏–≤–Ω–æ.

