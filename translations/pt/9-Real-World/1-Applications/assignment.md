# Uma Caça ao Tesouro de ML

## Instruções

Nesta lição, aprendeste sobre muitos casos de uso reais que foram resolvidos utilizando ML clássico. Embora o uso de aprendizagem profunda, novas técnicas e ferramentas em IA, e o aproveitamento de redes neurais tenham ajudado a acelerar a produção de ferramentas para ajudar nestes setores, o ML clássico utilizando as técnicas deste currículo ainda tem um grande valor.

Neste exercício, imagina que estás a participar num hackathon. Usa o que aprendeste no currículo para propor uma solução utilizando ML clássico para resolver um problema num dos setores discutidos nesta lição. Cria uma apresentação onde discutas como vais implementar a tua ideia. Pontos extra se conseguires reunir dados de exemplo e construir um modelo de ML para apoiar o teu conceito!
