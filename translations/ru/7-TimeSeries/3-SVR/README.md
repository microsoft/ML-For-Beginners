### –†–µ–∞–ª–∏–∑–∞—Ü–∏—è SVR [^1]

–¢–µ–ø–µ—Ä—å –ø—Ä–∏—à–ª–æ –≤—Ä–µ–º—è —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å SVR. –ß—Ç–æ–±—ã —É–∑–Ω–∞—Ç—å –±–æ–ª—å—à–µ –æ–± —ç—Ç–æ–π —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏, –≤—ã –º–æ–∂–µ—Ç–µ –æ–±—Ä–∞—Ç–∏—Ç—å—Å—è –∫ [—ç—Ç–æ–π –¥–æ–∫—É–º–µ–Ω—Ç–∞—Ü–∏–∏](https://scikit-learn.org/stable/modules/generated/sklearn.svm.SVR.html). –î–ª—è –Ω–∞—à–µ–π —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ –º—ã —Å–ª–µ–¥—É–µ–º —Å–ª–µ–¥—É—é—â–∏–º —à–∞–≥–∞–º:

  1. –û–ø—Ä–µ–¥–µ–ª–∏—Ç–µ –º–æ–¥–µ–ª—å, –≤—ã–∑–≤–∞–≤ —Ñ—É–Ω–∫—Ü–∏—é `SVR()` and passing in the model hyperparameters: kernel, gamma, c and epsilon
  2. Prepare the model for the training data by calling the `fit()` function
  3. Make predictions calling the `predict()`

–¢–µ–ø–µ—Ä—å –º—ã —Å–æ–∑–¥–∞–µ–º –º–æ–¥–µ–ª—å SVR. –ó–¥–µ—Å—å –º—ã –∏—Å–ø–æ–ª—å–∑—É–µ–º [—è–¥—Ä–æ RBF](https://scikit-learn.org/stable/modules/svm.html#parameters-of-the-rbf-kernel) –∏ —É—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ–º –≥–∏–ø–µ—Ä–ø–∞—Ä–∞–º–µ—Ç—Ä—ã gamma, C –∏ epsilon —Ä–∞–≤–Ω—ã–º–∏ 0.5, 10 –∏ 0.05 —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ.

