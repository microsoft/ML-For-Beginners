## เดเดญเตเดฏเดพเดธเด - เดจเดฟเดเตเดเดณเตเดเต เดฎเตเดกเตฝ เดจเดฟเตผเดฎเตเดฎเดฟเดเตเดเตเด

เดเดชเตเดชเตเตพ, เดกเดพเดฑเตเดฑ เดชเดฐเดฟเดถเตเดฒเดจเดตเตเด เดชเดฐเดฟเดถเตเดงเดจเดฏเตเด เดเตเดฐเตเดชเตเดชเตเดเดณเดพเดฏเดฟ เดตเดฟเดญเดเดฟเดเตเดเต เดฎเตเดกเตฝ เดชเดฐเดฟเดถเตเดฒเดฟเดชเตเดชเดฟเดเตเดเดพเตป เดคเดฏเตเดฏเดพเดฑเดพเดเดพเด.

1. เดจเดฟเดเตเดเตพ เดชเดฐเดฟเดถเตเดฒเดฟเดชเตเดชเดฟเดเตเดเดพเตป เดเดเตเดฐเดนเดฟเดเตเดเตเดจเตเดจ เดฎเตเดจเตเดจเต เดซเตเดเตเดเดฑเตเดเตพ X เดตเตเดเตเดเดฑเดพเดฏเดฟ เดคเดฟเดฐเดเตเดเตเดเตเดเตเดเตเด, y เดตเตเดเตเดเตผ `Country` เดเดฏเดฟเดฐเดฟเดเตเดเตเด. เดจเดฟเดเตเดเตพเดเตเดเต `Seconds`, `Latitude`, `Longitude` เดจเตฝเดเตเดฎเตเดชเตเตพ เดเดฐเต เดฐเดพเดเตเดฏ เดเดกเดฟ เดฒเดญเดฟเดเตเดเดฃเด.

    ```python
    from sklearn.model_selection import train_test_split
    
    Selected_features = ['Seconds','Latitude','Longitude']
    
    X = ufos[Selected_features]
    y = ufos['Country']
    
    X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)
    ```

1. เดฒเตเดเดฟเดธเตเดฑเตเดฑเดฟเดเต เดฑเตเดเตเดฐเดทเตป เดเดชเดฏเตเดเดฟเดเตเดเต เดฎเตเดกเตฝ เดชเดฐเดฟเดถเตเดฒเดฟเดชเตเดชเดฟเดเตเดเตเด:

    ```python
    from sklearn.metrics import accuracy_score, classification_report
    from sklearn.linear_model import LogisticRegression
    model = LogisticRegression()
    model.fit(X_train, y_train)
    predictions = model.predict(X_test)
    
    print(classification_report(y_test, predictions))
    print('Predicted labels: ', predictions)
    print('Accuracy: ', accuracy_score(y_test, predictions))
    ```
