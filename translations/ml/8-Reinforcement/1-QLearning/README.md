เดธเดเดเตเดทเตเดชเดฟเดเตเดเต, เด เดธเตเดจเดพเดฐเดฟเดฏเตเดฏเตเดเต เดคเดจเตเดคเตเดฐเดตเตเด เดฒเดเตเดทเตเดฏเดตเตเด เดเดชเตเดฐเดเดพเดฐเด เดเดฃเต:

- **เดคเดจเตเดคเตเดฐเด**, เดจเดฎเตเดฎเตเดเต เดเดเดจเตเดฑเต (เดชเตเดฑเตเดฑเตผ) เดจเดฟเตผเดตเดเดฟเดเตเดเตเดจเตเดจเดคเต เดเดฐเต **เดจเดฏเด** (policy) เดเดจเตเดจ เดซเดเดเตเดทเตป เดตเดดเดฟ เดเดฃเต. เดเดฐเต เดจเดฏเด เดเดคเตเดเตเดเดฟเดฒเตเด เดธเตเดฑเตเดฑเตเดฑเตเดฑเดฟเตฝ เดชเตเดฐเดตเตผเดคเตเดคเดจเด เดคเดฟเดฐเดฟเดเต เดจเตฝเดเตเดจเตเดจ เดซเดเดเตเดทเดจเดพเดฃเต. เดจเดฎเตเดฎเตเดเต เดชเตเดฐเดถเตเดจเดคเตเดคเดฟเดฒเต เดธเตเดฑเตเดฑเตเดฑเตเดฑเต เดฌเตเตผเดกเตเด เดเดณเดฟเดเตเดเดพเดฐเดจเตเดฑเต เดจเดฟเดฒเดตเดฟเดฒเต เดธเตเดฅเดพเดจเดตเตเด เดเตพเดเตเดเตเดณเตเดณเตเดจเตเดจเต.

- **เดฒเดเตเดทเตเดฏเด**, เดฑเตเดเตปเดซเตเดดเตโเดธเตโเดฎเตเดจเตเดฑเต เดฒเตเดฃเดฟเดเดเดฟเดจเตเดฑเต เดฒเดเตเดทเตเดฏเด เดจเดฒเตเดฒ เดเดฐเต เดจเดฏเด เดชเดเดฟเดเตเดเต เดชเตเดฐเดถเตเดจเด เดซเดฒเดชเตเดฐเดฆเดฎเดพเดฏเดฟ เดชเดฐเดฟเดนเดฐเดฟเดเตเดเตเด เดเดฃเต. เดเดจเตเดจเดพเตฝ เดเดเดฟเดธเตเดฅเดพเดจเดฎเดพเดฏเดฟ, เดเดฑเตเดฑเดตเตเด เดฒเดณเดฟเดคเดฎเดพเดฏ เดจเดฏเด **เดฑเดพเตปเดกเด เดตเดพเดเตเดเต** เดเดจเตเดจเดคเดพเดฃเต.

## เดฑเดพเตปเดกเด เดตเดพเดเตเดเต

เดเดฆเตเดฏเด เดจเดพเด เดฑเดพเตปเดกเด เดตเดพเดเตเดเต เดคเดจเตเดคเตเดฐเด เดจเดเดชเตเดชเดฟเดฒเดพเดเตเดเดฟ เดชเตเดฐเดถเตเดจเด เดชเดฐเดฟเดนเดฐเดฟเดเตเดเดพเด. เดฑเดพเตปเดกเด เดตเดพเดเตเดเดฟเตฝ, เดเดจเตเดตเดฆเดฟเดเตเด เดชเตเดฐเดตเตผเดคเตเดคเดจเดเตเดเดณเดฟเตฝ เดจเดฟเดจเตเดจเต เดเดเตเดคเตเดค เดชเตเดฐเดตเตผเดคเตเดคเดจเด เดฏเดพเดฆเตเดเตเดเดฟเดเดฎเดพเดฏเดฟ เดคเดฟเดฐเดเตเดเตเดเตเดเตเดเตเด, เดเดชเตเดชเดฟเตพ เดเดฃเตเดเตเดคเตเดคเตเดจเตเดจเดคเต เดตเดฐเต (เดเตเดกเต เดฌเตเดฒเตเดเตเดเต 3).

1. เดคเดพเดดเต เดเตเดเตเดคเตเดค เดเตเดกเต เดเดชเดฏเตเดเดฟเดเตเดเต เดฑเดพเตปเดกเด เดตเดพเดเตเดเต เดจเดเดชเตเดชเดฟเดฒเดพเดเตเดเตเด:

    ```python
    def random_policy(m):
        return random.choice(list(actions))
    
    def walk(m,policy,start_position=None):
        n = 0 # เดเตเดตเดเตเดเดณเตเดเต เดเดฃเตเดฃเด
        # เดชเตเดฐเดพเดฐเดเดญ เดธเตเดฅเดพเดจเด เดธเดเตเดเดฎเดพเดเตเดเตเด
        if start_position:
            m.human = start_position 
        else:
            m.random_start()
        while True:
            if m.at() == Board.Cell.apple:
                return n # เดตเดฟเดเดฏเด!
            if m.at() in [Board.Cell.wolf, Board.Cell.water]:
                return -1 # เดตเตเดเตเดเดฏเดพเตฝ เดเดดเดฟเดเตเดเดชเตเดชเตเดเตเดเต เดเดฒเตเดฒเตเดเตเดเดฟเตฝ เดฎเตเดเตเดเดฟ
            while True:
                a = actions[policy(m)]
                new_pos = m.move_pos(m.human,a)
                if m.is_valid(new_pos) and m.at(new_pos)!=Board.Cell.water:
                    m.move(a) # เดฏเดฅเดพเตผเดคเตเดฅ เดจเตเดเตเดเด เดเตเดฏเตเดฏเตเด
                    break
            n+=1
    
    walk(m,random_policy)
    ```

1. เดตเดพเดเตเดเต เดชเดฐเตเดเตเดทเดฃเด เดชเดฒ เดคเดตเดฃ (เดเดฆเดพ: 100) เดเดเดฟเดเตเดเต เดซเดฒเดฎเดพเดฏ เดธเตเดฅเดฟเดคเดฟเดตเดฟเดตเดฐเดเตเดเดฃเดเตเดเตเดเตพ เดชเตเดฐเดฟเดจเตเดฑเต เดเตเดฏเตเดฏเตเด (เดเตเดกเต เดฌเตเดฒเตเดเตเดเต 4):

    ```python

(เดเตเดกเต เดฌเตเดฒเตเดเตเดเต 5)

```python
move_reward = -0.1
goal_reward = 10
end_reward = -10

def reward(m,pos=None):
    pos = pos or m.human
    if not m.is_valid(pos):
        return end_reward
    x = m.at(pos)
    if x==Board.Cell.water or x == Board.Cell.wolf:
        return end_reward
    if x==Board.Cell.apple:
        return goal_reward
    return move_reward
```
