# 嘗試不同的模型

## 說明

現在您已使用訓練過的回歸模型建立了一個網頁應用程式，請使用之前回歸課程中的其中一個模型重新製作此網頁應用程式。您可以保留原有的風格，或者設計出不同的樣式以反映南瓜數據。請注意更改輸入以符合模型的訓練方法。
