‡∞≤‡∞ï‡±ç‡∞∑‡±ç‡∞Ø‡∞Ç **‡∞∞‡∞ø‡∞µ‡∞æ‡∞∞‡±ç‡∞°‡±ç ‡∞´‡∞Ç‡∞ï‡±ç‡∞∑‡∞®‡±ç** ‡∞∞‡±Ç‡∞™‡∞Ç‡∞≤‡±ã ‡∞®‡∞ø‡∞∞‡±ç‡∞µ‡∞ö‡∞ø‡∞Ç‡∞ö‡∞µ‡∞ö‡±ç‡∞ö‡±Å, ‡∞á‡∞¶‡∞ø ‡∞™‡±ç‡∞∞‡∞§‡∞ø ‡∞∏‡±ç‡∞ü‡±á‡∞ü‡±ç‚Äå‡∞ï‡±Å ‡∞ï‡±ä‡∞Ç‡∞§ ‡∞∏‡±ç‡∞ï‡±ã‡∞∞‡±ç ‡∞µ‡∞ø‡∞≤‡±Å‡∞µ‡∞®‡±Å ‡∞á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø. ‡∞∏‡∞Ç‡∞ñ‡±ç‡∞Ø ‡∞é‡∞ï‡±ç‡∞ï‡±Å‡∞µ‡±à‡∞§‡±á, ‡∞∞‡∞ø‡∞µ‡∞æ‡∞∞‡±ç‡∞°‡±ç ‡∞´‡∞Ç‡∞ï‡±ç‡∞∑‡∞®‡±ç ‡∞Æ‡±Ü‡∞∞‡±Å‡∞ó‡±à‡∞®‡∞¶‡∞ø.         return end_reward
    x = m.at(pos)
    if x==Board.Cell.water or x == Board.Cell.wolf:

    if x==Board.Cell.apple:
        return goal_reward
    return move_reward
