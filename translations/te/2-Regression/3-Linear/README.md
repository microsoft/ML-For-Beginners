‡∞ö‡∞ø‡∞µ‡∞∞‡∞ó‡∞æ, ‡∞≤‡±Ä‡∞®‡∞ø‡∞Ø‡∞∞‡±ç ‡∞∞‡∞ø‡∞ó‡±ç‡∞∞‡±Ü‡∞∑‡∞®‡±ç ‡∞Æ‡±ã‡∞°‡∞≤‡±ç ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£ ‡∞∞‡±Ü‡∞Ç‡∞°‡±Å ‡∞ï‡±ã‡∞°‡±ç ‡∞≤‡±à‡∞®‡±ç‡∞≤‡∞≤‡±ã‡∞®‡±á ‡∞ú‡∞∞‡±Å‡∞ó‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø. ‡∞Æ‡±á‡∞Æ‡±Å `LinearRegression` ‡∞Ü‡∞¨‡±ç‡∞ú‡±Ü‡∞ï‡±ç‡∞ü‡±ç‚Äå‡∞®‡±Å ‡∞®‡∞ø‡∞∞‡±ç‡∞µ‡∞ö‡∞ø‡∞Ç‡∞ö‡∞ø, `fit` ‡∞Æ‡±Ü‡∞•‡∞°‡±ç ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞Ç‡∞ö‡∞ø ‡∞Æ‡∞æ ‡∞°‡±á‡∞ü‡∞æ‡∞ï‡±Å ‡∞Ö‡∞®‡±Å‡∞ó‡±Å‡∞£‡∞Ç‡∞ó‡∞æ ‡∞ö‡±á‡∞∏‡±ç‡∞§‡∞æ‡∞Æ‡±Å:


‡∞Æ‡∞æ ‡∞Æ‡±ã‡∞°‡∞≤‡±ç ‡∞é‡∞Ç‡∞§ ‡∞ñ‡∞ö‡±ç‡∞ö‡∞ø‡∞§‡∞Ç‡∞ó‡∞æ ‡∞â‡∞Ç‡∞¶‡±ã ‡∞ö‡±Ç‡∞°‡∞ü‡∞æ‡∞®‡∞ø‡∞ï‡∞ø, ‡∞Æ‡±á‡∞Æ‡±Å ‡∞ü‡±Ü‡∞∏‡±ç‡∞ü‡±ç ‡∞°‡±á‡∞ü‡∞æ‡∞∏‡±Ü‡∞ü‡±ç‚Äå‡∞™‡±à ‡∞ß‡∞∞‡∞≤‡∞®‡±Å ‡∞Ö‡∞Ç‡∞ö‡∞®‡∞æ ‡∞µ‡±á‡∞Ø‡∞µ‡∞ö‡±ç‡∞ö‡±Å, ‡∞Ü‡∞™‡±à ‡∞Æ‡∞æ ‡∞Ö‡∞Ç‡∞ö‡∞®‡∞æ‡∞≤‡±Å ‡∞Ü‡∞∂‡∞ø‡∞Ç‡∞ö‡∞ø‡∞® ‡∞µ‡∞ø‡∞≤‡±Å‡∞µ‡∞≤‡∞ï‡±Å ‡∞é‡∞Ç‡∞§ ‡∞¶‡∞ó‡±ç‡∞ó‡∞∞‡∞ó‡∞æ ‡∞â‡∞®‡±ç‡∞®‡∞æ‡∞Ø‡±ã ‡∞ï‡±ä‡∞≤‡∞µ‡∞µ‡∞ö‡±ç‡∞ö‡±Å. ‡∞á‡∞¶‡∞ø ‡∞Æ‡∞ø‡∞®‡±Ä ‡∞∏‡±ç‡∞ï‡±ç‡∞µ‡±á‡∞∞‡±ç ‡∞é‡∞∞‡±ç‡∞∞‡∞∞‡±ç (MSE) ‡∞Æ‡±Ü‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç‡∞∏‡±ç ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞Ç‡∞ö‡∞ø ‡∞ö‡±á‡∞Ø‡∞µ‡∞ö‡±ç‡∞ö‡±Å, ‡∞á‡∞¶‡∞ø ‡∞Ü‡∞∂‡∞ø‡∞Ç‡∞ö‡∞ø‡∞® ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å ‡∞Ö‡∞Ç‡∞ö‡∞®‡∞æ ‡∞µ‡∞ø‡∞≤‡±Å‡∞µ‡∞≤ ‡∞Æ‡∞ß‡±ç‡∞Ø ‡∞Ö‡∞®‡±ç‡∞®‡∞ø ‡∞ö‡∞¶‡∞∞‡∞™‡±Å ‡∞§‡±á‡∞°‡∞æ‡∞≤ ‡∞∏‡∞ó‡∞ü‡±Å.

‡∞Æ‡∞æ ‡∞™‡±ä‡∞∞‡∞™‡∞æ‡∞ü‡±Å ‡∞∏‡±Å‡∞Æ‡∞æ‡∞∞‡±Å 2 ‡∞™‡∞æ‡∞Ø‡∞ø‡∞Ç‡∞ü‡±ç‡∞≤ ‡∞ö‡±Å‡∞ü‡±ç‡∞ü‡±Ç ‡∞â‡∞Ç‡∞¶‡∞ø, ‡∞á‡∞¶‡∞ø ‡∞∏‡±Å‡∞Æ‡∞æ‡∞∞‡±Å 17%. ‡∞ö‡∞æ‡∞≤‡∞æ ‡∞¨‡∞æ‡∞ó‡∞æ‡∞≤‡±á‡∞¶‡±Å. ‡∞Æ‡±ã‡∞°‡∞≤‡±ç ‡∞®‡∞æ‡∞£‡±ç‡∞Ø‡∞§‡∞ï‡±Å ‡∞Æ‡∞∞‡±ã ‡∞∏‡±Ç‡∞ö‡∞ø‡∞ï **‡∞®‡∞ø‡∞∞‡±ç‡∞£‡∞Ø ‡∞∏‡∞π‡∞ó‡±Å‡∞£‡∞Ç** (coefficient of determination), ‡∞á‡∞¶‡∞ø ‡∞á‡∞≤‡∞æ ‡∞™‡±ä‡∞Ç‡∞¶‡∞µ‡∞ö‡±ç‡∞ö‡±Å:

## ‡∞Ö‡∞®‡±ç‡∞®‡∞ø‡∞Ç‡∞ü‡∞ø‡∞®‡±Ä ‡∞ï‡∞≤‡∞ø‡∞™‡∞ø

‡∞â‡∞§‡±ç‡∞§‡∞Æ ‡∞Æ‡±ã‡∞°‡∞≤‡±ç ‡∞§‡∞Ø‡∞æ‡∞∞‡±Å‡∞ö‡±á‡∞Ø‡∞°‡∞æ‡∞®‡∞ø‡∞ï‡∞ø, ‡∞Æ‡±á‡∞Æ‡±Å ‡∞™‡±à ‡∞â‡∞¶‡∞æ‡∞π‡∞∞‡∞£‡∞≤‡±ã‡∞®‡∞ø ‡∞ï‡∞≤‡∞ø‡∞™‡∞ø‡∞® (‡∞µ‡∞®‡±ç-‡∞π‡∞æ‡∞ü‡±ç ‡∞é‡∞Ç‡∞ï‡±ã‡∞°‡±Ü‡∞°‡±ç ‡∞µ‡∞∞‡±ç‡∞ó‡±Ä‡∞ï‡±É‡∞§ + ‡∞∏‡∞Ç‡∞ñ‡±ç‡∞Ø‡∞æ‡∞§‡±ç‡∞Æ‡∞ï) ‡∞°‡±á‡∞ü‡∞æ‡∞®‡±Å ‡∞™‡∞æ‡∞≤‡∞ø‡∞®‡±ã‡∞Æ‡∞ø‡∞Ø‡∞≤‡±ç ‡∞∞‡∞ø‡∞ó‡±ç‡∞∞‡±Ü‡∞∑‡∞®‡±ç‚Äå‡∞§‡±ã ‡∞ï‡∞≤‡∞ø‡∞™‡∞ø ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞Ç‡∞ö‡∞µ‡∞ö‡±ç‡∞ö‡±Å. ‡∞Æ‡±Ä ‡∞∏‡±å‡∞ï‡∞∞‡±ç‡∞Ø‡∞æ‡∞®‡∞ø‡∞ï‡∞ø ‡∞™‡±Ç‡∞∞‡±ç‡∞§‡∞ø ‡∞ï‡±ã‡∞°‡±ç ‡∞á‡∞ï‡±ç‡∞ï‡∞° ‡∞â‡∞Ç‡∞¶‡∞ø:

```python
# ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£ ‡∞°‡±á‡∞ü‡∞æ‡∞®‡±Å ‡∞∏‡±Ü‡∞ü‡±ç ‡∞ö‡±á‡∞Ø‡∞Ç‡∞°‡∞ø
X = pd.get_dummies(new_pumpkins['Variety']) \
        .join(new_pumpkins['Month']) \
        .join(pd.get_dummies(new_pumpkins['City'])) \
        .join(pd.get_dummies(new_pumpkins['Package']))
y = new_pumpkins['Price']

# ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£-‡∞™‡∞∞‡±Ä‡∞ï‡±ç‡∞∑ ‡∞µ‡∞ø‡∞≠‡∞ú‡∞® ‡∞ö‡±á‡∞Ø‡∞Ç‡∞°‡∞ø
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)

# ‡∞™‡±à‡∞™‡±ç‚Äå‡∞≤‡±à‡∞®‡±ç‚Äå‡∞®‡±Å ‡∞∏‡±Ü‡∞ü‡±ç ‡∞ö‡±á‡∞∏‡∞ø ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£ ‡∞á‡∞µ‡±ç‡∞µ‡∞Ç‡∞°‡∞ø

# ‡∞™‡∞∞‡±Ä‡∞ï‡±ç‡∞∑ ‡∞°‡±á‡∞ü‡∞æ‡∞ï‡±Å ‡∞´‡∞≤‡∞ø‡∞§‡∞æ‡∞≤‡∞®‡±Å ‡∞Ö‡∞Ç‡∞ö‡∞®‡∞æ ‡∞µ‡±á‡∞Ø‡∞Ç‡∞°‡∞ø
pred = pipeline.predict(X_test)

# MSE ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å ‡∞®‡∞ø‡∞∞‡±ç‡∞£‡∞Ø‡∞æ‡∞®‡±ç‡∞®‡∞ø ‡∞≤‡±Ü‡∞ï‡±ç‡∞ï‡∞ø‡∞Ç‡∞ö‡∞Ç‡∞°‡∞ø
mse = np.sqrt(mean_squared_error(y_test,pred))
print(f'Mean error: {mse:3.3} ({mse/np.mean(pred)*100:3.3}%)')

score = pipeline.score(X_train,y_train)
print('Model determination: ', score)
```
