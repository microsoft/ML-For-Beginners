1. ‡∞Ü‡∞∞‡±ç‡∞°‡∞ø‡∞®‡∞≤‡±ç ‡∞é‡∞Ç‡∞ï‡±ã‡∞°‡∞∞‡±ç: ‡∞á‡∞¶‡∞ø ‡∞Ü‡∞∞‡±ç‡∞°‡∞ø‡∞®‡∞≤‡±ç ‡∞µ‡±á‡∞∞‡∞ø‡∞Ø‡∞¨‡±Å‡∞≤‡±ç‡∞∏‡±ç ‡∞ï‡±Å ‡∞∏‡∞∞‡∞ø‡∞™‡±ã‡∞§‡±Å‡∞Ç‡∞¶‡∞ø, ‡∞Ö‡∞µ‡∞ø ‡∞µ‡∞∞‡±ç‡∞ó‡±Ä‡∞ï‡∞∞‡∞£ ‡∞µ‡±á‡∞∞‡∞ø‡∞Ø‡∞¨‡±Å‡∞≤‡±ç‡∞∏‡±ç, ‡∞µ‡∞æ‡∞ü‡∞ø ‡∞°‡±á‡∞ü‡∞æ ‡∞§‡∞æ‡∞∞‡±ç‡∞ï‡∞ø‡∞ï ‡∞ï‡±ç‡∞∞‡∞Æ‡∞Ç‡∞≤‡±ã ‡∞â‡∞Ç‡∞ü‡±Å‡∞Ç‡∞¶‡∞ø, ‡∞Æ‡∞® ‡∞°‡±á‡∞ü‡∞æ‡∞∏‡±Ü‡∞ü‡±ç ‡∞≤‡±ã `Item Size` ‡∞ï‡∞æ‡∞≤‡∞Æ‡±ç ‡∞≤‡∞æ‡∞Ç‡∞ü‡∞ø‡∞¶‡∞ø. ‡∞á‡∞¶‡∞ø ‡∞™‡±ç‡∞∞‡∞§‡∞ø ‡∞µ‡∞∞‡±ç‡∞ó‡∞Ç ‡∞í‡∞ï ‡∞∏‡∞Ç‡∞ñ‡±ç‡∞Ø‡∞§‡±ã ‡∞™‡±ç‡∞∞‡∞æ‡∞§‡∞ø‡∞®‡∞ø‡∞ß‡±ç‡∞Ø‡∞Ç ‡∞µ‡∞π‡∞ø‡∞Ç‡∞ö‡±á ‡∞Æ‡±ç‡∞Ø‡∞æ‡∞™‡∞ø‡∞Ç‡∞ó‡±ç ‡∞∏‡±É‡∞∑‡±ç‡∞ü‡∞ø‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø, ‡∞Ö‡∞¶‡∞ø ‡∞Ü ‡∞ï‡∞æ‡∞≤‡∞Æ‡±ç ‡∞≤‡±ã ‡∞µ‡∞∞‡±ç‡∞ó‡∞Ç ‡∞ï‡±ç‡∞∞‡∞Æ‡∞Ç.

2. ‡∞µ‡∞∞‡±ç‡∞ó‡±Ä‡∞ï‡∞∞‡∞£ ‡∞é‡∞Ç‡∞ï‡±ã‡∞°‡∞∞‡±ç: ‡∞á‡∞¶‡∞ø ‡∞®‡∞æ‡∞Æ‡∞ø‡∞®‡∞≤‡±ç ‡∞µ‡±á‡∞∞‡∞ø‡∞Ø‡∞¨‡±Å‡∞≤‡±ç‡∞∏‡±ç ‡∞ï‡±Å ‡∞∏‡∞∞‡∞ø‡∞™‡±ã‡∞§‡±Å‡∞Ç‡∞¶‡∞ø, ‡∞Ö‡∞µ‡∞ø ‡∞µ‡∞∞‡±ç‡∞ó‡±Ä‡∞ï‡∞∞‡∞£ ‡∞µ‡±á‡∞∞‡∞ø‡∞Ø‡∞¨‡±Å‡∞≤‡±ç‡∞∏‡±ç, ‡∞µ‡∞æ‡∞ü‡∞ø ‡∞°‡±á‡∞ü‡∞æ ‡∞§‡∞æ‡∞∞‡±ç‡∞ï‡∞ø‡∞ï ‡∞ï‡±ç‡∞∞‡∞Æ‡∞Ç‡∞≤‡±ã ‡∞â‡∞Ç‡∞°‡∞¶‡±Å, ‡∞Æ‡∞® ‡∞°‡±á‡∞ü‡∞æ‡∞∏‡±Ü‡∞ü‡±ç ‡∞≤‡±ã `Item Size` ‡∞§‡∞™‡±ç‡∞™ ‡∞Ö‡∞®‡±ç‡∞®‡∞ø ‡∞´‡±Ä‡∞ö‡∞∞‡±ç‡∞∏‡±ç. ‡∞á‡∞¶‡∞ø ‡∞µ‡∞®‡±ç-‡∞π‡∞æ‡∞ü‡±ç ‡∞é‡∞Ç‡∞ï‡±ã‡∞°‡∞ø‡∞Ç‡∞ó‡±ç, ‡∞Ö‡∞Ç‡∞ü‡±á ‡∞™‡±ç‡∞∞‡∞§‡∞ø ‡∞µ‡∞∞‡±ç‡∞ó‡∞Ç ‡∞í‡∞ï ‡∞¨‡±à‡∞®‡∞∞‡±Ä ‡∞ï‡∞æ‡∞≤‡∞Æ‡±ç ‡∞§‡±ã ‡∞™‡±ç‡∞∞‡∞æ‡∞§‡∞ø‡∞®‡∞ø‡∞ß‡±ç‡∞Ø‡∞Ç ‡∞µ‡∞π‡∞ø‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø: ‡∞é‡∞Ç‡∞ï‡±ã‡∞°‡±Ü‡∞°‡±ç ‡∞µ‡±á‡∞∞‡∞ø‡∞Ø‡∞¨‡±Å‡∞≤‡±ç 1 ‡∞Ö‡∞µ‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø ‡∞Ü ‡∞™‡∞Ç‡∞™‡±ç‡∞ï‡∞ø‡∞®‡±ç ‡∞Ü Variety ‡∞ï‡∞ø ‡∞ö‡±Ü‡∞Ç‡∞¶‡∞ø‡∞§‡±á, ‡∞≤‡±á‡∞ï‡∞™‡±ã‡∞§‡±á 0.

‡∞§‡∞∞‡±ç‡∞µ‡∞æ‡∞§, `ColumnTransformer` ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞Ç‡∞ö‡∞ø ‡∞Ö‡∞®‡±á‡∞ï ‡∞é‡∞Ç‡∞ï‡±ã‡∞°‡∞∞‡±ç‡∞≤‡∞®‡±Å ‡∞í‡∞ï‡±á ‡∞¶‡∞∂‡∞≤‡±ã ‡∞ï‡∞≤‡∞ø‡∞™‡∞ø ‡∞∏‡∞∞‡±à‡∞® ‡∞ï‡∞æ‡∞≤‡∞Æ‡±ç‡∞∏‡±ç ‡∞ï‡±Å ‡∞µ‡∞∞‡±ç‡∞§‡∞ø‡∞Ç‡∞™‡∞ú‡±á‡∞∏‡±ç‡∞§‡∞æ‡∞∞‡±Å.

    from sklearn.preprocessing import LabelEncoder

üéì f1-score: (2 * precision * recall)/(precision + recall) precision ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å recall ‡∞Ø‡±ä‡∞ï‡±ç‡∞ï ‡∞¨‡∞∞‡±Å‡∞µ‡±Å ‡∞ï‡∞≤‡∞ø‡∞ó‡∞ø‡∞® ‡∞∏‡∞ó‡∞ü‡±Å, ‡∞â‡∞§‡±ç‡∞§‡∞Æ‡∞Ç 1 ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å ‡∞ö‡±Ü‡∞§‡±ç‡∞§‡∞¶‡∞ø 0

üéì Support: ‡∞™‡±ç‡∞∞‡∞§‡∞ø ‡∞≤‡±á‡∞¨‡±Å‡∞≤‡±ç ‡∞§‡∞ø‡∞∞‡∞ø‡∞ó‡∞ø ‡∞™‡±ä‡∞Ç‡∞¶‡∞ø‡∞® ‡∞∏‡∞Ç‡∞¶‡∞∞‡±ç‡∞≠‡∞æ‡∞≤ ‡∞∏‡∞Ç‡∞ñ‡±ç‡∞Ø

üéì Accuracy: (TP + TN)/(TP + TN + FP + FN) ‡∞®‡∞Æ‡±Ç‡∞®‡∞æ ‡∞ï‡±ã‡∞∏‡∞Ç ‡∞∏‡∞∞‡∞ø‡∞ó‡±ç‡∞ó‡∞æ ‡∞Ö‡∞Ç‡∞ö‡∞®‡∞æ ‡∞µ‡±á‡∞∏‡∞ø‡∞® ‡∞≤‡±á‡∞¨‡±Å‡∞≤‡±ç‡∞∏‡±ç ‡∞∂‡∞æ‡∞§‡∞Ç

üéì Macro Avg: ‡∞™‡±ç‡∞∞‡∞§‡∞ø ‡∞≤‡±á‡∞¨‡±Å‡∞≤‡±ç ‡∞ï‡±ã‡∞∏‡∞Ç ‡∞¨‡∞∞‡±Å‡∞µ‡±Å ‡∞≤‡±á‡∞®‡∞ø ‡∞∏‡∞ó‡∞ü‡±Å ‡∞ó‡∞£‡∞®, ‡∞≤‡±á‡∞¨‡±Å‡∞≤‡±ç ‡∞Ö‡∞∏‡∞Æ‡∞§‡±Å‡∞≤‡±ç‡∞Ø‡∞§‡∞®‡±Å ‡∞™‡∞∞‡∞ø‡∞ó‡∞£‡∞≤‡±ã‡∞ï‡∞ø ‡∞§‡±Ä‡∞∏‡±Å‡∞ï‡±ã‡∞¶‡±Å

üéì Weighted Avg: ‡∞™‡±ç‡∞∞‡∞§‡∞ø ‡∞≤‡±á‡∞¨‡±Å‡∞≤‡±ç ‡∞ï‡±ã‡∞∏‡∞Ç ‡∞∏‡∞ó‡∞ü‡±Å ‡∞ó‡∞£‡∞®, ‡∞≤‡±á‡∞¨‡±Å‡∞≤‡±ç ‡∞Ö‡∞∏‡∞Æ‡∞§‡±Å‡∞≤‡±ç‡∞Ø‡∞§‡∞®‡±Å ‡∞™‡∞∞‡∞ø‡∞ó‡∞£‡∞≤‡±ã‡∞ï‡∞ø ‡∞§‡±Ä‡∞∏‡±Å‡∞ï‡±Å‡∞®‡∞ø ‡∞µ‡∞æ‡∞ü‡∞ø support (‡∞™‡±ç‡∞∞‡∞§‡∞ø ‡∞≤‡±á‡∞¨‡±Å‡∞≤‡±ç ‡∞ï‡±ã‡∞∏‡∞Ç ‡∞®‡∞ø‡∞ú‡∞Æ‡±à‡∞® ‡∞â‡∞¶‡∞æ‡∞π‡∞∞‡∞£‡∞≤ ‡∞∏‡∞Ç‡∞ñ‡±ç‡∞Ø) ‡∞§‡±ã ‡∞¨‡∞∞‡±Å‡∞µ‡±Å ‡∞µ‡±á‡∞∏‡±ç‡∞§‡±Å‡∞Ç‡∞¶‡∞ø

‚úÖ ‡∞Æ‡±Ä‡∞∞‡±Å ‡∞Æ‡±Ä ‡∞Æ‡±ã‡∞°‡∞≤‡±ç ‡∞§‡∞™‡±ç‡∞™‡±Å ‡∞®‡±Ü‡∞ó‡∞ü‡∞ø‡∞µ‡±ç‡∞∏‡±ç ‡∞∏‡∞Ç‡∞ñ‡±ç‡∞Ø‡∞®‡±Å ‡∞§‡∞ó‡±ç‡∞ó‡∞ø‡∞Ç‡∞ö‡∞æ‡∞≤‡∞®‡∞ø ‡∞Ö‡∞®‡±Å‡∞ï‡±Å‡∞Ç‡∞ü‡±á ‡∞è ‡∞Æ‡±Ü‡∞ü‡±ç‡∞∞‡∞ø‡∞ï‡±ç ‡∞®‡±Å ‡∞ó‡∞Æ‡∞®‡∞ø‡∞Ç‡∞ö‡∞æ‡∞≤‡∞ø ‡∞Ö‡∞®‡±Å‡∞ï‡±Å‡∞Ç‡∞ü‡±Å‡∞®‡±ç‡∞®‡∞æ‡∞∞‡∞æ?

## ‡∞à ‡∞Æ‡±ã‡∞°‡∞≤‡±ç ‡∞Ø‡±ä‡∞ï‡±ç‡∞ï ROC ‡∞µ‡∞ï‡±ç‡∞∞‡∞æ‡∞®‡±ç‡∞®‡∞ø ‡∞¶‡±É‡∞∂‡±ç‡∞Ø‡±Ä‡∞ï‡∞∞‡∞ø‡∞Ç‡∞ö‡∞Ç‡∞°‡∞ø

[![ML for beginners - Analyzing Logistic Regression Performance with ROC Curves](https://img.youtube.com/vi/GApO575jTA0/0.jpg)](https://youtu.be/GApO575jTA0 "ML for beginners - Analyzing Logistic Regression Performance with ROC Curves")

> üé• ‡∞™‡±à ‡∞ö‡∞ø‡∞§‡±ç‡∞∞‡∞æ‡∞®‡±ç‡∞®‡∞ø ‡∞ï‡±ç‡∞≤‡∞ø‡∞ï‡±ç ‡∞ö‡±á‡∞∏‡∞ø ROC ‡∞µ‡∞ï‡±ç‡∞∞‡∞æ‡∞≤‡∞™‡±à ‡∞ö‡∞ø‡∞®‡±ç‡∞® ‡∞µ‡±Ä‡∞°‡∞ø‡∞Ø‡±ã ‡∞Ö‡∞µ‡∞≤‡±ã‡∞ï‡∞®‡∞Ç ‡∞ö‡±Ç‡∞°‡∞Ç‡∞°‡∞ø

‡∞Æ‡∞∞‡∞ø ‡∞í‡∞ï ‡∞¶‡±É‡∞∂‡±ç‡∞Ø‡±Ä‡∞ï‡∞∞‡∞£ ‡∞ö‡±á‡∞¶‡±ç‡∞¶‡∞æ‡∞Ç, ‡∞¶‡±Ä‡∞®‡∞ø‡∞®‡∞ø 'ROC' ‡∞µ‡∞ï‡±ç‡∞∞‡∞Ç ‡∞Ö‡∞Ç‡∞ü‡∞æ‡∞∞‡±Å:

```python

import matplotlib.pyplot as plt
%matplotlib inline

```
