‡∞Æ‡±Ä‡∞∞‡±Å ‡∞á‡∞ü‡±Ä‡∞µ‡∞≤ ‡∞í‡∞ï ‡∞™‡∞æ‡∞†‡∞Ç‡∞≤‡±ã ‡∞≤‡∞æ‡∞ú‡∞ø‡∞∏‡±ç‡∞ü‡∞ø‡∞ï‡±ç ‡∞∞‡∞ø‡∞ó‡±ç‡∞∞‡±Ü‡∞∑‡∞®‡±ç ‡∞ó‡±Å‡∞∞‡∞ø‡∞Ç‡∞ö‡∞ø ‡∞®‡±á‡∞∞‡±ç‡∞ö‡±Å‡∞ï‡±Å‡∞®‡±ç‡∞®‡∞Ç‡∞¶‡±Å‡∞®, ‡∞Æ‡±ä‡∞¶‡∞ü‡∞ø ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£ ‡∞™‡±ç‡∞∞‡∞Ø‡∞§‡±ç‡∞®‡∞æ‡∞®‡∞ø‡∞ï‡∞ø ‡∞≤‡∞æ‡∞ú‡∞ø‡∞∏‡±ç‡∞ü‡∞ø‡∞ï‡±ç ‡∞∞‡∞ø‡∞ó‡±ç‡∞∞‡±Ü‡∞∑‡∞®‡±ç‚Äå‡∞™‡±à ‡∞¶‡±É‡∞∑‡±ç‡∞ü‡∞ø ‡∞™‡±Ü‡∞ü‡±ç‡∞ü‡∞µ‡∞ö‡±ç‡∞ö‡±Å.

## ‡∞µ‡±ç‡∞Ø‡∞æ‡∞Ø‡∞æ‡∞Æ‡∞Ç - ‡∞≤‡∞æ‡∞ú‡∞ø‡∞∏‡±ç‡∞ü‡∞ø‡∞ï‡±ç ‡∞∞‡∞ø‡∞ó‡±ç‡∞∞‡±Ü‡∞∑‡∞®‡±ç ‡∞µ‡∞∞‡±ç‡∞§‡∞ø‡∞Ç‡∞ö‡∞Ç‡∞°‡∞ø

‡∞Æ‡±Ä‡∞∞‡±Å ‡∞Æ‡∞≤‡±ç‡∞ü‡∞ø‡∞ï‡±ç‡∞≤‡∞æ‡∞∏‡±ç ‡∞∏‡∞Ç‡∞¶‡∞∞‡±ç‡∞≠‡∞æ‡∞®‡±ç‡∞®‡∞ø ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞∏‡±ç‡∞§‡±Å‡∞®‡±ç‡∞®‡∞Ç‡∞¶‡±Å‡∞®, ‡∞è _‡∞™‡∞¶‡±ç‡∞ß‡∞§‡∞ø_ ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞Ç‡∞ö‡∞æ‡∞≤‡±ã ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å ‡∞è _solver_ ‡∞∏‡±Ü‡∞ü‡±ç ‡∞ö‡±á‡∞Ø‡∞æ‡∞≤‡±ã ‡∞é‡∞Ç‡∞ö‡±Å‡∞ï‡±ã‡∞µ‡∞æ‡∞≤‡∞ø. ‡∞Æ‡∞≤‡±ç‡∞ü‡∞ø‡∞ï‡±ç‡∞≤‡∞æ‡∞∏‡±ç ‡∞∏‡±Ü‡∞ü‡±ç‡∞ü‡∞ø‡∞Ç‡∞ó‡±ç‚Äå‡∞§‡±ã ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å **liblinear** solver‡∞§‡±ã LogisticRegression ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞Ç‡∞ö‡∞ø ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£ ‡∞á‡∞µ‡±ç‡∞µ‡∞Ç‡∞°‡∞ø.

1. multi_class ‡∞®‡±Å `ovr`‡∞ó‡∞æ ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å solver ‡∞®‡±Å `liblinear`‡∞ó‡∞æ ‡∞∏‡±Ü‡∞ü‡±ç ‡∞ö‡±á‡∞∏‡∞ø ‡∞≤‡∞æ‡∞ú‡∞ø‡∞∏‡±ç‡∞ü‡∞ø‡∞ï‡±ç ‡∞∞‡∞ø‡∞ó‡±ç‡∞∞‡±Ü‡∞∑‡∞®‡±ç ‡∞∏‡±É‡∞∑‡±ç‡∞ü‡∞ø‡∞Ç‡∞ö‡∞Ç‡∞°‡∞ø:

    ```python

    ```
