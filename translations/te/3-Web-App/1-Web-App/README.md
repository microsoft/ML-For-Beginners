## ‡∞µ‡±ç‡∞Ø‡∞æ‡∞Ø‡∞æ‡∞Æ‡∞Ç - ‡∞Æ‡±Ä ‡∞Æ‡±ã‡∞°‡∞≤‡±ç‚Äå‡∞®‡±Å ‡∞®‡∞ø‡∞∞‡±ç‡∞Æ‡∞ø‡∞Ç‡∞ö‡∞Ç‡∞°‡∞ø

‡∞á‡∞™‡±ç‡∞™‡±Å‡∞°‡±Å ‡∞Æ‡±Ä‡∞∞‡±Å ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£ ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å ‡∞™‡∞∞‡±Ä‡∞ï‡±ç‡∞∑‡∞æ ‡∞∏‡∞Æ‡±Ç‡∞π‡∞æ‡∞≤‡±Å‡∞ó‡∞æ ‡∞°‡±á‡∞ü‡∞æ‡∞®‡±Å ‡∞µ‡∞ø‡∞≠‡∞ú‡∞ø‡∞Ç‡∞ö‡∞ø ‡∞Æ‡±ã‡∞°‡∞≤‡±ç ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£‡∞ï‡±Å ‡∞∏‡∞ø‡∞¶‡±ç‡∞ß‡∞Ç ‡∞ï‡∞æ‡∞µ‡∞ö‡±ç‡∞ö‡±Å.

1. ‡∞Æ‡±Ä‡∞∞‡±Å ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£ ‡∞á‡∞µ‡±ç‡∞µ‡∞¶‡∞≤‡∞ö‡±Å‡∞ï‡±Å‡∞®‡±ç‡∞® ‡∞Æ‡±Ç‡∞°‡±Å ‡∞≤‡∞ï‡±ç‡∞∑‡∞£‡∞æ‡∞≤‡∞®‡±Å X ‡∞µ‡±Ü‡∞ï‡±ç‡∞ü‡∞∞‡±ç‚Äå‡∞ó‡∞æ ‡∞é‡∞Ç‡∞ö‡±Å‡∞ï‡±ã‡∞Ç‡∞°‡∞ø, ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å y ‡∞µ‡±Ü‡∞ï‡±ç‡∞ü‡∞∞‡±ç `Country` ‡∞Ö‡∞µ‡±Å‡∞§‡±Å‡∞Ç‡∞¶‡∞ø. ‡∞Æ‡±Ä‡∞∞‡±Å `Seconds`, `Latitude` ‡∞Æ‡∞∞‡∞ø‡∞Ø‡±Å `Longitude` ‡∞®‡±Å ‡∞á‡∞®‡±ç‚Äå‡∞™‡±Å‡∞ü‡±ç‚Äå‡∞ó‡∞æ ‡∞á‡∞ö‡±ç‡∞ö‡∞ø ‡∞¶‡±á‡∞∂ ID ‡∞®‡±Å ‡∞™‡±ä‡∞Ç‡∞¶‡∞ó‡∞≤‡∞ó‡∞æ‡∞≤‡∞ø.

    from sklearn.model_selection import train_test_split

    X = ufos[Selected_features]

1. ‡∞≤‡∞æ‡∞ú‡∞ø‡∞∏‡±ç‡∞ü‡∞ø‡∞ï‡±ç ‡∞∞‡∞ø‡∞ó‡±ç‡∞∞‡±Ü‡∞∑‡∞®‡±ç ‡∞â‡∞™‡∞Ø‡±ã‡∞ó‡∞ø‡∞Ç‡∞ö‡∞ø ‡∞Æ‡±Ä ‡∞Æ‡±ã‡∞°‡∞≤‡±ç‚Äå‡∞®‡±Å ‡∞∂‡∞ø‡∞ï‡±ç‡∞∑‡∞£ ‡∞á‡∞µ‡±ç‡∞µ‡∞Ç‡∞°‡∞ø:
