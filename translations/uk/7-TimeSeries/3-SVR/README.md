### SVR —É –∫–æ–Ω—Ç–µ–∫—Å—Ç—ñ —á–∞—Å–æ–≤–∏—Ö —Ä—è–¥—ñ–≤ [^1]

–ü–µ—Ä–µ–¥ —Ç–∏–º —è–∫ –∑—Ä–æ–∑—É–º—ñ—Ç–∏ –≤–∞–∂–ª–∏–≤—ñ—Å—Ç—å SVR —É –ø—Ä–æ–≥–Ω–æ–∑—É–≤–∞–Ω–Ω—ñ —á–∞—Å–æ–≤–∏—Ö —Ä—è–¥—ñ–≤, –æ—Å—å –∫—ñ–ª—å–∫–∞ –≤–∞–∂–ª–∏–≤–∏—Ö –∫–æ–Ω—Ü–µ–ø—Ü—ñ–π, —è–∫—ñ –≤–∞–º –ø–æ—Ç—Ä—ñ–±–Ω–æ –∑–Ω–∞—Ç–∏:

- **–†–µ–≥—Ä–µ—Å—ñ—è:** –¢–µ—Ö–Ω—ñ–∫–∞ –Ω–∞–≤—á–∞–Ω–Ω—è –∑ —É—á–∏—Ç–µ–ª–µ–º –¥–ª—è –ø—Ä–æ–≥–Ω–æ–∑—É–≤–∞–Ω–Ω—è –±–µ–∑–ø–µ—Ä–µ—Ä–≤–Ω–∏—Ö –∑–Ω–∞—á–µ–Ω—å –Ω–∞ –æ—Å–Ω–æ–≤—ñ –∑–∞–¥–∞–Ω–æ–≥–æ –Ω–∞–±–æ—Ä—É –≤—Ö—ñ–¥–Ω–∏—Ö –¥–∞–Ω–∏—Ö. –Ü–¥–µ—è –ø–æ–ª—è–≥–∞—î —É –ø–æ–±—É–¥–æ–≤—ñ –∫—Ä–∏–≤–æ—ó (–∞–±–æ –ª—ñ–Ω—ñ—ó) —É –ø—Ä–æ—Å—Ç–æ—Ä—ñ –æ–∑–Ω–∞–∫, —è–∫–∞ –º–∞—î –º–∞–∫—Å–∏–º–∞–ª—å–Ω—É –∫—ñ–ª—å–∫—ñ—Å—Ç—å —Ç–æ—á–æ–∫ –¥–∞–Ω–∏—Ö. [–ù–∞—Ç–∏—Å–Ω—ñ—Ç—å —Ç—É—Ç](https://en.wikipedia.org/wiki/Regression_analysis) –¥–ª—è –æ—Ç—Ä–∏–º–∞–Ω–Ω—è –¥–æ–¥–∞—Ç–∫–æ–≤–æ—ó —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—ó.
- **–ú–∞—à–∏–Ω–∞ –æ–ø–æ—Ä–Ω–∏—Ö –≤–µ–∫—Ç–æ—Ä—ñ–≤ (SVM):** –¢–∏–ø –º–æ–¥–µ–ª—ñ –º–∞—à–∏–Ω–Ω–æ–≥–æ –Ω–∞–≤—á–∞–Ω–Ω—è –∑ —É—á–∏—Ç–µ–ª–µ–º, —è–∫–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è –¥–ª—è –∫–ª–∞—Å–∏—Ñ—ñ–∫–∞—Ü—ñ—ó, —Ä–µ–≥—Ä–µ—Å—ñ—ó —Ç–∞ –≤–∏—è–≤–ª–µ–Ω–Ω—è –∞–Ω–æ–º–∞–ª—ñ–π. –ú–æ–¥–µ–ª—å —î –≥—ñ–ø–µ—Ä–ø–ª–æ—â–∏–Ω–æ—é —É –ø—Ä–æ—Å—Ç–æ—Ä—ñ –æ–∑–Ω–∞–∫, —è–∫–∞ —É –≤–∏–ø–∞–¥–∫—É –∫–ª–∞—Å–∏—Ñ—ñ–∫–∞—Ü—ñ—ó –≤–∏—Å—Ç—É–ø–∞—î —è–∫ –º–µ–∂–∞, –∞ —É –≤–∏–ø–∞–¥–∫—É —Ä–µ–≥—Ä–µ—Å—ñ—ó ‚Äî —è–∫ –ª—ñ–Ω—ñ—è –Ω–∞–π–∫—Ä–∞—â–æ–≥–æ –ø—ñ–¥—Ö–æ–¥—É. –£ SVM –∑–∞–∑–≤–∏—á–∞–π –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è —Ñ—É–Ω–∫—Ü—ñ—è —è–¥—Ä–∞ –¥–ª—è –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è –Ω–∞–±–æ—Ä—É –¥–∞–Ω–∏—Ö —É –ø—Ä–æ—Å—Ç—ñ—Ä –∑ –±—ñ–ª—å—à–æ—é –∫—ñ–ª—å–∫—ñ—Å—Ç—é –≤–∏–º—ñ—Ä—ñ–≤, —â–æ–± –≤–æ–Ω–∏ –º–æ–≥–ª–∏ –±—É—Ç–∏ –ª–µ–≥–∫–æ —Ä–æ–∑–¥—ñ–ª–µ–Ω—ñ. [–ù–∞—Ç–∏—Å–Ω—ñ—Ç—å —Ç—É—Ç](https://en.wikipedia.org/wiki/Support-vector_machine) –¥–ª—è –æ—Ç—Ä–∏–º–∞–Ω–Ω—è –¥–æ–¥–∞—Ç–∫–æ–≤–æ—ó —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—ó –ø—Ä–æ SVM.

### –ß–æ–º—É SVR? [^1]

–£ –ø–æ–ø–µ—Ä–µ–¥–Ω—å–æ–º—É —É—Ä–æ—Ü—ñ –≤–∏ –¥—ñ–∑–Ω–∞–ª–∏—Å—è –ø—Ä–æ ARIMA, —è–∫–∞ —î –¥—É–∂–µ —É—Å–ø—ñ—à–Ω–∏–º —Å—Ç–∞—Ç–∏—Å—Ç–∏—á–Ω–∏–º –ª—ñ–Ω—ñ–π–Ω–∏–º –º–µ—Ç–æ–¥–æ–º –¥–ª—è –ø—Ä–æ–≥–Ω–æ–∑—É–≤–∞–Ω–Ω—è –¥–∞–Ω–∏—Ö —á–∞—Å–æ–≤–∏—Ö —Ä—è–¥—ñ–≤. –û–¥–Ω–∞–∫ —É –±–∞–≥–∞—Ç—å–æ—Ö –≤–∏–ø–∞–¥–∫–∞—Ö –¥–∞–Ω—ñ —á–∞—Å–æ–≤–∏—Ö —Ä—è–¥—ñ–≤ –º–∞—é—Ç—å *–Ω–µ–ª—ñ–Ω—ñ–π–Ω—ñ—Å—Ç—å*, —è–∫—É –Ω–µ–º–æ–∂–ª–∏–≤–æ –≤—ñ–¥–æ–±—Ä–∞–∑–∏—Ç–∏ –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –ª—ñ–Ω—ñ–π–Ω–∏—Ö –º–æ–¥–µ–ª–µ–π. –£ —Ç–∞–∫–∏—Ö –≤–∏–ø–∞–¥–∫–∞—Ö –∑–¥–∞—Ç–Ω—ñ—Å—Ç—å SVM –≤—Ä–∞—Ö–æ–≤—É–≤–∞—Ç–∏ –Ω–µ–ª—ñ–Ω—ñ–π–Ω—ñ—Å—Ç—å –¥–∞–Ω–∏—Ö –¥–ª—è –∑–∞–¥–∞—á —Ä–µ–≥—Ä–µ—Å—ñ—ó —Ä–æ–±–∏—Ç—å SVR —É—Å–ø—ñ—à–Ω–∏–º —É –ø—Ä–æ–≥–Ω–æ–∑—É–≤–∞–Ω–Ω—ñ —á–∞—Å–æ–≤–∏—Ö —Ä—è–¥—ñ–≤.

### –ü—ñ–¥–≥–æ—Ç–æ–≤–∫–∞ –¥–∞–Ω–∏—Ö –¥–ª—è –Ω–∞–≤—á–∞–Ω–Ω—è

–¢–µ–ø–µ—Ä –≤–∞–º –ø–æ—Ç—Ä—ñ–±–Ω–æ –ø—ñ–¥–≥–æ—Ç—É–≤–∞—Ç–∏ –¥–∞–Ω—ñ –¥–ª—è –Ω–∞–≤—á–∞–Ω–Ω—è, –≤–∏–∫–æ–Ω–∞–≤—à–∏ —Ñ—ñ–ª—å—Ç—Ä–∞—Ü—ñ—é —Ç–∞ –º–∞—Å—à—Ç–∞–±—É–≤–∞–Ω–Ω—è –¥–∞–Ω–∏—Ö. –í—ñ–¥—Ñ—ñ–ª—å—Ç—Ä—É–π—Ç–µ –≤–∞—à –Ω–∞–±—ñ—Ä –¥–∞–Ω–∏—Ö, —â–æ–± –≤–∫–ª—é—á–∏—Ç–∏ –ª–∏—à–µ –ø–æ—Ç—Ä—ñ–±–Ω—ñ –ø–µ—Ä—ñ–æ–¥–∏ —á–∞—Å—É —Ç–∞ —Å—Ç–æ–≤–ø—Ü—ñ, –∞ —Ç–∞–∫–æ–∂ –≤–∏–∫–æ–Ω–∞–π—Ç–µ –º–∞—Å—à—Ç–∞–±—É–≤–∞–Ω–Ω—è, —â–æ–± –¥–∞–Ω—ñ –±—É–ª–∏ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω—ñ –≤ —ñ–Ω—Ç–µ—Ä–≤–∞–ª—ñ 0,1.

1. –í—ñ–¥—Ñ—ñ–ª—å—Ç—Ä—É–π—Ç–µ –æ—Ä–∏–≥—ñ–Ω–∞–ª—å–Ω–∏–π –Ω–∞–±—ñ—Ä –¥–∞–Ω–∏—Ö, —â–æ–± –≤–∫–ª—é—á–∏—Ç–∏ –ª–∏—à–µ –∑–∞–∑–Ω–∞—á–µ–Ω—ñ –ø–µ—Ä—ñ–æ–¥–∏ —á–∞—Å—É –¥–ª—è –∫–æ–∂–Ω–æ–≥–æ –Ω–∞–±–æ—Ä—É —Ç–∞ –ª–∏—à–µ –ø–æ—Ç—Ä—ñ–±–Ω–∏–π —Å—Ç–æ–≤–ø–µ—Ü—å 'load' –ø–ª—é—Å –¥–∞—Ç—É: [^2]


   ```python
   scaler = MinMaxScaler()
   ```

4. –¢–µ–ø–µ—Ä –º–∞—Å—à—Ç–∞–±—É–π—Ç–µ —Ç–µ—Å—Ç–æ–≤—ñ –¥–∞–Ω—ñ: [^2]

–¢–µ–ø–µ—Ä –Ω–∞—Å—Ç–∞–≤ —á–∞—Å —Ä–µ–∞–ª—ñ–∑—É–≤–∞—Ç–∏ SVR. –©–æ–± –¥—ñ–∑–Ω–∞—Ç–∏—Å—è –±—ñ–ª—å—à–µ –ø—Ä–æ —Ü—é —Ä–µ–∞–ª—ñ–∑–∞—Ü—ñ—é, –≤–∏ –º–æ–∂–µ—Ç–µ –∑–≤–µ—Ä–Ω—É—Ç–∏—Å—è –¥–æ [—Ü—ñ—î—ó –¥–æ–∫—É–º–µ–Ω—Ç–∞—Ü—ñ—ó](https://scikit-learn.org/stable/modules/generated/sklearn.svm.SVR.html). –î–ª—è –Ω–∞—à–æ—ó —Ä–µ–∞–ª—ñ–∑–∞—Ü—ñ—ó –º–∏ –¥–æ—Ç—Ä–∏–º—É—î–º–æ—Å—è —Ç–∞–∫–∏—Ö –∫—Ä–æ–∫—ñ–≤:

  1. –í–∏–∑–Ω–∞—á—Ç–µ –º–æ–¥–µ–ª—å, –≤–∏–∫–ª–∏–∫–∞–≤—à–∏ `SVR()` —ñ –ø–µ—Ä–µ–¥–∞–≤—à–∏ –≥—ñ–ø–µ—Ä–ø–∞—Ä–∞–º–µ—Ç—Ä–∏ –º–æ–¥–µ–ª—ñ: kernel, gamma, c —Ç–∞ epsilon
  2. –ü—ñ–¥–≥–æ—Ç—É–π—Ç–µ –º–æ–¥–µ–ª—å –¥–ª—è –Ω–∞–≤—á–∞–ª—å–Ω–∏—Ö –¥–∞–Ω–∏—Ö, –≤–∏–∫–ª–∏–∫–∞–≤—à–∏ —Ñ—É–Ω–∫—Ü—ñ—é `fit()`
  3. –ó—Ä–æ–±—ñ—Ç—å –ø—Ä–æ–≥–Ω–æ–∑–∏, –≤–∏–∫–ª–∏–∫–∞–≤—à–∏ —Ñ—É–Ω–∫—Ü—ñ—é `predict()`

–¢–µ–ø–µ—Ä –º–∏ —Å—Ç–≤–æ—Ä—é—î–º–æ –º–æ–¥–µ–ª—å SVR. –¢—É—Ç –º–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î–º–æ [RBF kernel](https://scikit-learn.org/stable/modules/svm.html#parameters-of-the-rbf-kernel) —ñ –≤—Å—Ç–∞–Ω–æ–≤–ª—é—î–º–æ –≥—ñ–ø–µ—Ä–ø–∞—Ä–∞–º–µ—Ç—Ä–∏ gamma, C —Ç–∞ epsilon —è–∫ 0.5, 10 —Ç–∞ 0.05 –≤—ñ–¥–ø–æ–≤—ñ–¥–Ω–æ.

