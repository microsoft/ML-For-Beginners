## –î—ñ—ó —Ç–∞ –ø–æ–ª—ñ—Ç–∏–∫–∞

–£ –Ω–∞—à–æ–º—É –ø—Ä–∏–∫–ª–∞–¥—ñ –º–µ—Ç–∞ –ü–µ—Ç—Ä–∏–∫–∞ ‚Äî –∑–Ω–∞–π—Ç–∏ —è–±–ª—É–∫–æ, —É–Ω–∏–∫–∞—é—á–∏ –≤–æ–≤–∫–∞ —Ç–∞ —ñ–Ω—à–∏—Ö –ø–µ—Ä–µ—à–∫–æ–¥. –î–ª—è —Ü—å–æ–≥–æ –≤—ñ–Ω –º–æ–∂–µ –ø—Ä–æ—Å—Ç–æ —Ö–æ–¥–∏—Ç–∏, –ø–æ–∫–∏ –Ω–µ –∑–Ω–∞–π–¥–µ —è–±–ª—É–∫–æ.

–¢–∞–∫–∏–º —á–∏–Ω–æ–º, —É –±—É–¥—å-—è–∫—ñ–π –ø–æ–∑–∏—Ü—ñ—ó –≤—ñ–Ω –º–æ–∂–µ –≤–∏–±—Ä–∞—Ç–∏ –æ–¥–Ω—É –∑ –Ω–∞—Å—Ç—É–ø–Ω–∏—Ö –¥—ñ–π: –≤–≥–æ—Ä—É, –≤–Ω–∏–∑, –≤–ª—ñ–≤–æ —ñ –≤–ø—Ä–∞–≤–æ.

- **–ú–µ—Ç–∞** –Ω–∞–≤—á–∞–Ω–Ω—è –∑ –ø—ñ–¥–∫—Ä—ñ–ø–ª–µ–Ω–Ω—è–º ‚Äî –∑—Ä–µ—à—Ç–æ—é –Ω–∞–≤—á–∏—Ç–∏—Å—è —Ö–æ—Ä–æ—à—ñ–π –ø–æ–ª—ñ—Ç–∏—Ü—ñ, —è–∫–∞ –¥–æ–∑–≤–æ–ª–∏—Ç—å –Ω–∞–º –µ—Ñ–µ–∫—Ç–∏–≤–Ω–æ –≤–∏—Ä—ñ—à—É–≤–∞—Ç–∏ –ø—Ä–æ–±–ª–µ–º—É. –û–¥–Ω–∞–∫, —è–∫ –±–∞–∑–æ–≤–∏–π –≤–∞—Ä—ñ–∞–Ω—Ç, —Ä–æ–∑–≥–ª—è–Ω–µ–º–æ –Ω–∞–π–ø—Ä–æ—Å—Ç—ñ—à—É –ø–æ–ª—ñ—Ç–∏–∫—É, —è–∫—É –Ω–∞–∑–∏–≤–∞—é—Ç—å **–≤–∏–ø–∞–¥–∫–æ–≤–æ—é –ø—Ä–æ–≥—É–ª—è–Ω–∫–æ—é**.

## –í–∏–ø–∞–¥–∫–æ–≤–∞ –ø—Ä–æ–≥—É–ª—è–Ω–∫–∞

–°–ø–æ—á–∞—Ç–∫—É –≤–∏—Ä—ñ—à–∏–º–æ –Ω–∞—à—É –ø—Ä–æ–±–ª–µ–º—É, —Ä–µ–∞–ª—ñ–∑—É–≤–∞–≤—à–∏ —Å—Ç—Ä–∞—Ç–µ–≥—ñ—é –≤–∏–ø–∞–¥–∫–æ–≤–æ—ó –ø—Ä–æ–≥—É–ª—è–Ω–∫–∏. –£ –≤–∏–ø–∞–¥–∫–æ–≤—ñ–π –ø—Ä–æ–≥—É–ª—è–Ω—Ü—ñ –º–∏ –±—É–¥–µ–º–æ –≤–∏–ø–∞–¥–∫–æ–≤–æ –≤–∏–±–∏—Ä–∞—Ç–∏ –Ω–∞—Å—Ç—É–ø–Ω—É –¥—ñ—é –∑ –¥–æ–∑–≤–æ–ª–µ–Ω–∏—Ö –¥—ñ–π, –ø–æ–∫–∏ –Ω–µ –¥–æ—Å—è–≥–Ω–µ–º–æ —è–±–ª—É–∫–∞ (–±–ª–æ–∫ –∫–æ–¥—É 3).

1. –†–µ–∞–ª—ñ–∑—É–π—Ç–µ –≤–∏–ø–∞–¥–∫–æ–≤—É –ø—Ä–æ–≥—É–ª—è–Ω–∫—É –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –Ω–∞–≤–µ–¥–µ–Ω–æ–≥–æ –Ω–∏–∂—á–µ –∫–æ–¥—É:

    ```python
    def random_policy(m):
        return random.choice(list(actions))
    
    def walk(m,policy,start_position=None):
        n = 0 # number of steps
        # set initial position
        if start_position:
            m.human = start_position 
        else:
            m.random_start()
        while True:
            if m.at() == Board.Cell.apple:
                return n # success!
            if m.at() in [Board.Cell.wolf, Board.Cell.water]:
                return -1 # eaten by wolf or drowned
            while True:
                a = actions[policy(m)]
                new_pos = m.move_pos(m.human,a)
                if m.is_valid(new_pos) and m.at(new_pos)!=Board.Cell.water:
                    m.move(a) # do the actual move
                    break
            n+=1
    
    walk(m,random_policy)
    ```

    –í–∏–∫–ª–∏–∫ `walk` –ø–æ–≤–∏–Ω–µ–Ω –ø–æ–≤–µ—Ä–Ω—É—Ç–∏ –¥–æ–≤–∂–∏–Ω—É –≤—ñ–¥–ø–æ–≤—ñ–¥–Ω–æ–≥–æ —à–ª—è—Ö—É, —è–∫–∞ –º–æ–∂–µ –∑–º—ñ–Ω—é–≤–∞—Ç–∏—Å—è –≤—ñ–¥ –æ–¥–Ω–æ–≥–æ –∑–∞–ø—É—Å–∫—É –¥–æ —ñ–Ω—à–æ–≥–æ.

1. –ó–∞–ø—É—Å—Ç—ñ—Ç—å –µ–∫—Å–ø–µ—Ä–∏–º–µ–Ω—Ç –ø—Ä–æ–≥—É–ª—è–Ω–∫–∏ –∫—ñ–ª—å–∫–∞ —Ä–∞–∑—ñ–≤ (—Å–∫–∞–∂—ñ–º–æ, 100) —ñ –Ω–∞–¥—Ä—É–∫—É–π—Ç–µ –æ—Ç—Ä–∏–º–∞–Ω—É —Å—Ç–∞—Ç–∏—Å—Ç–∏–∫—É (–±–ª–æ–∫ –∫–æ–¥—É 4):

    ```python
    def print_statistics(policy):
        s,w,n = 0,0,0
        for _ in range(100):
            z = walk(m,policy)
            if z<0:
                w+=1
            else:
                s += z
                n += 1
        print(f"Average path length = {s/n}, eaten by wolf: {w} times")
    
    print_statistics(random_policy)
    ```

## –§—É–Ω–∫—Ü—ñ—è –≤–∏–Ω–∞–≥–æ—Ä–æ–¥–∏

–©–æ–± –∑—Ä–æ–±–∏—Ç–∏ –Ω–∞—à—É –ø–æ–ª—ñ—Ç–∏–∫—É –±—ñ–ª—å—à —Ä–æ–∑—É–º–Ω–æ—é, –Ω–∞–º –ø–æ—Ç—Ä—ñ–±–Ω–æ –∑—Ä–æ–∑—É–º—ñ—Ç–∏, —è–∫—ñ –∫—Ä–æ–∫–∏ —î "–∫—Ä–∞—â–∏–º–∏" –∑–∞ —ñ–Ω—à—ñ. –î–ª—è —Ü—å–æ–≥–æ –Ω–∞–º –ø–æ—Ç—Ä—ñ–±–Ω–æ –≤–∏–∑–Ω–∞—á–∏—Ç–∏ –Ω–∞—à—É –º–µ—Ç—É.

–ú–µ—Ç–∞ –º–æ–∂–µ –±—É—Ç–∏ –≤–∏–∑–Ω–∞—á–µ–Ω–∞ —É –≤–∏–≥–ª—è–¥—ñ **—Ñ—É–Ω–∫—Ü—ñ—ó –≤–∏–Ω–∞–≥–æ—Ä–æ–¥–∏**, —è–∫–∞ –ø–æ–≤–µ—Ä—Ç–∞—î –ø–µ–≤–Ω–µ –∑–Ω–∞—á–µ–Ω–Ω—è –æ—Ü—ñ–Ω–∫–∏ –¥–ª—è –∫–æ–∂–Ω–æ–≥–æ —Å—Ç–∞–Ω—É. –ß–∏–º –±—ñ–ª—å—à–µ —á–∏—Å–ª–æ, —Ç–∏–º –∫—Ä–∞—â–∞ —Ñ—É–Ω–∫—Ü—ñ—è –≤–∏–Ω–∞–≥–æ—Ä–æ–¥–∏. (–±–ª–æ–∫ –∫–æ–¥—É 5)

```python
move_reward = -0.1
goal_reward = 10
end_reward = -10

def reward(m,pos=None):
    pos = pos or m.human
    if not m.is_valid(pos):
        return end_reward
    x = m.at(pos)
    if x==Board.Cell.water or x == Board.Cell.wolf:
        return end_reward
    if x==Board.Cell.apple:
        return goal_reward
    return move_reward
```

–¶—ñ–∫–∞–≤–æ, —â–æ —É –±—ñ–ª—å—à–æ—Å—Ç—ñ –≤–∏–ø–∞–¥–∫—ñ–≤ *–º–∏ –æ—Ç—Ä–∏–º—É—î–º–æ –∑–Ω–∞—á–Ω—É –≤–∏–Ω–∞–≥–æ—Ä–æ–¥—É –ª–∏—à–µ –Ω–∞–ø—Ä–∏–∫—ñ–Ω—Ü—ñ –≥—Ä–∏*. –¶–µ –æ–∑–Ω–∞—á–∞—î, —â–æ –Ω–∞—à –∞–ª–≥–æ—Ä–∏—Ç–º –ø–æ–≤–∏–Ω–µ–Ω —è–∫–æ—Å—å –∑–∞–ø–∞–º‚Äô—è—Ç–æ–≤—É–≤–∞—Ç–∏ "—Ö–æ—Ä–æ—à—ñ" –∫—Ä–æ–∫–∏, —è–∫—ñ –ø—Ä–∏–∑–≤–æ–¥—è—Ç—å –¥–æ –ø–æ–∑–∏—Ç–∏–≤–Ω–æ—ó –≤–∏–Ω–∞–≥–æ—Ä–æ–¥–∏ –≤ –∫—ñ–Ω—Ü—ñ, —ñ –∑–±—ñ–ª—å—à—É–≤–∞—Ç–∏ —ó—Ö –≤–∞–∂–ª–∏–≤—ñ—Å—Ç—å. –ê–Ω–∞–ª–æ–≥—ñ—á–Ω–æ, –≤—Å—ñ –∫—Ä–æ–∫–∏, —è–∫—ñ –ø—Ä–∏–∑–≤–æ–¥—è—Ç—å –¥–æ –ø–æ–≥–∞–Ω–∏—Ö —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ñ–≤, —Å–ª—ñ–¥ –∑–Ω–µ–æ—Ö–æ—á—É–≤–∞—Ç–∏.

## Q-Learning

–ê–ª–≥–æ—Ä–∏—Ç–º, —è–∫–∏–π –º–∏ —Ç—É—Ç –æ–±–≥–æ–≤–æ—Ä–∏–º–æ, –Ω–∞–∑–∏–≤–∞—î—Ç—å—Å—è **Q-Learning**. –£ —Ü—å–æ–º—É –∞–ª–≥–æ—Ä–∏—Ç–º—ñ –ø–æ–ª—ñ—Ç–∏–∫–∞ –≤–∏–∑–Ω–∞—á–∞—î—Ç—å—Å—è —Ñ—É–Ω–∫—Ü—ñ—î—é (–∞–±–æ —Å—Ç—Ä—É–∫—Ç—É—Ä–æ—é –¥–∞–Ω–∏—Ö), —è–∫–∞ –Ω–∞–∑–∏–≤–∞—î—Ç—å—Å—è **Q-—Ç–∞–±–ª–∏—Ü—è**. –í–æ–Ω–∞ –∑–∞–ø–∏—Å—É—î "–∫–æ—Ä–∏—Å–Ω—ñ—Å—Ç—å" –∫–æ–∂–Ω–æ—ó –¥—ñ—ó –≤ –¥–∞–Ω–æ–º—É —Å—Ç–∞–Ω—ñ.

–í–æ–Ω–∞ –Ω–∞–∑–∏–≤–∞—î—Ç—å—Å—è Q-—Ç–∞–±–ª–∏—Ü–µ—é, —Ç–æ–º—É —â–æ —ó—ó —á–∞—Å—Ç–æ –∑—Ä—É—á–Ω–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—Ç–∏ —É –≤–∏–≥–ª—è–¥—ñ —Ç–∞–±–ª–∏—Ü—ñ –∞–±–æ –±–∞–≥–∞—Ç–æ–≤–∏–º—ñ—Ä–Ω–æ–≥–æ –º–∞—Å–∏–≤—É. –û—Å–∫—ñ–ª—å–∫–∏ –Ω–∞—à–∞ –¥–æ—à–∫–∞ –º–∞—î —Ä–æ–∑–º—ñ—Ä–∏ `width` x `height`, –º–∏ –º–æ–∂–µ–º–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–∏—Ç–∏ Q-—Ç–∞–±–ª–∏—Ü—é –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –º–∞—Å–∏–≤—É numpy –∑ —Ñ–æ—Ä–º–æ—é `width` x `height` x `len(actions)`: (–±–ª–æ–∫ –∫–æ–¥—É 6)

```python
Q = np.ones((width,height,len(actions)),dtype=np.float)*1.0/len(actions)
```

–ó–≤–µ—Ä–Ω—ñ—Ç—å —É–≤–∞–≥—É, —â–æ –º–∏ —ñ–Ω—ñ—Ü—ñ–∞–ª—ñ–∑—É—î–º–æ –≤—Å—ñ –∑–Ω–∞—á–µ–Ω–Ω—è Q-—Ç–∞–±–ª–∏—Ü—ñ –æ–¥–Ω–∞–∫–æ–≤–∏–º –∑–Ω–∞—á–µ–Ω–Ω—è–º, —É –Ω–∞—à–æ–º—É –≤–∏–ø–∞–¥–∫—É ‚Äî 0.25. –¶–µ –≤—ñ–¥–ø–æ–≤—ñ–¥–∞—î –ø–æ–ª—ñ—Ç–∏—Ü—ñ "–≤–∏–ø–∞–¥–∫–æ–≤–æ—ó –ø—Ä–æ–≥—É–ª—è–Ω–∫–∏", –æ—Å–∫—ñ–ª—å–∫–∏ –≤—Å—ñ –∫—Ä–æ–∫–∏ –≤ –∫–æ–∂–Ω–æ–º—É —Å—Ç–∞–Ω—ñ –æ–¥–Ω–∞–∫–æ–≤–æ —Ö–æ—Ä–æ—à—ñ. –ú–∏ –º–æ–∂–µ–º–æ –ø–µ—Ä–µ–¥–∞—Ç–∏ Q-—Ç–∞–±–ª–∏—Ü—é –¥–æ —Ñ—É–Ω–∫—Ü—ñ—ó `plot`, —â–æ–± –≤—ñ–∑—É–∞–ª—ñ–∑—É–≤–∞—Ç–∏ —Ç–∞–±–ª–∏—Ü—é –Ω–∞ –¥–æ—à—Ü—ñ: `m.plot(Q)`.

## –°—É—Ç—å Q-Learning: —Ä—ñ–≤–Ω—è–Ω–Ω—è –ë–µ–ª–ª–º–∞–Ω–∞

–Ø–∫ —Ç—ñ–ª—å–∫–∏ –º–∏ –ø–æ—á–∏–Ω–∞—î–º–æ —Ä—É—Ö–∞—Ç–∏—Å—è, –∫–æ–∂–Ω–∞ –¥—ñ—è –º–∞—Ç–∏–º–µ –≤—ñ–¥–ø–æ–≤—ñ–¥–Ω—É –≤–∏–Ω–∞–≥–æ—Ä–æ–¥—É, —Ç–æ–±—Ç–æ —Ç–µ–æ—Ä–µ—Ç–∏—á–Ω–æ –º–∏ –º–æ–∂–µ–º–æ –≤–∏–±—Ä–∞—Ç–∏ –Ω–∞—Å—Ç—É–ø–Ω—É –¥—ñ—é –Ω–∞ –æ—Å–Ω–æ–≤—ñ –Ω–∞–π–≤–∏—â–æ—ó –º–∏—Ç—Ç—î–≤–æ—ó –≤–∏–Ω–∞–≥–æ—Ä–æ–¥–∏. –û–¥–Ω–∞–∫ —É –±—ñ–ª—å—à–æ—Å—Ç—ñ —Å—Ç–∞–Ω—ñ–≤ —Ü–µ–π –∫—Ä–æ–∫ –Ω–µ –¥–æ—Å—è–≥–Ω–µ –Ω–∞—à–æ—ó –º–µ—Ç–∏ ‚Äî –¥–æ—Å—è–≥—Ç–∏ —è–±–ª—É–∫–∞, —ñ —Ç–æ–º—É –º–∏ –Ω–µ –º–æ–∂–µ–º–æ –æ–¥—Ä–∞–∑—É –≤–∏—Ä—ñ—à–∏—Ç–∏, —è–∫–∏–π –Ω–∞–ø—Ä—è–º–æ–∫ –∫—Ä–∞—â–∏–π.

> –ü–∞–º‚Äô—è—Ç–∞–π—Ç–µ, —â–æ –≤–∞–∂–ª–∏–≤–∏–π –Ω–µ –º–∏—Ç—Ç—î–≤–∏–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç, –∞ –∫—ñ–Ω—Ü–µ–≤–∏–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç, —è–∫–∏–π –º–∏ –æ—Ç—Ä–∏–º–∞—î–º–æ –Ω–∞–ø—Ä–∏–∫—ñ–Ω—Ü—ñ —Å–∏–º—É–ª—è—Ü—ñ—ó.

–©–æ–± –≤—Ä–∞—Ö—É–≤–∞—Ç–∏ —Ü—é –≤—ñ–¥–∫–ª–∞–¥–µ–Ω—É –≤–∏–Ω–∞–≥–æ—Ä–æ–¥—É, –Ω–∞–º –ø–æ—Ç—Ä—ñ–±–Ω–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –ø—Ä–∏–Ω—Ü–∏–ø–∏ **[–¥–∏–Ω–∞–º—ñ—á–Ω–æ–≥–æ –ø—Ä–æ–≥—Ä–∞–º—É–≤–∞–Ω–Ω—è](https://en.wikipedia.org/wiki/Dynamic_programming)**, —è–∫—ñ –¥–æ–∑–≤–æ–ª—è—é—Ç—å –º–∏—Å–ª–∏—Ç–∏ –ø—Ä–æ –Ω–∞—à—É –ø—Ä–æ–±–ª–µ–º—É —Ä–µ–∫—É—Ä—Å–∏–≤–Ω–æ.

–ü—Ä–∏–ø—É—Å—Ç–∏–º–æ, —â–æ –∑–∞—Ä–∞–∑ –º–∏ –∑–Ω–∞—Ö–æ–¥–∏–º–æ—Å—è –≤ —Å—Ç–∞–Ω—ñ *s*, —ñ —Ö–æ—á–µ–º–æ –ø–µ—Ä–µ–π—Ç–∏ –¥–æ –Ω–∞—Å—Ç—É–ø–Ω–æ–≥–æ —Å—Ç–∞–Ω—É *s'*. –†–æ–±–ª—è—á–∏ —Ü–µ, –º–∏ –æ—Ç—Ä–∏–º–∞—î–º–æ –º–∏—Ç—Ç—î–≤—É –≤–∏–Ω–∞–≥–æ—Ä–æ–¥—É *r(s,a)*, –≤–∏–∑–Ω–∞—á–µ–Ω—É —Ñ—É–Ω–∫—Ü—ñ—î—é –≤–∏–Ω–∞–≥–æ—Ä–æ–¥–∏, –ø–ª—é—Å –ø–µ–≤–Ω—É –º–∞–π–±—É—Ç–Ω—é –≤–∏–Ω–∞–≥–æ—Ä–æ–¥—É. –Ø–∫—â–æ –ø—Ä–∏–ø—É—Å—Ç–∏—Ç–∏, —â–æ –Ω–∞—à–∞ Q-—Ç–∞–±–ª–∏—Ü—è –ø—Ä–∞–≤–∏–ª—å–Ω–æ –≤—ñ–¥–æ–±—Ä–∞–∂–∞—î "–ø—Ä–∏–≤–∞–±–ª–∏–≤—ñ—Å—Ç—å" –∫–æ–∂–Ω–æ—ó –¥—ñ—ó, —Ç–æ –≤ —Å—Ç–∞–Ω—ñ *s'* –º–∏ –≤–∏–±–µ—Ä–µ–º–æ –¥—ñ—é *a*, —è–∫–∞ –≤—ñ–¥–ø–æ–≤—ñ–¥–∞—î –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–º—É –∑–Ω–∞—á–µ–Ω–Ω—é *Q(s',a')*. –¢–∞–∫–∏–º —á–∏–Ω–æ–º, –Ω–∞–π–∫—Ä–∞—â–∞ –º–æ–∂–ª–∏–≤–∞ –º–∞–π–±—É—Ç–Ω—è –≤–∏–Ω–∞–≥–æ—Ä–æ–¥–∞, —è–∫—É –º–∏ –º–æ–≥–ª–∏ –± –æ—Ç—Ä–∏–º–∞—Ç–∏ –≤ —Å—Ç–∞–Ω—ñ *s*, –±—É–¥–µ –≤–∏–∑–Ω–∞—á–µ–Ω–∞ —è–∫ `max` *Q(s',a')* (–º–∞–∫—Å–∏–º—É–º —Ç—É—Ç –æ–±—á–∏—Å–ª—é—î—Ç—å—Å—è –¥–ª—è –≤—Å—ñ—Ö –º–æ–∂–ª–∏–≤–∏—Ö –¥—ñ–π *a'* —É —Å—Ç–∞–Ω—ñ *s'*).
