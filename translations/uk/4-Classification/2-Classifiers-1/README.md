–ó–≥—ñ–¥–Ω–æ –∑ –¥–æ–∫—É–º–µ–Ω—Ç–∞—Ü—ñ—î—é, —É –≤–∏–ø–∞–¥–∫—É –±–∞–≥–∞—Ç–æ–∫–ª–∞—Å–æ–≤–æ—ó –∑–∞–¥–∞—á—ñ, –∞–ª–≥–æ—Ä–∏—Ç–º –Ω–∞–≤—á–∞–Ω–Ω—è:

- **–í–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î —Ñ—É–Ω–∫—Ü—ñ—é –≤—Ç—Ä–∞—Ç –∫—Ä–æ—Å-–µ–Ω—Ç—Ä–æ–ø—ñ—ó**, —è–∫—â–æ –ø–∞—Ä–∞–º–µ—Ç—Ä `multi_class` –≤—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–æ —è–∫ `multinomial`.
