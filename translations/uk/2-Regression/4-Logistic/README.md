–ù–∞ –¥–∞–Ω–∏–π –º–æ–º–µ–Ω—Ç –≤–∏ –∑–∞–≤–∞–Ω—Ç–∞–∂–∏–ª–∏ [–ø–æ—á–∞—Ç–∫–æ–≤–∏–π –Ω–æ—É—Ç–±—É–∫](../../../../2-Regression/4-Logistic/notebook.ipynb) –∑ –¥–∞–Ω–∏–º–∏ –ø—Ä–æ –≥–∞—Ä–±—É–∑–∏ —Ç–∞ –æ—á–∏—Å—Ç–∏–ª–∏ –π–æ–≥–æ, —â–æ–± –∑–±–µ—Ä–µ–≥—Ç–∏ –Ω–∞–±—ñ—Ä –¥–∞–Ω–∏—Ö, —â–æ –º—ñ—Å—Ç–∏—Ç—å –∫—ñ–ª—å–∫–∞ –∑–º—ñ–Ω–Ω–∏—Ö, –≤–∫–ª—é—á–∞—é—á–∏ `Color`. –î–∞–≤–∞–π—Ç–µ –≤—ñ–∑—É–∞–ª—ñ–∑—É—î–º–æ –¥–∞—Ç–∞—Ñ—Ä–µ–π–º —É –Ω–æ—É—Ç–±—É—Ü—ñ, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—á–∏ —ñ–Ω—à—É –±—ñ–±–ª—ñ–æ—Ç–µ–∫—É: [Seaborn](https://seaborn.pydata.org/index.html), —è–∫–∞ –ø–æ–±—É–¥–æ–≤–∞–Ω–∞ –Ω–∞ Matplotlib, —è–∫—É –º–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞–ª–∏ —Ä–∞–Ω—ñ—à–µ.

Seaborn –ø—Ä–æ–ø–æ–Ω—É—î —Ü—ñ–∫–∞–≤—ñ —Å–ø–æ—Å–æ–±–∏ –≤—ñ–∑—É–∞–ª—ñ–∑–∞—Ü—ñ—ó –≤–∞—à–∏—Ö –¥–∞–Ω–∏—Ö. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, –≤–∏ –º–æ–∂–µ—Ç–µ –ø–æ—Ä—ñ–≤–Ω—è—Ç–∏ —Ä–æ–∑–ø–æ–¥—ñ–ª–∏ –¥–∞–Ω–∏—Ö –¥–ª—è –∫–æ–∂–Ω–æ–≥–æ `Variety` —ñ `Color` —É –∫–∞—Ç–µ–≥–æ—Ä—ñ–∞–ª—å–Ω–æ–º—É –≥—Ä–∞—Ñ—ñ–∫—É.

1. –°—Ç–≤–æ—Ä—ñ—Ç—å —Ç–∞–∫–∏–π –≥—Ä–∞—Ñ—ñ–∫, –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—á–∏ —Ñ—É–Ω–∫—Ü—ñ—é `catplot`, –¥–∞–Ω—ñ –ø—Ä–æ –≥–∞—Ä–±—É–∑–∏ `pumpkins` —ñ –≤–∫–∞–∑–∞–≤—à–∏ –∫–æ–ª—å–æ—Ä–æ–≤–µ –≤—ñ–¥–æ–±—Ä–∞–∂–µ–Ω–Ω—è –¥–ª—è –∫–æ–∂–Ω–æ—ó –∫–∞—Ç–µ–≥–æ—Ä—ñ—ó –≥–∞—Ä–±—É–∑—ñ–≤ (–ø–æ–º–∞—Ä–∞–Ω—á–µ–≤–∏–π –∞–±–æ –±—ñ–ª–∏–π):

    ```python
    import seaborn as sns
    
    palette = {
    'ORANGE': 'orange',

    }

    sns.catplot(

    ```

    –°–ø–æ—Å—Ç–µ—Ä—ñ–≥–∞—é—á–∏ –∑–∞ –¥–∞–Ω–∏–º–∏, –≤–∏ –º–æ–∂–µ—Ç–µ –ø–æ–±–∞—á–∏—Ç–∏, —è–∫ –¥–∞–Ω—ñ –ø—Ä–æ Color —Å–ø—ñ–≤–≤—ñ–¥–Ω–æ—Å—è—Ç—å—Å—è –∑ Variety.

    ‚úÖ –í—Ä–∞—Ö–æ–≤—É—é—á–∏ —Ü–µ–π –∫–∞—Ç–µ–≥–æ—Ä—ñ–∞–ª—å–Ω–∏–π –≥—Ä–∞—Ñ—ñ–∫, —è–∫—ñ —Ü—ñ–∫–∞–≤—ñ –¥–æ—Å–ª—ñ–¥–∂–µ–Ω–Ω—è –≤–∏ –º–æ–∂–µ—Ç–µ —É—è–≤–∏—Ç–∏?

### –ü–æ–ø–µ—Ä–µ–¥–Ω—è –æ–±—Ä–æ–±–∫–∞ –¥–∞–Ω–∏—Ö: –∫–æ–¥—É–≤–∞–Ω–Ω—è –æ–∑–Ω–∞–∫ —ñ –º—ñ—Ç–æ–∫

–ù–∞—à –Ω–∞–±—ñ—Ä –¥–∞–Ω–∏—Ö –ø—Ä–æ –≥–∞—Ä–±—É–∑–∏ –º—ñ—Å—Ç–∏—Ç—å —Ç–µ–∫—Å—Ç–æ–≤—ñ –∑–Ω–∞—á–µ–Ω–Ω—è –¥–ª—è –≤—Å—ñ—Ö —Å–≤–æ—ó—Ö —Å—Ç–æ–≤–ø—Ü—ñ–≤. –†–æ–±–æ—Ç–∞ –∑ –∫–∞—Ç–µ–≥–æ—Ä—ñ–∞–ª—å–Ω–∏–º–∏ –¥–∞–Ω–∏–º–∏ —î —ñ–Ω—Ç—É—ó—Ç–∏–≤–Ω–æ—é –¥–ª—è –ª—é–¥–µ–π, –∞–ª–µ –Ω–µ –¥–ª—è –º–∞—à–∏–Ω. –ê–ª–≥–æ—Ä–∏—Ç–º–∏ –º–∞—à–∏–Ω–Ω–æ–≥–æ –Ω–∞–≤—á–∞–Ω–Ω—è –¥–æ–±—Ä–µ –ø—Ä–∞—Ü—é—é—Ç—å —ñ–∑ —á–∏—Å–ª–∞–º–∏. –°–∞–º–µ —Ç–æ–º—É –∫–æ–¥—É–≤–∞–Ω–Ω—è —î –¥—É–∂–µ –≤–∞–∂–ª–∏–≤–∏–º –µ—Ç–∞–ø–æ–º —É —Ñ–∞–∑—ñ –ø–æ–ø–µ—Ä–µ–¥–Ω—å–æ—ó –æ–±—Ä–æ–±–∫–∏ –¥–∞–Ω–∏—Ö, –æ—Å–∫—ñ–ª—å–∫–∏ –≤–æ–Ω–æ –¥–æ–∑–≤–æ–ª—è—î –ø–µ—Ä–µ—Ç–≤–æ—Ä–∏—Ç–∏ –∫–∞—Ç–µ–≥–æ—Ä—ñ–∞–ª—å–Ω—ñ –¥–∞–Ω—ñ –≤ —á–∏—Å–ª–æ–≤—ñ, –Ω–µ –≤—Ç—Ä–∞—á–∞—é—á–∏ –∂–æ–¥–Ω–æ—ó —ñ–Ω—Ñ–æ—Ä–º–∞—Ü—ñ—ó. –•–æ—Ä–æ—à–µ –∫–æ–¥—É–≤–∞–Ω–Ω—è —Å–ø—Ä–∏—è—î —Å—Ç–≤–æ—Ä–µ–Ω–Ω—é —Ö–æ—Ä–æ—à–æ—ó –º–æ–¥–µ–ª—ñ.

–î–ª—è –∫–æ–¥—É–≤–∞–Ω–Ω—è –æ–∑–Ω–∞–∫ —ñ—Å–Ω—É—î –¥–≤–∞ –æ—Å–Ω–æ–≤–Ω–∏—Ö —Ç–∏–ø–∏ –∫–æ–¥–µ—Ä—ñ–≤:

1. Ordinal encoder: –¥–æ–±—Ä–µ –ø—ñ–¥—Ö–æ–¥–∏—Ç—å –¥–ª—è –ø–æ—Ä—è–¥–∫–æ–≤–∏—Ö –∑–º—ñ–Ω–Ω–∏—Ö, —è–∫—ñ —î –∫–∞—Ç–µ–≥–æ—Ä—ñ–∞–ª—å–Ω–∏–º–∏ –∑–º—ñ–Ω–Ω–∏–º–∏, –¥–µ —ó—Ö –¥–∞–Ω—ñ –º–∞—é—Ç—å –ª–æ–≥—ñ—á–Ω–∏–π –ø–æ—Ä—è–¥–æ–∫, —è–∫ —Å—Ç–æ–≤–ø–µ—Ü—å `Item Size` —É –Ω–∞—à–æ–º—É –Ω–∞–±–æ—Ä—ñ –¥–∞–Ω–∏—Ö. –í—ñ–Ω —Å—Ç–≤–æ—Ä—é—î –≤—ñ–¥–æ–±—Ä–∞–∂–µ–Ω–Ω—è, —Ç–∞–∫ —â–æ –∫–æ–∂–Ω–∞ –∫–∞—Ç–µ–≥–æ—Ä—ñ—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∞ —á–∏—Å–ª–æ–º, —è–∫–µ —î –ø–æ—Ä—è–¥–∫–æ–º –∫–∞—Ç–µ–≥–æ—Ä—ñ—ó –≤ —Å—Ç–æ–≤–ø—Ü—ñ.

    ```python
    from sklearn.preprocessing import OrdinalEncoder

    item_size_categories = [['sml', 'med', 'med-lge', 'lge', 'xlge', 'jbo', 'exjbo']]
    ordinal_features = ['Item Size']
    ordinal_encoder = OrdinalEncoder(categories=item_size_categories)
    ```

2. Categorical encoder: –¥–æ–±—Ä–µ –ø—ñ–¥—Ö–æ–¥–∏—Ç—å –¥–ª—è –Ω–æ–º—ñ–Ω–∞–ª—å–Ω–∏—Ö –∑–º—ñ–Ω–Ω–∏—Ö, —è–∫—ñ —î –∫–∞—Ç–µ–≥–æ—Ä—ñ–∞–ª—å–Ω–∏–º–∏ –∑–º—ñ–Ω–Ω–∏–º–∏, –¥–µ —ó—Ö –¥–∞–Ω—ñ –Ω–µ –º–∞—é—Ç—å –ª–æ–≥—ñ—á–Ω–æ–≥–æ –ø–æ—Ä—è–¥–∫—É, —è–∫ —É—Å—ñ –æ–∑–Ω–∞–∫–∏, –≤—ñ–¥–º—ñ–Ω–Ω—ñ –≤—ñ–¥ `Item Size` —É –Ω–∞—à–æ–º—É –Ω–∞–±–æ—Ä—ñ –¥–∞–Ω–∏—Ö. –¶–µ –∫–æ–¥—É–≤–∞–Ω–Ω—è "–æ–¥–∏–Ω-–≥–∞—Ä—è—á–∏–π", —â–æ –æ–∑–Ω–∞—á–∞—î, —â–æ –∫–æ–∂–Ω–∞ –∫–∞—Ç–µ–≥–æ—Ä—ñ—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∞ –±—ñ–Ω–∞—Ä–Ω–∏–º —Å—Ç–æ–≤–ø—Ü–µ–º: –∑–∞–∫–æ–¥–æ–≤–∞–Ω–∞ –∑–º—ñ–Ω–Ω–∞ –¥–æ—Ä—ñ–≤–Ω—é—î 1, —è–∫—â–æ –≥–∞—Ä–±—É–∑ –Ω–∞–ª–µ–∂–∏—Ç—å –¥–æ —Ü—ñ—î—ó Variety, —ñ 0 –≤ —ñ–Ω—à–æ–º—É –≤–∏–ø–∞–¥–∫—É.

    ```python
    from sklearn.preprocessing import OneHotEncoder

    categorical_features = ['City Name', 'Package', 'Variety', 'Origin']
    categorical_encoder = OneHotEncoder(sparse_output=False)
    ```

–ü–æ—Ç—ñ–º `ColumnTransformer` –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è –¥–ª—è –æ–±'—î–¥–Ω–∞–Ω–Ω—è –∫—ñ–ª—å–∫–æ—Ö –∫–æ–¥–µ—Ä—ñ–≤ –≤ –æ–¥–∏–Ω –µ—Ç–∞–ø —ñ –∑–∞—Å—Ç–æ—Å—É–≤–∞–Ω–Ω—è —ó—Ö –¥–æ –≤—ñ–¥–ø–æ–≤—ñ–¥–Ω–∏—Ö —Å—Ç–æ–≤–ø—Ü—ñ–≤.

```python
    from sklearn.compose import ColumnTransformer
    
    ct = ColumnTransformer(transformers=[
        ('ord', ordinal_encoder, ordinal_features),
        ('cat', categorical_encoder, categorical_features)
        ])
    
    ct.set_output(transform='pandas')
    encoded_features = ct.fit_transform(pumpkins)
```

–î–ª—è –∫–æ–¥—É–≤–∞–Ω–Ω—è –º—ñ—Ç–∫–∏ –º–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î–º–æ –∫–ª–∞—Å `LabelEncoder` –∑ –±—ñ–±–ª—ñ–æ—Ç–µ–∫–∏ scikit-learn, —è–∫–∏–π —î —É—Ç–∏–ª—ñ—Ç–æ—é –¥–ª—è –Ω–æ—Ä–º–∞–ª—ñ–∑–∞—Ü—ñ—ó –º—ñ—Ç–æ–∫ —Ç–∞–∫, —â–æ–± –≤–æ–Ω–∏ –º—ñ—Å—Ç–∏–ª–∏ –ª–∏—à–µ –∑–Ω–∞—á–µ–Ω–Ω—è –º—ñ–∂ 0 —ñ n_classes-1 (—Ç—É—Ç, 0 —ñ 1).

```python
    from sklearn.preprocessing import LabelEncoder

    label_encoder = LabelEncoder()
    encoded_label = label_encoder.fit_transform(pumpkins['Color'])
```

–ü—ñ—Å–ª—è —Ç–æ–≥–æ, —è–∫ –º–∏ –∑–∞–∫–æ–¥—É–≤–∞–ª–∏ –æ–∑–Ω–∞–∫–∏ —Ç–∞ –º—ñ—Ç–∫—É, –º–∏ –º–æ–∂–µ–º–æ –æ–±'—î–¥–Ω–∞—Ç–∏ —ó—Ö —É –Ω–æ–≤–∏–π –¥–∞—Ç–∞—Ñ—Ä–µ–π–º `encoded_pumpkins`.

```python
    encoded_pumpkins = encoded_features.assign(Color=encoded_label)
```

‚úÖ –Ø–∫—ñ –ø–µ—Ä–µ–≤–∞–≥–∏ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞–Ω–Ω—è –ø–æ—Ä—è–¥–∫–æ–≤–æ–≥–æ –∫–æ–¥–µ—Ä–∞ –¥–ª—è —Å—Ç–æ–≤–ø—Ü—è `Item Size`?

### –ê–Ω–∞–ª—ñ–∑ –≤–∑–∞—î–º–æ–∑–≤'—è–∑–∫—ñ–≤ –º—ñ–∂ –∑–º—ñ–Ω–Ω–∏–º–∏

–¢–µ–ø–µ—Ä, –∫–æ–ª–∏ –º–∏ –ø–æ–ø–µ—Ä–µ–¥–Ω—å–æ –æ–±—Ä–æ–±–∏–ª–∏ –Ω–∞—à—ñ –¥–∞–Ω—ñ, –º–∏ –º–æ–∂–µ–º–æ –∞–Ω–∞–ª—ñ–∑—É–≤–∞—Ç–∏ –≤–∑–∞—î–º–æ–∑–≤'—è–∑–∫–∏ –º—ñ–∂ –æ–∑–Ω–∞–∫–∞–º–∏ —Ç–∞ –º—ñ—Ç–∫–æ—é, —â–æ–± –∑—Ä–æ–∑—É–º—ñ—Ç–∏, –Ω–∞—Å–∫—ñ–ª—å–∫–∏ –¥–æ–±—Ä–µ –º–æ–¥–µ–ª—å –∑–º–æ–∂–µ –ø—Ä–æ–≥–Ω–æ–∑—É–≤–∞—Ç–∏ –º—ñ—Ç–∫—É, –≤—Ä–∞—Ö–æ–≤—É—é—á–∏ –æ–∑–Ω–∞–∫–∏. –ù–∞–π–∫—Ä–∞—â–∏–π —Å–ø–æ—Å—ñ–± –≤–∏–∫–æ–Ω–∞—Ç–∏ —Ç–∞–∫–∏–π –∞–Ω–∞–ª—ñ–∑ - —Ü–µ –ø–æ–±—É–¥–æ–≤–∞ –≥—Ä–∞—Ñ—ñ–∫—ñ–≤. –ú–∏ –∑–Ω–æ–≤—É –±—É–¥–µ–º–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ —Ñ—É–Ω–∫—Ü—ñ—é `catplot` –∑ –±—ñ–±–ª—ñ–æ—Ç–µ–∫–∏ Seaborn, —â–æ–± –≤—ñ–∑—É–∞–ª—ñ–∑—É–≤–∞—Ç–∏ –≤–∑–∞—î–º–æ–∑–≤'—è–∑–∫–∏ –º—ñ–∂ `Item Size`, `Variety` —ñ `Color` —É –∫–∞—Ç–µ–≥–æ—Ä—ñ–∞–ª—å–Ω–æ–º—É –≥—Ä–∞—Ñ—ñ–∫—É. –î–ª—è –∫—Ä–∞—â–æ—ó –ø–æ–±—É–¥–æ–≤–∏ –≥—Ä–∞—Ñ—ñ–∫–∞ –º–∏ –±—É–¥–µ–º–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –∑–∞–∫–æ–¥–æ–≤–∞–Ω–∏–π —Å—Ç–æ–≤–ø–µ—Ü—å `Item Size` —ñ –Ω–µ–∑–∞–∫–æ–¥–æ–≤–∞–Ω–∏–π —Å—Ç–æ–≤–ø–µ—Ü—å `Variety`.
