## р╕Бр╕▓р╕гр╣Бр╕кр╕Фр╕Зр╕Ьр╕е ROC Curve р╕Вр╕нр╕Зр╣Вр╕бр╣Ар╕Фр╕ер╕Щр╕╡р╣Й

[![ML for beginners - р╕Бр╕▓р╕гр╕зр╕┤р╣Ар╕Др╕гр╕▓р╕░р╕лр╣Мр╕Ыр╕гр╕░р╕кр╕┤р╕Чр╕Шр╕┤р╕ар╕▓р╕Ю Logistic Regression р╕Фр╣Йр╕зр╕в ROC Curves](https://img.youtube.com/vi/GApO575jTA0/0.jpg)](https://youtu.be/GApO575jTA0 "ML for beginners - р╕Бр╕▓р╕гр╕зр╕┤р╣Ар╕Др╕гр╕▓р╕░р╕лр╣Мр╕Ыр╕гр╕░р╕кр╕┤р╕Чр╕Шр╕┤р╕ар╕▓р╕Ю Logistic Regression р╕Фр╣Йр╕зр╕в ROC Curves")

> ЁЯОе р╕Др╕ер╕┤р╕Бр╕Чр╕╡р╣Ир╕ар╕▓р╕Юр╕Фр╣Йр╕▓р╕Щр╕Ър╕Щр╣Ар╕Юр╕╖р╣Ир╕нр╕Фр╕╣р╕зр╕┤р╕Фр╕╡р╣Вр╕нр╕кр╕▒р╣Йр╕Щр╣Ар╕Бр╕╡р╣Ир╕вр╕зр╕Бр╕▒р╕Ъ ROC Curves

р╕бр╕▓р╕Чр╕│р╕Бр╕▓р╕гр╣Бр╕кр╕Фр╕Зр╕Ьр╕ер╕нр╕╡р╕Бр╕Др╕гр╕▒р╣Йр╕Зр╣Ар╕Юр╕╖р╣Ир╕нр╕Фр╕╣ 'ROC' Curve:

```python
from sklearn.metrics import roc_curve, roc_auc_score
import matplotlib
import matplotlib.pyplot as plt
%matplotlib inline

y_scores = model.predict_proba(X_test)
fpr, tpr, thresholds = roc_curve(y_test, y_scores[:,1])

fig = plt.figure(figsize=(6, 6))
plt.plot([0, 1], [0, 1], 'k--')
plt.plot(fpr, tpr)
plt.xlabel('False Positive Rate')
plt.ylabel('True Positive Rate')
plt.title('ROC Curve')
plt.show()
```

р╕кр╕╕р╕Фр╕Чр╣Йр╕▓р╕в р╣Гр╕Кр╣Й [`roc_auc_score` API](https://scikit-learn.org/stable/modules/generated/sklearn.metrics.roc_auc_score.html?highlight=roc_auc#sklearn.metrics.roc_auc_score) р╕Вр╕нр╕З Scikit-learn р╣Ар╕Юр╕╖р╣Ир╕нр╕Др╕│р╕Щр╕зр╕У 'Area Under the Curve' (AUC) р╕Ир╕гр╕┤р╕З:
