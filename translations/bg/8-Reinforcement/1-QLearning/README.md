–ù–µ–∫–∞ –ø—ä—Ä–≤–æ —Ä–µ—à–∏–º –ø—Ä–æ–±–ª–µ–º–∞, –∫–∞—Ç–æ –ø—Ä–∏–ª–æ–∂–∏–º —Å—Ç—Ä–∞—Ç–µ–≥–∏—è—Ç–∞ –∑–∞ —Å–ª—É—á–∞–π–Ω–∞ —Ä–∞–∑—Ö–æ–¥–∫–∞. –ü—Ä–∏ —Å–ª—É—á–∞–π–Ω–∞ —Ä–∞–∑—Ö–æ–¥–∫–∞ —â–µ –∏–∑–±–∏—Ä–∞–º–µ —Å–ª–µ–¥–≤–∞—â–æ—Ç–æ –¥–µ–π—Å—Ç–≤–∏–µ —Å–ª—É—á–∞–π–Ω–æ –æ—Ç –ø–æ–∑–≤–æ–ª–µ–Ω–∏—Ç–µ –¥–µ–π—Å—Ç–≤–∏—è, –¥–æ–∫–∞—Ç–æ —Å—Ç–∏–≥–Ω–µ–º –¥–æ —è–±—ä–ª–∫–∞—Ç–∞ (–∫–æ–¥ –±–ª–æ–∫ 3).

1. –†–µ–∞–ª–∏–∑–∏—Ä–∞–π—Ç–µ —Å–ª—É—á–∞–π–Ω–∞—Ç–∞ —Ä–∞–∑—Ö–æ–¥–∫–∞ —Å –∫–æ–¥–∞ –ø–æ-–¥–æ–ª—É:

    ```python
    def random_policy(m):
        return random.choice(list(actions))
    
    def walk(m,policy,start_position=None):
        n = 0 # number of steps
        # set initial position
        if start_position:
            m.human = start_position 
        else:
            m.random_start()
        while True:
            if m.at() == Board.Cell.apple:
                return n # success!
            if m.at() in [Board.Cell.wolf, Board.Cell.water]:
                return -1 # eaten by wolf or drowned
            while True:
                a = actions[policy(m)]
                new_pos = m.move_pos(m.human,a)
                if m.is_valid(new_pos) and m.at(new_pos)!=Board.Cell.water:
                    m.move(a) # do the actual move
                    break
            n+=1
    
    walk(m,random_policy)
    ```

1. –ò–∑–ø—ä–ª–Ω–µ—Ç–µ –µ–∫—Å–ø–µ—Ä–∏–º–µ–Ω—Ç–∞ –∑–∞ —Ä–∞–∑—Ö–æ–¥–∫–∞ –Ω—è–∫–æ–ª–∫–æ –ø—ä—Ç–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä 100) –∏ –æ—Ç–ø–µ—á–∞—Ç–∞–π—Ç–µ –ø–æ–ª—É—á–µ–Ω–∏—Ç–µ —Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–∏ (–∫–æ–¥ –±–ª–æ–∫ 4):

## –§—É–Ω–∫—Ü–∏—è –∑–∞ –Ω–∞–≥—Ä–∞–¥–∞

–ó–∞ –¥–∞ –Ω–∞–ø—Ä–∞–≤–∏–º –Ω–∞—à–∞—Ç–∞ –ø–æ–ª–∏—Ç–∏–∫–∞ –ø–æ-–∏–Ω—Ç–µ–ª–∏–≥–µ–Ω—Ç–Ω–∞, —Ç—Ä—è–±–≤–∞ –¥–∞ —Ä–∞–∑–±–µ—Ä–µ–º –∫–æ–∏ —Ö–æ–¥–æ–≤–µ —Å–∞ "–ø–æ-–¥–æ–±—Ä–∏" –æ—Ç –¥—Ä—É–≥–∏. –ó–∞ –¥–∞ –Ω–∞–ø—Ä–∞–≤–∏–º —Ç–æ–≤–∞, —Ç—Ä—è–±–≤–∞ –¥–∞ –¥–µ—Ñ–∏–Ω–∏—Ä–∞–º–µ –Ω–∞—à–∞—Ç–∞ —Ü–µ–ª.

–¶–µ–ª—Ç–∞ –º–æ–∂–µ –¥–∞ –±—ä–¥–µ –¥–µ—Ñ–∏–Ω–∏—Ä–∞–Ω–∞ —á—Ä–µ–∑ **—Ñ—É–Ω–∫—Ü–∏—è –∑–∞ –Ω–∞–≥—Ä–∞–¥–∞**, –∫–æ—è—Ç–æ —â–µ –≤—Ä—ä—â–∞ –Ω—è–∫–∞–∫–≤–∞ —Å—Ç–æ–π–Ω–æ—Å—Ç –∑–∞ –≤—Å—è–∫–æ —Å—ä—Å—Ç–æ—è–Ω–∏–µ. –ö–æ–ª–∫–æ—Ç–æ –ø–æ-–≤–∏—Å–æ–∫–∞ –µ —Å—Ç–æ–π–Ω–æ—Å—Ç—Ç–∞, —Ç–æ–ª–∫–æ–≤–∞ –ø–æ-–¥–æ–±—Ä–∞ –µ –Ω–∞–≥—Ä–∞–¥–∞—Ç–∞. (–∫–æ–¥ –±–ª–æ–∫ 5)

```python
move_reward = -0.1
goal_reward = 10
end_reward = -10

def reward(m,pos=None):
    pos = pos or m.human
    if not m.is_valid(pos):
        return end_reward
    x = m.at(pos)
    if x==Board.Cell.water or x == Board.Cell.wolf:
        return end_reward
    if x==Board.Cell.apple:
        return goal_reward
    return move_reward
```

## Q-Learning

–ê–ª–≥–æ—Ä–∏—Ç—ä–º—ä—Ç, –∫–æ–π—Ç–æ —â–µ –æ–±—Å—ä–¥–∏–º —Ç—É–∫, —Å–µ –Ω–∞—Ä–∏—á–∞ **Q-Learning**. –í —Ç–æ–∑–∏ –∞–ª–≥–æ—Ä–∏—Ç—ä–º –ø–æ–ª–∏—Ç–∏–∫–∞—Ç–∞ —Å–µ –¥–µ—Ñ–∏–Ω–∏—Ä–∞ —á—Ä–µ–∑ —Ñ—É–Ω–∫—Ü–∏—è (–∏–ª–∏ —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –æ—Ç –¥–∞–Ω–Ω–∏), –Ω–∞—Ä–µ—á–µ–Ω–∞ **Q-—Ç–∞–±–ª–∏—Ü–∞**. –¢—è –∑–∞–ø–∏—Å–≤–∞ "–¥–æ–±—Ä–æ—Ç–æ" –Ω–∞ –≤—Å—è–∫–æ –¥–µ–π—Å—Ç–≤–∏–µ –≤ –¥–∞–¥–µ–Ω–æ —Å—ä—Å—Ç–æ—è–Ω–∏–µ.

–¢—è —Å–µ –Ω–∞—Ä–∏—á–∞ Q-—Ç–∞–±–ª–∏—Ü–∞, –∑–∞—â–æ—Ç–æ —á–µ—Å—Ç–æ –µ —É–¥–æ–±–Ω–æ –¥–∞ —Å–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–∏ –∫–∞—Ç–æ —Ç–∞–±–ª–∏—Ü–∞ –∏–ª–∏ –º–Ω–æ–≥–æ–º–µ—Ä–µ–Ω –º–∞—Å–∏–≤. –¢—ä–π –∫–∞—Ç–æ –Ω–∞—à–∞—Ç–∞ –¥—ä—Å–∫–∞ –∏–º–∞ —Ä–∞–∑–º–µ—Ä–∏ `—à–∏—Ä–∏–Ω–∞` x `–≤–∏—Å–æ—á–∏–Ω–∞`, –º–æ–∂–µ–º –¥–∞ –ø—Ä–µ–¥—Å—Ç–∞–≤–∏–º Q-—Ç–∞–±–ª–∏—Ü–∞—Ç–∞, –∏–∑–ø–æ–ª–∑–≤–∞–π–∫–∏ numpy –º–∞—Å–∏–≤ —Å —Ñ–æ—Ä–º–∞ `—à–∏—Ä–∏–Ω–∞` x `–≤–∏—Å–æ—á–∏–Ω–∞` x `len(actions)`: (–∫–æ–¥ –±–ª–æ–∫ 6)
