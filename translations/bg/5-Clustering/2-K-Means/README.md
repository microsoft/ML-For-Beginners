1. –°–µ–≥–∞ —Ç—Ä—è–±–≤–∞ –¥–∞ –∏–∑–±–µ—Ä–µ—Ç–µ –∫–æ–ª–∫–æ –∫–ª—ä—Å—Ç–µ—Ä–∏ –¥–∞ –Ω–∞—Å–æ—á–∏—Ç–µ. –ó–Ω–∞–µ—Ç–µ, —á–µ –∏–º–∞ 3 –∂–∞–Ω—Ä–∞ –ø–µ—Å–Ω–∏, –∫–æ–∏—Ç–æ –∏–∑–≤–ª—è–∫–æ—Ö–º–µ –æ—Ç –Ω–∞–±–æ—Ä–∞ –æ—Ç –¥–∞–Ω–Ω–∏, —Ç–∞–∫–∞ —á–µ –Ω–µ–∫–∞ –æ–ø–∏—Ç–∞–º–µ —Å 3:
