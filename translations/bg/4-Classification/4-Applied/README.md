### –ö–æ–Ω–≤–µ—Ä—Ç–∏—Ä–∞–π—Ç–µ –º–æ–¥–µ–ª–∞ —Å–∏ –≤ Onnx

–£–≤–µ—Ä–µ—Ç–µ —Å–µ, —á–µ –ø—Ä–∞–≤–∏—Ç–µ –∫–æ–Ω–≤–µ—Ä—Å–∏—è—Ç–∞ —Å –ø—Ä–∞–≤–∏–ª–Ω–∏—è –±—Ä–æ–π —Ç–µ–Ω–∑–æ—Ä–∏. –¢–æ–∑–∏ –Ω–∞–±–æ—Ä –æ—Ç –¥–∞–Ω–Ω–∏ –∏–º–∞ 380 –∏–∑–±—Ä–æ–µ–Ω–∏ —Å—ä—Å—Ç–∞–≤–∫–∏, —Ç–∞–∫–∞ —á–µ —Ç—Ä—è–±–≤–∞ –¥–∞ –æ—Ç–±–µ–ª–µ–∂–∏—Ç–µ —Ç–æ–∑–∏ –±—Ä–æ–π –≤ `FloatTensorType`:

1. –ö–æ–Ω–≤–µ—Ä—Ç–∏—Ä–∞–π—Ç–µ, –∏–∑–ø–æ–ª–∑–≤–∞–π–∫–∏ —Ç–µ–Ω–∑–æ—Ä —Å –±—Ä–æ–π 380.

    ```python
    from skl2onnx import convert_sklearn
    from skl2onnx.common.data_types import FloatTensorType
    
    initial_type = [('float_input', FloatTensorType([None, 380]))]
    options = {id(model): {'nocl': True, 'zipmap': False}}
    ```

1. –°—ä–∑–¥–∞–π—Ç–µ onx —Ñ–∞–π–ª –∏ –≥–æ –∑–∞–ø–∞–∑–µ—Ç–µ –∫–∞—Ç–æ **model.onnx**:

    ```python
    onx = convert_sklearn(model, initial_types=initial_type, options=options)
    with open("./model.onnx", "wb") as f:
        f.write(onx.SerializeToString())
    ```

    > –ó–∞–±–µ–ª–µ–∂–∫–∞: –ú–æ–∂–µ—Ç–µ –¥–∞ –ø—Ä–µ–¥–∞–¥–µ—Ç–µ [–æ–ø—Ü–∏–∏](https://onnx.ai/sklearn-onnx/parameterized.html) –≤—ä–≤ –≤–∞—à–∏—è —Å–∫—Ä–∏–ø—Ç –∑–∞ –∫–æ–Ω–≤–µ—Ä—Å–∏—è. –í —Ç–æ–∑–∏ —Å–ª—É—á–∞–π –∑–∞–¥–∞–¥–æ—Ö–º–µ 'nocl' –¥–∞ –±—ä–¥–µ True –∏ 'zipmap' –¥–∞ –±—ä–¥–µ False. –¢—ä–π –∫–∞—Ç–æ —Ç–æ–≤–∞ –µ –º–æ–¥–µ–ª –∑–∞ –∫–ª–∞—Å–∏—Ñ–∏–∫–∞—Ü–∏—è, –∏–º–∞—Ç–µ –≤—ä–∑–º–æ–∂–Ω–æ—Å—Ç –¥–∞ –ø—Ä–µ–º–∞—Ö–Ω–µ—Ç–µ ZipMap, –∫–æ–π—Ç–æ –ø—Ä–æ–∏–∑–≤–µ–∂–¥–∞ —Å–ø–∏—Å—ä–∫ –æ—Ç —Ä–µ—á–Ω–∏—Ü–∏ (–Ω–µ –µ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ). `nocl` —Å–µ –æ—Ç–Ω–∞—Å—è –¥–æ –≤–∫–ª—é—á–≤–∞–Ω–µ—Ç–æ –Ω–∞ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è –∑–∞ –∫–ª–∞—Å–æ–≤–µ—Ç–µ –≤ –º–æ–¥–µ–ª–∞. –ù–∞–º–∞–ª–µ—Ç–µ —Ä–∞–∑–º–µ—Ä–∞ –Ω–∞ –º–æ–¥–µ–ª–∞ —Å–∏, –∫–∞—Ç–æ –∑–∞–¥–∞–¥–µ—Ç–µ `nocl` –Ω–∞ 'True'.

–ò–∑–ø—ä–ª–Ω–µ–Ω–∏–µ—Ç–æ –Ω–∞ —Ü–µ–ª–∏—è –Ω–æ—É—Ç–±—É–∫ —Å–µ–≥–∞ —â–µ —Å—ä–∑–¥–∞–¥–µ Onnx –º–æ–¥–µ–ª –∏ —â–µ –≥–æ –∑–∞–ø–∞–∑–∏ –≤ —Ç–∞–∑–∏ –ø–∞–ø–∫–∞.

## –ü—Ä–µ–≥–ª–µ–¥–∞–π—Ç–µ –º–æ–¥–µ–ª–∞ —Å–∏
