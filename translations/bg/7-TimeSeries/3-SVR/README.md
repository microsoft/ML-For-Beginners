### SVR –≤ –∫–æ–Ω—Ç–µ–∫—Å—Ç–∞ –Ω–∞ –≤—Ä–µ–º–µ–≤–∏ —Å–µ—Ä–∏–∏ [^1]

–ü—Ä–µ–¥–∏ –¥–∞ —Ä–∞–∑–±–µ—Ä–µ—Ç–µ –∑–Ω–∞—á–µ–Ω–∏–µ—Ç–æ –Ω–∞ SVR –∑–∞ –ø—Ä–æ–≥–Ω–æ–∑–∏—Ä–∞–Ω–µ –Ω–∞ –≤—Ä–µ–º–µ–≤–∏ —Å–µ—Ä–∏–∏, –µ—Ç–æ –Ω—è–∫–æ–∏ –≤–∞–∂–Ω–∏ –∫–æ–Ω—Ü–µ–ø—Ü–∏–∏, –∫–æ–∏—Ç–æ —Ç—Ä—è–±–≤–∞ –¥–∞ –∑–Ω–∞–µ—Ç–µ:

- **–†–µ–≥—Ä–µ—Å–∏—è:** –¢–µ—Ö–Ω–∏–∫–∞ –∑–∞ –æ–±—É—á–µ–Ω–∏–µ —Å –Ω–∞–¥–∑–æ—Ä, –∫–æ—è—Ç–æ –ø—Ä–µ–¥—Å–∫–∞–∑–≤–∞ –Ω–µ–ø—Ä–µ–∫—ä—Å–Ω–∞—Ç–∏ —Å—Ç–æ–π–Ω–æ—Å—Ç–∏ –æ—Ç –¥–∞–¥–µ–Ω –Ω–∞–±–æ—Ä –æ—Ç –≤—Ö–æ–¥–Ω–∏ –¥–∞–Ω–Ω–∏. –ò–¥–µ—è—Ç–∞ –µ –¥–∞ —Å–µ –Ω–∞–º–µ—Ä–∏ –∫—Ä–∏–≤–∞ (–∏–ª–∏ –ª–∏–Ω–∏—è) –≤ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ—Ç–æ –Ω–∞ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∏—Ç–µ, –∫–æ—è—Ç–æ –∏–º–∞ –º–∞–∫—Å–∏–º–∞–ª–µ–Ω –±—Ä–æ–π —Ç–æ—á–∫–∏ –æ—Ç –¥–∞–Ω–Ω–∏. [–ö–ª–∏–∫–Ω–µ—Ç–µ —Ç—É–∫](https://en.wikipedia.org/wiki/Regression_analysis) –∑–∞ –ø–æ–≤–µ—á–µ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è.
- **Support Vector Machine (SVM):** –í–∏–¥ –º–æ–¥–µ–ª –∑–∞ –º–∞—à–∏–Ω–Ω–æ –æ–±—É—á–µ–Ω–∏–µ —Å –Ω–∞–¥–∑–æ—Ä, –∏–∑–ø–æ–ª–∑–≤–∞–Ω –∑–∞ –∫–ª–∞—Å–∏—Ñ–∏–∫–∞—Ü–∏—è, —Ä–µ–≥—Ä–µ—Å–∏—è –∏ –æ—Ç–∫—Ä–∏–≤–∞–Ω–µ –Ω–∞ –∞–Ω–æ–º–∞–ª–∏–∏. –ú–æ–¥–µ–ª—ä—Ç –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–≤–∞ —Ö–∏–ø–µ—Ä–ø–ª–æ—Å–∫–æ—Å—Ç –≤ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ—Ç–æ –Ω–∞ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∏—Ç–µ, –∫–æ—è—Ç–æ –≤ —Å–ª—É—á–∞–π –Ω–∞ –∫–ª–∞—Å–∏—Ñ–∏–∫–∞—Ü–∏—è –¥–µ–π—Å—Ç–≤–∞ –∫–∞—Ç–æ –≥—Ä–∞–Ω–∏—Ü–∞, –∞ –≤ —Å–ª—É—á–∞–π –Ω–∞ —Ä–µ–≥—Ä–µ—Å–∏—è - –∫–∞—Ç–æ –ª–∏–Ω–∏—è –Ω–∞ –Ω–∞–π-–¥–æ–±—Ä–æ —Å—ä–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏–µ. –í SVM –æ–±–∏–∫–Ω–æ–≤–µ–Ω–æ —Å–µ –∏–∑–ø–æ–ª–∑–≤–∞ Kernel —Ñ—É–Ω–∫—Ü–∏—è –∑–∞ —Ç—Ä–∞–Ω—Å—Ñ–æ—Ä–º–∏—Ä–∞–Ω–µ –Ω–∞ –Ω–∞–±–æ—Ä–∞ –æ—Ç –¥–∞–Ω–Ω–∏ –≤ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ —Å –ø–æ-–≥–æ–ª—è–º –±—Ä–æ–π –∏–∑–º–µ—Ä–µ–Ω–∏—è, —Ç–∞–∫–∞ —á–µ —Ç–µ –¥–∞ –±—ä–¥–∞—Ç –ª–µ—Å–Ω–æ —Ä–∞–∑–¥–µ–ª–∏–º–∏. [–ö–ª–∏–∫–Ω–µ—Ç–µ —Ç—É–∫](https://en.wikipedia.org/wiki/Support-vector_machine) –∑–∞ –ø–æ–≤–µ—á–µ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è –∑–∞ SVM.
- **Support Vector Regressor (SVR):** –í–∏–¥ SVM, –∫–æ–π—Ç–æ –Ω–∞–º–∏—Ä–∞ –ª–∏–Ω–∏—è –Ω–∞ –Ω–∞–π-–¥–æ–±—Ä–æ —Å—ä–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏–µ (–∫–æ—è—Ç–æ –≤ —Å–ª—É—á–∞—è –Ω–∞ SVM –µ —Ö–∏–ø–µ—Ä–ø–ª–æ—Å–∫–æ—Å—Ç), –∫–æ—è—Ç–æ –∏–º–∞ –º–∞–∫—Å–∏–º–∞–ª–µ–Ω –±—Ä–æ–π —Ç–æ—á–∫–∏ –æ—Ç –¥–∞–Ω–Ω–∏.

### –ó–∞—â–æ SVR? [^1]

### –ü–æ–¥–≥–æ—Ç–æ–≤–∫–∞ –Ω–∞ –¥–∞–Ω–Ω–∏—Ç–µ –∑–∞ –æ–±—É—á–µ–Ω–∏–µ

### –ò–º–ø–ª–µ–º–µ–Ω—Ç–∏—Ä–∞–Ω–µ –Ω–∞ SVR [^1]

–°–µ–≥–∞ –µ –≤—Ä–µ–º–µ –¥–∞ –∏–º–ø–ª–µ–º–µ–Ω—Ç–∏—Ä–∞—Ç–µ SVR. –ó–∞ –ø–æ–≤–µ—á–µ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è –æ—Ç–Ω–æ—Å–Ω–æ —Ç–∞–∑–∏ –∏–º–ø–ª–µ–º–µ–Ω—Ç–∞—Ü–∏—è –º–æ–∂–µ—Ç–µ –¥–∞ —Å–µ –æ–±—ä—Ä–Ω–µ—Ç–µ –∫—ä–º [—Ç–∞–∑–∏ –¥–æ–∫—É–º–µ–Ω—Ç–∞—Ü–∏—è](https://scikit-learn.org/stable/modules/generated/sklearn.svm.SVR.html). –ó–∞ –Ω–∞—à–∞—Ç–∞ –∏–º–ø–ª–µ–º–µ–Ω—Ç–∞—Ü–∏—è —Å–ª–µ–¥–≤–∞–º–µ —Ç–µ–∑–∏ —Å—Ç—ä–ø–∫–∏:

  1. –î–µ—Ñ–∏–Ω–∏—Ä–∞–π—Ç–µ –º–æ–¥–µ–ª–∞, –∫–∞—Ç–æ –∏–∑–≤–∏–∫–∞—Ç–µ `SVR()` –∏ –∑–∞–¥–∞–¥–µ—Ç–µ —Ö–∏–ø–µ—Ä–ø–∞—Ä–∞–º–µ—Ç—Ä–∏—Ç–µ –Ω–∞ –º–æ–¥–µ–ª–∞: kernel, gamma, c –∏ epsilon
  2. –ü–æ–¥–≥–æ—Ç–≤–µ—Ç–µ –º–æ–¥–µ–ª–∞ –∑–∞ —Ç—Ä–µ–Ω–∏—Ä–æ–≤—ä—á–Ω–∏—Ç–µ –¥–∞–Ω–Ω–∏, –∫–∞—Ç–æ –∏–∑–≤–∏–∫–∞—Ç–µ —Ñ—É–Ω–∫—Ü–∏—è—Ç–∞ `fit()`
  3. –ù–∞–ø—Ä–∞–≤–µ—Ç–µ –ø—Ä–æ–≥–Ω–æ–∑–∏, –∫–∞—Ç–æ –∏–∑–≤–∏–∫–∞—Ç–µ —Ñ—É–Ω–∫—Ü–∏—è—Ç–∞ `predict()`
