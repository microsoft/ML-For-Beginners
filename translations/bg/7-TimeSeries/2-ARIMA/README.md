## ARIMA –≤ –∫–æ–Ω—Ç–µ–∫—Å—Ç–∞ –Ω–∞ –≤—Ä–µ–º–µ–≤–∏ —Ä–µ–¥–æ–≤–µ

–ù–µ–∫–∞ —Ä–∞–∑–≥–ª–µ–¥–∞–º–µ —á–∞—Å—Ç–∏—Ç–µ –Ω–∞ ARIMA, –∑–∞ –¥–∞ —Ä–∞–∑–±–µ—Ä–µ–º –ø–æ-–¥–æ–±—Ä–µ –∫–∞–∫ –ø–æ–º–∞–≥–∞ –∑–∞ –º–æ–¥–µ–ª–∏—Ä–∞–Ω–µ –Ω–∞ –≤—Ä–µ–º–µ–≤–∏ —Ä–µ–¥–æ–≤–µ –∏ –∑–∞ –ø—Ä–∞–≤–µ–Ω–µ –Ω–∞ –ø—Ä–æ–≥–Ω–æ–∑–∏.

- **AR - –ê–≤—Ç–æ—Ä–µ–≥—Ä–µ—Å–∏–≤–µ–Ω**. –ê–≤—Ç–æ—Ä–µ–≥—Ä–µ—Å–∏–≤–Ω–∏—Ç–µ –º–æ–¥–µ–ª–∏, –∫–∞–∫—Ç–æ –ø–æ–¥—Å–∫–∞–∑–≤–∞ –∏–º–µ—Ç–æ, –≥–ª–µ–¥–∞—Ç "–Ω–∞–∑–∞–¥" –≤—ä–≤ –≤—Ä–µ–º–µ—Ç–æ, –∑–∞ –¥–∞ –∞–Ω–∞–ª–∏–∑–∏—Ä–∞—Ç –ø—Ä–µ–¥–∏—à–Ω–∏ —Å—Ç–æ–π–Ω–æ—Å—Ç–∏ –≤ –¥–∞–Ω–Ω–∏—Ç–µ –∏ –¥–∞ –ø—Ä–∞–≤—è—Ç –ø—Ä–µ–¥–ø–æ–ª–æ–∂–µ–Ω–∏—è –∑–∞ —Ç—è—Ö. –¢–µ–∑–∏ –ø—Ä–µ–¥–∏—à–Ω–∏ —Å—Ç–æ–π–Ω–æ—Å—Ç–∏ —Å–µ –Ω–∞—Ä–∏—á–∞—Ç "–ª–∞–≥–æ–≤–µ". –ü—Ä–∏–º–µ—Ä –±–∏ –±–∏–ª –¥–∞–Ω–Ω–∏, –ø–æ–∫–∞–∑–≤–∞—â–∏ –º–µ—Å–µ—á–Ω–∏ –ø—Ä–æ–¥–∞–∂–±–∏ –Ω–∞ –º–æ–ª–∏–≤–∏. –û–±—â–∞—Ç–∞ —Å—Ç–æ–π–Ω–æ—Å—Ç –Ω–∞ –ø—Ä–æ–¥–∞–∂–±–∏—Ç–µ –∑–∞ –≤—Å–µ–∫–∏ –º–µ—Å–µ—Ü –±–∏ —Å–µ —Å—á–∏—Ç–∞–ª–∞ –∑–∞ "–µ–≤–æ–ª—é–∏—Ä–∞—â–∞ –ø—Ä–æ–º–µ–Ω–ª–∏–≤–∞" –≤ –Ω–∞–±–æ—Ä–∞ –æ—Ç –¥–∞–Ω–Ω–∏. –¢–æ–∑–∏ –º–æ–¥–µ–ª —Å–µ –∏–∑–≥—Ä–∞–∂–¥–∞ –∫–∞—Ç–æ "–µ–≤–æ–ª—é–∏—Ä–∞—â–∞—Ç–∞ –ø—Ä–æ–º–µ–Ω–ª–∏–≤–∞ –æ—Ç –∏–Ω—Ç–µ—Ä–µ—Å —Å–µ —Ä–µ–≥—Ä–µ—Å–∏—Ä–∞ –≤—ä—Ä—Ö—É —Å–æ–±—Å—Ç–≤–µ–Ω–∏—Ç–µ —Å–∏ –ª–∞–≥–æ–≤–∏ (—Ç.–µ. –ø—Ä–µ–¥–∏—à–Ω–∏) —Å—Ç–æ–π–Ω–æ—Å—Ç–∏." ### –ò–º–ø–ª–µ–º–µ–Ω—Ç–∏—Ä–∞–Ω–µ –Ω–∞ ARIMA

–í—Ä–µ–º–µ –µ –¥–∞ –∏–º–ø–ª–µ–º–µ–Ω—Ç–∏—Ä–∞—Ç–µ ARIMA! –°–µ–≥–∞ —â–µ –∏–∑–ø–æ–ª–∑–≤–∞—Ç–µ –±–∏–±–ª–∏–æ—Ç–µ–∫–∞—Ç–∞ `statsmodels`, –∫–æ—è—Ç–æ –∏–Ω—Å—Ç–∞–ª–∏—Ä–∞—Ö—Ç–µ –ø–æ-—Ä–∞–Ω–æ.

–°–µ–≥–∞ —Ç—Ä—è–±–≤–∞ –¥–∞ —Å–ª–µ–¥–≤–∞—Ç–µ –Ω—è–∫–æ–ª–∫–æ —Å—Ç—ä–ø–∫–∏:

   1. –î–µ—Ñ–∏–Ω–∏—Ä–∞–π—Ç–µ –º–æ–¥–µ–ª–∞, –∫–∞—Ç–æ –∏–∑–≤–∏–∫–∞—Ç–µ `SARIMAX()` –∏ –ø–æ–¥–∞–¥–µ—Ç–µ –ø–∞—Ä–∞–º–µ—Ç—Ä–∏—Ç–µ –Ω–∞ –º–æ–¥–µ–ª–∞: –ø–∞—Ä–∞–º–µ—Ç—Ä–∏—Ç–µ p, d –∏ q, –∫–∞–∫—Ç–æ –∏ –ø–∞—Ä–∞–º–µ—Ç—Ä–∏—Ç–µ P, D –∏ Q.
   2. –ü–æ–¥–≥–æ—Ç–≤–µ—Ç–µ –º–æ–¥–µ–ª–∞ –∑–∞ —Ç—Ä–µ–Ω–∏—Ä–æ–≤—ä—á–Ω–∏—Ç–µ –¥–∞–Ω–Ω–∏, –∫–∞—Ç–æ –∏–∑–≤–∏–∫–∞—Ç–µ —Ñ—É–Ω–∫—Ü–∏—è—Ç–∞ fit().
   3. –ù–∞–ø—Ä–∞–≤–µ—Ç–µ –ø—Ä–æ–≥–Ω–æ–∑–∏, –∫–∞—Ç–æ –∏–∑–≤–∏–∫–∞—Ç–µ —Ñ—É–Ω–∫—Ü–∏—è—Ç–∞ `forecast()` –∏ –∑–∞–¥–∞–¥–µ—Ç–µ –±—Ä–æ—è –Ω–∞ —Å—Ç—ä–ø–∫–∏—Ç–µ (—Ö–æ—Ä–∏–∑–æ–Ω—Ç–∞), –∫–æ–∏—Ç–æ –¥–∞ –ø—Ä–æ–≥–Ω–æ–∑–∏—Ä–∞—Ç–µ.

> üéì –ö–∞–∫–≤–æ –æ–∑–Ω–∞—á–∞–≤–∞—Ç –≤—Å–∏—á–∫–∏ —Ç–µ–∑–∏ –ø–∞—Ä–∞–º–µ—Ç—Ä–∏? –í ARIMA –º–æ–¥–µ–ª –∏–º–∞ 3 –ø–∞—Ä–∞–º–µ—Ç—ä—Ä–∞, –∫–æ–∏—Ç–æ —Å–µ –∏–∑–ø–æ–ª–∑–≤–∞—Ç –∑–∞ –º–æ–¥–µ–ª–∏—Ä–∞–Ω–µ –Ω–∞ –æ—Å–Ω–æ–≤–Ω–∏—Ç–µ –∞—Å–ø–µ–∫—Ç–∏ –Ω–∞ –≤—Ä–µ–º–µ–≤–∏ —Ä–µ–¥–æ–≤–µ: —Å–µ–∑–æ–Ω–Ω–æ—Å—Ç, —Ç–µ–Ω–¥–µ–Ω—Ü–∏—è –∏ —à—É–º. –¢–µ–∑–∏ –ø–∞—Ä–∞–º–µ—Ç—Ä–∏ —Å–∞:

`p`: –ø–∞—Ä–∞–º–µ—Ç—ä—Ä, —Å–≤—ä—Ä–∑–∞–Ω —Å –∞–≤—Ç–æ—Ä–µ–≥—Ä–µ—Å–∏–≤–Ω–∏—è –∞—Å–ø–µ–∫—Ç –Ω–∞ –º–æ–¥–µ–ª–∞, –∫–æ–π—Ç–æ –≤–∫–ª—é—á–≤–∞ *–º–∏–Ω–∞–ª–∏* —Å—Ç–æ–π–Ω–æ—Å—Ç–∏.
`d`: –ø–∞—Ä–∞–º–µ—Ç—ä—Ä, —Å–≤—ä—Ä–∑–∞–Ω —Å –∏–Ω—Ç–µ–≥—Ä–∏—Ä–∞–Ω–∏—è –∞—Å–ø–µ–∫—Ç –Ω–∞ –º–æ–¥–µ–ª–∞, –∫–æ–π—Ç–æ –≤–ª–∏—è–µ –≤—ä—Ä—Ö—É –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ—Ç–æ *–¥–∏—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–∞–Ω–µ* (üéì –ø–æ–º–Ω–∏—Ç–µ –ª–∏ –¥–∏—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–∞–Ω–µ—Ç–æ üëÜ?) –∑–∞ –ø—Ä–∏–ª–∞–≥–∞–Ω–µ –∫—ä–º –≤—Ä–µ–º–µ–≤–∏—è —Ä–µ–¥.
`q`: –ø–∞—Ä–∞–º–µ—Ç—ä—Ä, —Å–≤—ä—Ä–∑–∞–Ω —Å –∞—Å–ø–µ–∫—Ç–∞ –Ω–∞ –ø–æ–¥–≤–∏–∂–Ω–∞—Ç–∞ —Å—Ä–µ–¥–Ω–∞ –≤ –º–æ–¥–µ–ª–∞.

> –ó–∞–±–µ–ª–µ–∂–∫–∞: –ê–∫–æ –≤–∞—à–∏—Ç–µ –¥–∞–Ω–Ω–∏ –∏–º–∞—Ç —Å–µ–∑–æ–Ω–µ–Ω –∞—Å–ø–µ–∫—Ç - –∫–∞–∫—ä–≤—Ç–æ –µ —Å–ª—É—á–∞—è—Ç —Ç—É–∫ - –∏–∑–ø–æ–ª–∑–≤–∞–º–µ —Å–µ–∑–æ–Ω–µ–Ω ARIMA –º–æ–¥–µ–ª (SARIMA). –í —Ç–æ–∑–∏ —Å–ª—É—á–∞–π —Ç—Ä—è–±–≤–∞ –¥–∞ –∏–∑–ø–æ–ª–∑–≤–∞—Ç–µ –¥—Ä—É–≥ –Ω–∞–±–æ—Ä –æ—Ç –ø–∞—Ä–∞–º–µ—Ç—Ä–∏: `P`, `D` –∏ `Q`, –∫–æ–∏—Ç–æ –æ–ø–∏—Å–≤–∞—Ç —Å—ä—â–∏—Ç–µ –∞—Å–æ—Ü–∏–∞—Ü–∏–∏ –∫–∞—Ç–æ `p`, `d` –∏ `q`, –Ω–æ —Å—ä–æ—Ç–≤–µ—Ç—Å—Ç–≤–∞—Ç –Ω–∞ —Å–µ–∑–æ–Ω–Ω–∏—Ç–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∏ –Ω–∞ –º–æ–¥–µ–ª–∞.

1. –ó–∞–ø–æ—á–Ω–µ—Ç–µ, –∫–∞—Ç–æ –∑–∞–¥–∞–¥–µ—Ç–µ –ø—Ä–µ–¥–ø–æ—á–∏—Ç–∞–Ω–∞—Ç–∞ —Å—Ç–æ–π–Ω–æ—Å—Ç –∑–∞ —Ö–æ—Ä–∏–∑–æ–Ω—Ç–∞. –ù–µ–∫–∞ –æ–ø–∏—Ç–∞–º–µ 3 —á–∞—Å–∞:

    ```python
    # Specify the number of steps to forecast ahead
    HORIZON = 3

    –ò–∑–±–æ—Ä—ä—Ç –Ω–∞ –Ω–∞–π-–¥–æ–±—Ä–∏—Ç–µ —Å—Ç–æ–π–Ω–æ—Å—Ç–∏ –∑–∞ –ø–∞—Ä–∞–º–µ—Ç—Ä–∏—Ç–µ –Ω–∞ ARIMA –º–æ–¥–µ–ª–∞ –º–æ–∂–µ –¥–∞ –±—ä–¥–µ –ø—Ä–µ–¥–∏–∑–≤–∏–∫–∞—Ç–µ–ª—Å—Ç–≤–æ, —Ç—ä–π –∫–∞—Ç–æ –µ –¥–æ–Ω—è–∫—ä–¥–µ —Å—É–±–µ–∫—Ç–∏–≤–µ–Ω –∏ –æ—Ç–Ω–µ–º–∞ –≤—Ä–µ–º–µ. –ú–æ–∂–µ –¥–∞ –æ–±–º–∏—Å–ª–∏—Ç–µ –∏–∑–ø–æ–ª–∑–≤–∞–Ω–µ—Ç–æ –Ω–∞ —Ñ—É–Ω–∫—Ü–∏—è—Ç–∞ `auto_arima()` –æ—Ç –±–∏–±–ª–∏–æ—Ç–µ–∫–∞—Ç–∞ [`pyramid`](https://alkaline-ml.com/pmdarima/0.9.0/modules/generated/pyramid.arima.auto_arima.html).

    training_window = 720 # dedicate 30 days (720 hours) for training

    train_ts = train['load']
    test_ts = test_shifted

    history = [x for x in train_ts]
    history = history[(-training_window):]

    predictions = list()

    order = (2, 1, 0)
    seasonal_order = (1, 1, 0, 24)

    for t in range(test_ts.shape[0]):
        model = SARIMAX(endog=history, order=order, seasonal_order=seasonal_order)
        model_fit = model.fit()
        yhat = model_fit.forecast(steps = HORIZON)
        predictions.append(yhat)
        obs = list(test_ts.iloc[t])
        # move the training window
        history.append(obs[0])
        history.pop(0)
        print(test_ts.index[t])
        print(t+1, ': predicted =', yhat, 'expected =', obs)
    ```
