# 采访数据科学家

## 指导说明

在你的公司、用户组、朋友或同学中，找一位专业从事数据科学工作的人员进行交流。撰写一篇简短的文章（500字），描述他们的日常工作内容。他们是专攻某一领域，还是从事“全栈”工作？

## 评分标准

| 标准     | 卓越表现                                                                 | 合格表现                                                       | 需要改进             |
| -------- | ----------------------------------------------------------------------- | ------------------------------------------------------------- | -------------------- |
|          | 提交一篇符合字数要求、带有明确来源的文章，并以 .doc 文件形式呈现         | 文章来源不明确或字数少于要求                                   | 未提交文章           |
