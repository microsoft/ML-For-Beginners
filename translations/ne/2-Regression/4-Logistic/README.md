1. Ordinal encoder: рдпреЛ ordinal variables рдХрд╛ рд▓рд╛рдЧрд┐ рдЙрдкрдпреБрдХреНрдд рдЫ, рдЬреБрди categorical variables рд╣реБрдиреН рдЬрд╣рд╛рдБ рддрд┐рдиреАрд╣рд░реВрдХреЛ рдбреЗрдЯрд╛ рддрд╛рд░реНрдХрд┐рдХ рдХреНрд░рдо рдЕрдиреБрд╕рд░рдг рдЧрд░реНрджрдЫ, рдЬрд╕реНрддреИ рд╣рд╛рдореНрд░реЛ dataset рдорд╛ `Item Size` рд╕реНрддрдореНрднред рдпреЛ mapping рд╕рд┐рд░реНрдЬрдирд╛ рдЧрд░реНрджрдЫ рдЬрд╕рд▓реЗ рдкреНрд░рддреНрдпреЗрдХ рд╡рд░реНрдЧрд▓рд╛рдИ рдПрдХ рд╕рдВрдЦреНрдпрд╛рд▓реЗ рдкреНрд░рддрд┐рдирд┐рдзрд┐рддреНрд╡ рдЧрд░реНрджрдЫ, рдЬреБрди рд╕реНрддрдореНрднрдорд╛ рд╡рд░реНрдЧрдХреЛ рдХреНрд░рдо рд╣реЛред
