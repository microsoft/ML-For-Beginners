## рдЕрднреНрдпрд╛рд╕ - Logistic Regression рд▓рд╛рдЧреВ рдЧрд░реНрдиреБрд╣реЛрд╕реН

рддрдкрд╛рдИрдВ рдмрд╣реБ-рд╡рд░реНрдЧ рдХреЗрд╕ рдкреНрд░рдпреЛрдЧ рдЧрд░реНрджреИ рд╣реБрдиреБрд╣реБрдиреНрдЫ, рддреНрдпрд╕реИрд▓реЗ рддрдкрд╛рдИрдВрд▓реЗ рдХреБрди _scheme_ рдкреНрд░рдпреЛрдЧ рдЧрд░реНрдиреЗ рд░ рдХреБрди _solver_ рд╕реЗрдЯ рдЧрд░реНрдиреЗ рдирд┐рд░реНрдгрдп рдЧрд░реНрдиреБрдкрд░реНрдЫред рдмрд╣реБ-рд╡рд░реНрдЧ рд╕реЗрдЯрд┐рдЩ рд░ **liblinear** solver рдкреНрд░рдпреЛрдЧ рдЧрд░реЗрд░ Logistic Regression рд▓рд╛рдЧреВ рдЧрд░реНрдиреБрд╣реЛрд╕реНред

1. `multi_class` рд▓рд╛рдИ `ovr` рдорд╛ рд╕реЗрдЯ рдЧрд░реЗрд░ рд░ solver рд▓рд╛рдИ `liblinear` рдорд╛ рд╕реЗрдЯ рдЧрд░реЗрд░ Logistic Regression рд╕рд┐рд░реНрдЬрдирд╛ рдЧрд░реНрдиреБрд╣реЛрд╕реН:

    ```python
    lr = LogisticRegression(multi_class='ovr',solver='liblinear')
    model = lr.fit(X_train, np.ravel(y_train))
    
    accuracy = model.score(X_test, y_test)
    ```
