2. рд╢реНрд░реЗрдгреАрдмрджреНрдз рдПрдиреНрдХреЛрдбрд░: рдпрд╣ рдирд╛рдордорд╛рддреНрд░ рд╡реЗрд░рд┐рдПрдмрд▓реНрд╕ рдХреЗ рд▓рд┐рдП рдЙрдкрдпреБрдХреНрдд рд╣реИ, рдЬреЛ рд╢реНрд░реЗрдгреАрдмрджреНрдз рд╡реЗрд░рд┐рдПрдмрд▓реНрд╕ рд╣реИрдВ рдЬрд╣рд╛рдВ рдЙрдирдХрд╛ рдбреЗрдЯрд╛ рддрд╛рд░реНрдХрд┐рдХ рдХреНрд░рдо рдХрд╛ рдкрд╛рд▓рди рдирд╣реАрдВ рдХрд░рддрд╛, рдЬреИрд╕реЗ рд╣рдорд╛рд░реЗ рдбреЗрдЯрд╛рд╕реЗрдЯ рдореЗрдВ `Item Size` рдХреЗ рдЕрд▓рд╛рд╡рд╛ рдЕрдиреНрдп рд╕рднреА рдлреАрдЪрд░реНрд╕ред рдпрд╣ рдПрдХ рд╡рди-рд╣реЙрдЯ рдПрдиреНрдХреЛрдбрд┐рдВрдЧ рд╣реИ, рдЬрд┐рд╕рдХрд╛ рдЕрд░реНрде рд╣реИ рдХрд┐ рдкреНрд░рддреНрдпреЗрдХ рд╢реНрд░реЗрдгреА рдХреЛ рдПрдХ рдмрд╛рдЗрдирд░реА рдХреЙрд▓рдо рджреНрд╡рд╛рд░рд╛ рджрд░реНрд╢рд╛рдпрд╛ рдЬрд╛рддрд╛ рд╣реИ: рдПрдиреНрдХреЛрдбреЗрдб рд╡реЗрд░рд┐рдПрдмрд▓ 1 рдХреЗ рдмрд░рд╛рдмрд░ рд╣реИ рдпрджрд┐ рдХрджреНрджреВ рдЙрд╕ `Variety` рд╕реЗ рд╕рдВрдмрдВрдзрд┐рдд рд╣реИ рдФрд░ рдЕрдиреНрдпрдерд╛ 0ред
