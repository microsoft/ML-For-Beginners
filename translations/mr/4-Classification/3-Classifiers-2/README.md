Probability ‡§°‡•Ä‡§´‡•â‡§≤‡•ç‡§ü‡§®‡•Å‡§∏‡§æ‡§∞ 'false' ‡§Ö‡§∏‡§§‡•á; ‡§Ø‡•á‡§•‡•á ‡§Ü‡§Æ‡•ç‡§π‡•Ä probability estimates ‡§ó‡•ã‡§≥‡§æ ‡§ï‡§∞‡§£‡•ç‡§Ø‡§æ‡§∏‡§æ‡§†‡•Ä 'true' ‡§∏‡•á‡§ü ‡§ï‡§∞‡§§‡•ã. ‡§Ü‡§Æ‡•ç‡§π‡•Ä random state '0' ‡§µ‡§∞ ‡§∏‡•á‡§ü ‡§ï‡§∞‡§§‡•ã ‡§ú‡•á‡§£‡•á‡§ï‡§∞‡•Ç‡§® ‡§°‡•á‡§ü‡§æ shuffle ‡§π‡•ã‡§à‡§≤ ‡§Ü‡§£‡§ø probabilities ‡§Æ‡§ø‡§≥‡§§‡•Ä‡§≤.

### ‡§µ‡•ç‡§Ø‡§æ‡§Ø‡§æ‡§Æ - Linear SVC ‡§≤‡§æ‡§ó‡•Ç ‡§ï‡§∞‡§æ

‡§µ‡§∞‡•ç‡§ó‡•Ä‡§ï‡§∞‡•ç‡§§‡•ç‡§Ø‡§æ‡§Ç‡§ö‡•Ä ‡§è‡§ï array ‡§§‡§Ø‡§æ‡§∞ ‡§ï‡§∞‡•Ç‡§® ‡§∏‡•Å‡§∞‡•Å‡§µ‡§æ‡§§ ‡§ï‡§∞‡§æ. ‡§Ü‡§Æ‡•ç‡§π‡•Ä ‡§ö‡§æ‡§ö‡§£‡•Ä ‡§ï‡§∞‡§§ ‡§Ö‡§∏‡§§‡§æ‡§®‡§æ ‡§§‡•Å‡§Æ‡•ç‡§π‡•Ä ‡§π‡§≥‡•Ç‡§π‡§≥‡•Ç ‡§Ø‡§æ array ‡§Æ‡§ß‡•ç‡§Ø‡•á ‡§≠‡§∞ ‡§ò‡§æ‡§≤‡§æ‡§≤.

2. Linear SVC ‡§µ‡§æ‡§™‡§∞‡•Ç‡§® ‡§§‡•Å‡§Æ‡§ö‡•á ‡§Æ‡•â‡§°‡•á‡§≤ ‡§™‡•ç‡§∞‡§∂‡§ø‡§ï‡•ç‡§∑‡§ø‡§§ ‡§ï‡§∞‡§æ ‡§Ü‡§£‡§ø ‡§∞‡§ø‡§™‡•ã‡§∞‡•ç‡§ü ‡§™‡•ç‡§∞‡§ø‡§Ç‡§ü ‡§ï‡§∞‡§æ:
